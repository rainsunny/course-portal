# Day 7：深度Q网络（DQN）及其改进

> **核心问题**：当状态空间无限大时，如何用神经网络替代表格？

## 引言：从Atari到AlphaGo的跨越

2013年，DeepMind 发表了一篇震惊学界的论文：**用同一个神经网络，同一个算法，在 49 款 Atari 游戏上达到人类水平**。

这不是简单的技术改进，而是范式转移：

| 特性 | 表格方法 | 深度Q网络 |
|------|----------|-----------|
| 状态表示 | 离散索引 | 连续特征（如图像像素） |
| Q值存储 | 表格 Q(s,a) | 神经网络 Q(s,a;θ) |
| 泛化能力 | 无 | 有（相似状态相似值） |
| 状态空间 | 小（<10^6） | 无限大 |

**核心创新**：用神经网络近似 Q 函数，处理高维状态空间。

今天，我们深入理解 DQN 及其改进，这是现代深度强化学习的基石。

---

## 1. 为什么需要函数逼近？

### 1.1 表格方法的局限

**维度灾难**：状态空间随变量数量指数增长。

**例子**：
- 国际象棋：~10^47 个状态
- 围棋：~10^170 个状态
- Atari 游戏（210×160×3 像素）：~10^100,000 个状态

**表格 Q-Learning 的问题**：
1. **存储**：无法存储所有 Q 值
2. **学习**：无法访问所有状态-动作对
3. **泛化**：无法处理新状态

### 1.2 函数逼近的思想

**核心思想**：用参数化函数近似 Q 函数。

$$Q(s, a) \approx Q(s, a; \theta)$$

其中 θ 是函数参数（神经网络权重）。

**好处**：
1. **存储高效**：只需存储参数 θ，而非 Q 表
2. **泛化能力**：相似状态产生相似 Q 值
3. **处理连续状态**：状态可以是任意特征向量

**挑战**：
1. **非平稳目标**：目标值随学习变化
2. **相关性**：连续样本高度相关
3. **函数逼近误差**：可能导致发散

### 1.3 函数逼近的形式

| 类型 | 输入 | 输出 | 示例 |
|------|------|------|------|
| **状态价值** V(s;θ) | 状态 s | 标量 | 评估状态好坏 |
| **状态-动作价值** Q(s,a;θ) | (s,a) | 标量 | 评估动作好坏 |
| **动作价值** Q(s;θ) | 状态 s | 向量（每个动作一个值） | DQN 采用这种 |

**DQN 的选择**：输入状态，输出所有动作的 Q 值。

```
状态 s（图像） → 卷积网络 → 全连接层 → [Q(s,a₁), Q(s,a₂), ..., Q(s,aₙ)]
```

## 2. DQN：深度Q网络

### 2.1 核心架构

**输入**：84×84×4 的图像栈（4 帧灰度图像堆叠）

**网络结构**：
```
输入: 84×84×4
  ↓
卷积层1: 32个 8×8 滤波器, 步长4 → 20×20×32
  ↓ ReLU
卷积层2: 64个 4×4 滤波器, 步长2 → 9×9×64
  ↓ ReLU
卷积层3: 64个 3×3 滤波器, 步长1 → 7×7×64
  ↓ ReLU
展平: 3136
  ↓
全连接层1: 512
  ↓ ReLU
全连接层2: n_actions（输出Q值）
```

### 2.2 DQN 的两大创新

**创新一：经验回放（Experience Replay）**

**问题**：连续样本高度相关，违反独立同分布假设。

**解决方案**：存储经验到缓冲区，随机采样训练。

```python
class ReplayBuffer:
    """经验回放缓冲区"""
    
    def __init__(self, capacity=100000):
        self.capacity = capacity
        self.buffer = []
        self.position = 0
    
    def push(self, state, action, reward, next_state, done):
        """添加经验"""
        if len(self.buffer) < self.capacity:
            self.buffer.append(None)
        self.buffer[self.position] = (state, action, reward, next_state, done)
        self.position = (self.position + 1) % self.capacity
    
    def sample(self, batch_size):
        """随机采样"""
        batch = random.sample(self.buffer, batch_size)
        states, actions, rewards, next_states, dones = zip(*batch)
        return (
            np.array(states),
            np.array(actions),
            np.array(rewards),
            np.array(next_states),
            np.array(dones)
        )
    
    def __len__(self):
        return len(self.buffer)
```

**好处**：
1. 打破样本相关性
2. 提高数据利用率（一条经验可多次使用）
3. 防止遗忘旧经验

**创新二：目标网络（Target Network）**

**问题**：Q-Learning 的目标是 $y = r + \gamma \max_{a'} Q(s', a'; \theta)$，与当前网络参数 θ 相关，导致训练不稳定。

**解决方案**：使用独立的目标网络 $\theta^-$ 计算目标，定期同步。

$$y = r + \gamma \max_{a'} Q(s', a'; \theta^-)$$

```python
# 目标网络更新
if step % target_update_freq == 0:
    target_network.load_state_dict(q_network.state_dict())
```

**好处**：
1. 固定目标一段时间，减少震荡
2. 类似于监督学习的稳定训练

### 2.3 DQN 算法流程

```
初始化：
    Q 网络 Q(s, a; θ)
    目标网络 Q(s, a; θ⁻)，θ⁻ = θ
    经验回放缓冲区 D

循环 episode = 1, 2, ...:
    初始化状态 s₁
    
    循环 t = 1, 2, ...:
        # 动作选择（ε-贪心）
        以 ε 概率随机选择动作
        否则 a_t = argmax_a Q(s_t, a; θ)
        
        # 执行动作
        执行 a_t，观察 r_t, s_{t+1}, done
        
        # 存储经验
        D.push(s_t, a_t, r_t, s_{t+1}, done)
        
        # 采样并训练
        if len(D) >= batch_size:
            从 D 随机采样 batch_size 个经验
            计算 y_j = r_j + γ (1 - done_j) max_a' Q(s'_j, a'; θ⁻)
            计算损失 L = (y_j - Q(s_j, a_j; θ))²
            梯度下降更新 θ
        
        # 更新目标网络
        if t % target_update_freq == 0:
            θ⁻ = θ
        
        s_t = s_{t+1}
        
        if done:
            break
```

### 2.4 完整实现

```python
import torch
import torch.nn as nn
import torch.optim as optim
import numpy as np
import random
from collections import deque

class DQNetwork(nn.Module):
    """DQN 网络"""
    
    def __init__(self, state_dim, action_dim, hidden_dim=128):
        super().__init__()
        
        # 对于简单状态（如 CartPole）
        self.network = nn.Sequential(
            nn.Linear(state_dim, hidden_dim),
            nn.ReLU(),
            nn.Linear(hidden_dim, hidden_dim),
            nn.ReLU(),
            nn.Linear(hidden_dim, action_dim)
        )
    
    def forward(self, x):
        return self.network(x)


class DQN:
    """DQN 智能体"""
    
    def __init__(
        self,
        state_dim,
        action_dim,
        hidden_dim=128,
        learning_rate=1e-3,
        gamma=0.99,
        epsilon_start=1.0,
        epsilon_end=0.01,
        epsilon_decay=0.995,
        buffer_size=100000,
        batch_size=64,
        target_update_freq=10
    ):
        self.state_dim = state_dim
        self.action_dim = action_dim
        self.gamma = gamma
        self.epsilon = epsilon_start
        self.epsilon_end = epsilon_end
        self.epsilon_decay = epsilon_decay
        self.batch_size = batch_size
        self.target_update_freq = target_update_freq
        
        # Q 网络和目标网络
        self.q_network = DQNetwork(state_dim, action_dim, hidden_dim)
        self.target_network = DQNetwork(state_dim, action_dim, hidden_dim)
        self.target_network.load_state_dict(self.q_network.state_dict())
        
        # 优化器
        self.optimizer = optim.Adam(self.q_network.parameters(), lr=learning_rate)
        
        # 经验回放
        self.replay_buffer = deque(maxlen=buffer_size)
        
        # 计步器
        self.step_count = 0
    
    def select_action(self, state, training=True):
        """选择动作"""
        if training and random.random() < self.epsilon:
            return random.randint(0, self.action_dim - 1)
        
        with torch.no_grad():
            state = torch.FloatTensor(state).unsqueeze(0)
            q_values = self.q_network(state)
            return q_values.argmax(dim=1).item()
    
    def store_transition(self, state, action, reward, next_state, done):
        """存储经验"""
        self.replay_buffer.append((state, action, reward, next_state, done))
    
    def update(self):
        """更新网络"""
        if len(self.replay_buffer) < self.batch_size:
            return None
        
        # 采样
        batch = random.sample(self.replay_buffer, self.batch_size)
        states, actions, rewards, next_states, dones = zip(*batch)
        
        states = torch.FloatTensor(np.array(states))
        actions = torch.LongTensor(actions)
        rewards = torch.FloatTensor(rewards)
        next_states = torch.FloatTensor(np.array(next_states))
        dones = torch.FloatTensor(dones)
        
        # 计算当前 Q 值
        current_q = self.q_network(states).gather(1, actions.unsqueeze(1)).squeeze(1)
        
        # 计算目标 Q 值
        with torch.no_grad():
            next_q = self.target_network(next_states).max(dim=1)[0]
            target_q = rewards + self.gamma * next_q * (1 - dones)
        
        # 计算损失
        loss = nn.MSELoss()(current_q, target_q)
        
        # 梯度下降
        self.optimizer.zero_grad()
        loss.backward()
        self.optimizer.step()
        
        # 更新目标网络
        self.step_count += 1
        if self.step_count % self.target_update_freq == 0:
            self.target_network.load_state_dict(self.q_network.state_dict())
        
        # 衰减探索率
        self.epsilon = max(self.epsilon_end, self.epsilon * self.epsilon_decay)
        
        return loss.item()
    
    def train(self, env, episodes=1000, max_steps=1000):
        """训练"""
        rewards_history = []
        loss_history = []
        
        for episode in range(episodes):
            state = env.reset()
            episode_reward = 0
            
            for step in range(max_steps):
                # 选择动作
                action = self.select_action(state)
                
                # 执行动作
                next_state, reward, done, _ = env.step(action)
                
                # 存储经验
                self.store_transition(state, action, reward, next_state, done)
                
                # 更新网络
                loss = self.update()
                if loss is not None:
                    loss_history.append(loss)
                
                episode_reward += reward
                state = next_state
                
                if done:
                    break
            
            rewards_history.append(episode_reward)
            
            if (episode + 1) % 50 == 0:
                avg_reward = np.mean(rewards_history[-50:])
                print(f"Episode {episode + 1}, Avg Reward: {avg_reward:.2f}, Epsilon: {self.epsilon:.3f}")
        
        return rewards_history, loss_history
```

## 3. DQN 的改进

### 3.1 Double DQN

**问题**：标准 DQN 存在**过高估计偏差**。

**原因**：$\max_{a'} Q(s', a')$ 倾向于选择被高估的动作。

**Double DQN 的解决方案**：解耦选择和评估。

$$y = r + \gamma Q\left(s', \arg\max_{a'} Q(s', a'; \theta); \theta^-\right)$$

- 用 Q 网络（θ）选择动作
- 用目标网络（θ⁻）评估动作

```python
class DoubleDQN(DQN):
    """Double DQN"""
    
    def update(self):
        if len(self.replay_buffer) < self.batch_size:
            return None
        
        batch = random.sample(self.replay_buffer, self.batch_size)
        states, actions, rewards, next_states, dones = zip(*batch)
        
        states = torch.FloatTensor(np.array(states))
        actions = torch.LongTensor(actions)
        rewards = torch.FloatTensor(rewards)
        next_states = torch.FloatTensor(np.array(next_states))
        dones = torch.FloatTensor(dones)
        
        # 当前 Q 值
        current_q = self.q_network(states).gather(1, actions.unsqueeze(1)).squeeze(1)
        
        # Double DQN：用 Q 网络选择动作，用目标网络评估
        with torch.no_grad():
            # Q 网络选择最优动作
            best_actions = self.q_network(next_states).argmax(dim=1)
            # 目标网络评估
            next_q = self.target_network(next_states).gather(1, best_actions.unsqueeze(1)).squeeze(1)
            target_q = rewards + self.gamma * next_q * (1 - dones)
        
        loss = nn.MSELoss()(current_q, target_q)
        
        self.optimizer.zero_grad()
        loss.backward()
        self.optimizer.step()
        
        self.step_count += 1
        if self.step_count % self.target_update_freq == 0:
            self.target_network.load_state_dict(self.q_network.state_dict())
        
        self.epsilon = max(self.epsilon_end, self.epsilon * self.epsilon_decay)
        
        return loss.item()
```

### 3.2 Dueling DQN

**核心思想**：分解 Q 值为状态价值和优势函数。

$$Q(s, a) = V(s) + A(s, a)$$

其中：
- $V(s)$：状态价值（在这个状态有多好）
- $A(s, a)$：优势函数（动作 a 比平均好多少）

**网络结构**：

```
状态 s
  ↓
共享特征层
  ↓
┌─────────┴─────────┐
↓                   ↓
V(s) 流           A(s,a) 流
(标量)           (动作维度)
↓                   ↓
└─────────┬─────────┘
          ↓
      Q(s,a) = V(s) + A(s,a) - mean(A)
```

**为什么要减去 mean(A)？**

因为 $Q(s,a) = V(s) + A(s,a)$ 有无穷多解（可以给 V 加任意常数，同时给 A 减同样常数）。约束 $A$ 的均值为 0：

$$Q(s, a) = V(s) + \left( A(s, a) - \frac{1}{|\mathcal{A}|} \sum_{a'} A(s, a') \right)$$

```python
class DuelingDQN(nn.Module):
    """Dueling DQN 网络"""
    
    def __init__(self, state_dim, action_dim, hidden_dim=128):
        super().__init__()
        
        # 共享特征层
        self.feature = nn.Sequential(
            nn.Linear(state_dim, hidden_dim),
            nn.ReLU()
        )
        
        # 状态价值流
        self.value_stream = nn.Sequential(
            nn.Linear(hidden_dim, hidden_dim),
            nn.ReLU(),
            nn.Linear(hidden_dim, 1)
        )
        
        # 优势流
        self.advantage_stream = nn.Sequential(
            nn.Linear(hidden_dim, hidden_dim),
            nn.ReLU(),
            nn.Linear(hidden_dim, action_dim)
        )
    
    def forward(self, x):
        features = self.feature(x)
        
        value = self.value_stream(features)
        advantage = self.advantage_stream(features)
        
        # Q = V + (A - mean(A))
        q_values = value + (advantage - advantage.mean(dim=1, keepdim=True))
        
        return q_values


class DuelingDQNAgent(DQN):
    """Dueling DQN 智能体"""
    
    def __init__(self, *args, **kwargs):
        super().__init__(*args, **kwargs)
        
        # 替换为 Dueling 网络
        self.q_network = DuelingDQN(
            kwargs['state_dim'], 
            kwargs['action_dim'],
            kwargs.get('hidden_dim', 128)
        )
        self.target_network = DuelingDQN(
            kwargs['state_dim'], 
            kwargs['action_dim'],
            kwargs.get('hidden_dim', 128)
        )
        self.target_network.load_state_dict(self.q_network.state_dict())
        
        self.optimizer = optim.Adam(
            self.q_network.parameters(), 
            lr=kwargs.get('learning_rate', 1e-3)
        )
```

### 3.3 优先级经验回放

**动机**：并非所有经验都同等重要。

**思想**：根据 TD 误差的大小，优先采样"有学习价值"的经验。

**优先级定义**：

$$p_i = |\delta_i| + \epsilon$$

其中 $\delta_i$ 是 TD 误差，$\epsilon$ 是小常数防止优先级为 0。

**采样概率**：

$$P(i) = \frac{p_i^\alpha}{\sum_k p_k^\alpha}$$

α 控制优先程度：α=0 为均匀采样，α=1 为完全优先。

**重要性采样权重**：

$$w_i = \left( \frac{1}{N \cdot P(i)} \right)^\beta$$

用于修正采样偏差。

```python
class PrioritizedReplayBuffer:
    """优先级经验回放"""
    
    def __init__(self, capacity=100000, alpha=0.6, beta_start=0.4, beta_frames=100000):
        self.capacity = capacity
        self.alpha = alpha
        self.beta_start = beta_start
        self.beta_frames = beta_frames
        self.frame = 0
        
        self.buffer = []
        self.priorities = np.zeros(capacity, dtype=np.float32)
        self.position = 0
        self.max_priority = 1.0
    
    def push(self, state, action, reward, next_state, done):
        """添加经验"""
        if len(self.buffer) < self.capacity:
            self.buffer.append((state, action, reward, next_state, done))
        else:
            self.buffer[self.position] = (state, action, reward, next_state, done)
        
        # 新经验赋予最大优先级
        self.priorities[self.position] = self.max_priority
        self.position = (self.position + 1) % self.capacity
    
    def sample(self, batch_size):
        """优先级采样"""
        if len(self.buffer) < self.capacity:
            probs = self.priorities[:len(self.buffer)]
        else:
            probs = self.priorities
        
        probs = probs ** self.alpha
        probs = probs / probs.sum()
        
        indices = np.random.choice(len(self.buffer), batch_size, p=probs)
        samples = [self.buffer[i] for i in indices]
        
        # 计算重要性采样权重
        beta = self.beta_start + (1 - self.beta_start) * (self.frame / self.beta_frames)
        self.frame += 1
        
        weights = (len(self.buffer) * probs[indices]) ** (-beta)
        weights = weights / weights.max()
        
        states, actions, rewards, next_states, dones = zip(*samples)
        return (
            np.array(states),
            np.array(actions),
            np.array(rewards),
            np.array(next_states),
            np.array(dones),
            indices,
            weights
        )
    
    def update_priorities(self, indices, td_errors):
        """更新优先级"""
        for i, td_error in zip(indices, td_errors):
            self.priorities[i] = abs(td_error) + 1e-5
            self.max_priority = max(self.max_priority, self.priorities[i])
    
    def __len__(self):
        return len(self.buffer)


class PrioritizedDQN(DQN):
    """带优先级回放的 DQN"""
    
    def __init__(self, *args, **kwargs):
        super().__init__(*args, **kwargs)
        
        # 替换为优先级回放缓冲区
        self.replay_buffer = PrioritizedReplayBuffer(
            capacity=kwargs.get('buffer_size', 100000)
        )
    
    def update(self):
        if len(self.replay_buffer) < self.batch_size:
            return None
        
        # 采样（包含优先级信息）
        states, actions, rewards, next_states, dones, indices, weights = \
            self.replay_buffer.sample(self.batch_size)
        
        states = torch.FloatTensor(states)
        actions = torch.LongTensor(actions)
        rewards = torch.FloatTensor(rewards)
        next_states = torch.FloatTensor(next_states)
        dones = torch.FloatTensor(dones)
        weights = torch.FloatTensor(weights)
        
        # 计算 Q 值
        current_q = self.q_network(states).gather(1, actions.unsqueeze(1)).squeeze(1)
        
        with torch.no_grad():
            next_q = self.target_network(next_states).max(dim=1)[0]
            target_q = rewards + self.gamma * next_q * (1 - dones)
        
        # TD 误差
        td_errors = target_q - current_q
        
        # 加权损失
        loss = (weights * td_errors ** 2).mean()
        
        self.optimizer.zero_grad()
        loss.backward()
        self.optimizer.step()
        
        # 更新优先级
        self.replay_buffer.update_priorities(indices, td_errors.detach().numpy())
        
        # 更新目标网络
        self.step_count += 1
        if self.step_count % self.target_update_freq == 0:
            self.target_network.load_state_dict(self.q_network.state_dict())
        
        self.epsilon = max(self.epsilon_end, self.epsilon * self.epsilon_decay)
        
        return loss.item()
```

### 3.4 Rainbow DQN

**Rainbow DQN** 整合了多种改进：

1. **Double DQN**：解耦选择和评估
2. **Dueling DQN**：分解 Q 为 V 和 A
3. **优先级回放**：优先采样重要经验
4. **Noisy Net**：参数噪声探索
5. **分布式 RL**：学习价值分布
6. **n-step 学习**：多步回报

```python
class RainbowDQN(nn.Module):
    """Rainbow DQN 网络（简化版：整合 Double + Dueling）"""
    
    def __init__(self, state_dim, action_dim, hidden_dim=128):
        super().__init__()
        
        self.feature = nn.Sequential(
            nn.Linear(state_dim, hidden_dim),
            nn.ReLU(),
            nn.Linear(hidden_dim, hidden_dim),
            nn.ReLU()
        )
        
        self.value_stream = nn.Sequential(
            nn.Linear(hidden_dim, hidden_dim // 2),
            nn.ReLU(),
            nn.Linear(hidden_dim // 2, 1)
        )
        
        self.advantage_stream = nn.Sequential(
            nn.Linear(hidden_dim, hidden_dim // 2),
            nn.ReLU(),
            nn.Linear(hidden_dim // 2, action_dim)
        )
    
    def forward(self, x):
        features = self.feature(x)
        value = self.value_stream(features)
        advantage = self.advantage_stream(features)
        q_values = value + (advantage - advantage.mean(dim=1, keepdim=True))
        return q_values
```

### 4.2 算法特点总结

| 改进 | 解决的问题 | 核心思想 | 效果 |
|------|-----------|----------|------|
| **经验回放** | 样本相关性 | 存储经验，随机采样 | 稳定训练 |
| **目标网络** | 目标不稳定 | 独立网络计算目标 | 减少震荡 |
| **Double DQN** | 过高估计 | 解耦选择和评估 | 更准确 |
| **Dueling DQN** | 价值估计 | 分解 Q = V + A | 更高效 |
| **优先级回放** | 样本效率 | 优先采样重要经验 | 更快学习 |

## 5. DQN 的局限性

### 5.1 主要局限

| 局限 | 原因 | 后续解决方案 |
|------|------|--------------|
| **样本效率低** | 需要大量样本 | 分布式RL、模型RL |
| **过高估计** | max 操作 | Double DQN |
| **探索不足** | ε-贪心简单 | Noisy Net、好奇心 |
| **离散动作** | 输出层设计 | DDPG、SAC（连续） |
| **训练不稳定** | 超参数敏感 | PPO（下一讲） |

### 5.2 与策略梯度的对比

| 特性 | DQN（基于价值） | 策略梯度 |
|------|-----------------|----------|
| **动作空间** | 离散 | 离散/连续 |
| **策略类型** | 确定性（贪婪） | 随机 |
| **样本效率** | 高（离策略） | 低（在策略） |
| **训练稳定性** | 较不稳定 | 较稳定 |
| **探索方式** | ε-贪心 | 策略随机性 |

| 概念 | 定义 | 重要性 |
|------|------|--------|
| **函数逼近** | 用神经网络近似 Q 函数 | ⭐⭐⭐⭐⭐ |
| **经验回放** | 存储经验，随机采样 | ⭐⭐⭐⭐⭐ |
| **目标网络** | 独立网络计算目标 | ⭐⭐⭐⭐⭐ |
| **Double DQN** | 解耦选择和评估 | ⭐⭐⭐⭐ |
| **Dueling DQN** | Q = V + A | ⭐⭐⭐⭐ |
| **优先级回放** | 优先采样重要经验 | ⭐⭐⭐⭐ |

**核心公式速查**：

| 公式 | 表达式 |
|------|--------|
| DQN 目标 | $y = r + \gamma \max_{a'} Q(s', a'; \theta^-)$ |
| DQN 损失 | $L = \mathbb{E}[(y - Q(s, a; \theta))^2]$ |
| Double DQN 目标 | $y = r + \gamma Q(s', \arg\max Q(s', a; \theta); \theta^-)$ |
| Dueling DQN | $Q(s,a) = V(s) + A(s,a) - \frac{1}{\|A\|}\sum A$ |
| 优先级采样 | $P(i) = \frac{p_i^\alpha}{\sum_k p_k^\alpha}$ |

**一句话总结**：DQN 将 Q-Learning 与深度学习结合，通过经验回放和目标网络解决训练稳定性问题，开启了深度强化学习的时代。
