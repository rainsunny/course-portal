# Day 4：动态规划与表格方法

> **核心问题**：如果完全知道环境模型，如何高效计算最优策略？

## 引言：国际象棋的完美分析

假设你有一个超级计算机，知道国际象棋的所有规则：

- 完全知道每步之后的所有可能状态
- 完全知道每种局面的胜负判定
- 目标：找到从当前局面到胜利的最优路径

这就是**动态规划**要解决的问题：**已知环境模型，求最优策略**。

**关键洞察**：

1. **贝尔曼方程是递归的**：当前最优 = 即时奖励 + 折扣后的未来最优
2. **递归意味着可分解**：大问题可以拆成小问题
3. **小问题可缓存**：避免重复计算，这就是"动态规划"

Day 2 我们学习了贝尔曼方程，今天我们将**完全掌握动态规划算法**——这是强化学习的算法基石。

## 1. 动态规划基础

### 1.1 什么是动态规划？

**动态规划（Dynamic Programming, DP）**是一种算法设计范式：

> 将复杂问题分解为重叠子问题，通过求解子问题并存储结果来高效求解原问题。

**关键特征**：
1. **最优子结构**：问题的最优解包含子问题的最优解
2. **重叠子问题**：相同的子问题会被多次求解

**经典例子**：斐波那契数列

```python
# 朴素递归：O(2^n)
def fib_slow(n):
    if n <= 1:
        return n
    return fib_slow(n-1) + fib_slow(n-2)

# 动态规划：O(n)
def fib_dp(n):
    if n <= 1:
        return n
    dp = [0] * (n + 1)
    dp[1] = 1
    for i in range(2, n + 1):
        dp[i] = dp[i-1] + dp[i-2]
    return dp[n]
```

### 1.2 为什么强化学习适合动态规划？

**强化学习的贝尔曼方程天然满足动态规划的条件**：

1. **最优子结构**：
   $$V^*(s) = \max_a \left[ R(s,a) + \gamma \sum_{s'} P(s'|s,a) V^*(s') \right]$$
   最优价值函数由子问题（下一状态的价值）的最优解构成。

2. **重叠子问题**：
   - 不同状态可能转移到相同状态
   - 相同状态的价值会被多次访问

**前提条件**：**需要已知的环境模型**
- 状态转移概率 P(s'|s,a)
- 奖励函数 R(s,a,s')

这意味着动态规划是**有模型（Model-Based）方法**。

### 1.3 动态规划的两大问题

| 问题 | 描述 | 输入 | 输出 |
|------|------|------|------|
| **预测问题** | 给定策略，计算价值函数 | MDP + 策略 π | V^π 或 Q^π |
| **控制问题** | 找到最优策略 | MDP | π* 和 V* 或 Q* |

**预测问题**（策略评估）：

$$\text{给定 } \pi, \text{ 计算 } V^\pi(s) = \mathbb{E}_\pi\left[\sum_{t=0}^\infty \gamma^t R_t \bigg| S_0 = s\right]$$

**控制问题**（策略优化）：

$$\text{找到 } \pi^* = \arg\max_\pi V^\pi(s), \forall s$$

## 2. 策略评估：计算 V^π

### 2.1 迭代策略评估算法

**目标**：给定策略 π，计算其价值函数 V^π。

**方法**：迭代应用贝尔曼期望方程。

**算法**：

```
输入：MDP (S, A, P, R, γ)，策略 π
输出：V^π

初始化：V(s) = 0 对所有 s ∈ S

重复：
    对于每个状态 s：
        V_new(s) = Σ_a π(a|s) Σ_s' P(s'|s,a)[R(s,a,s') + γ V(s')]
    
    δ = max_s |V_new(s) - V(s)|
    V = V_new
    
直到 δ < θ（收敛）
```

### 2.2 为什么迭代有效？

**压缩映射定理**：

定义贝尔曼算子 $T_\pi$：

$$[T_\pi V](s) = \sum_a \pi(a|s) \sum_{s'} P(s'|s,a) [R(s,a,s') + \gamma V(s')]$$

**定理**：$T_\pi$ 是 γ-压缩映射，即：

$$\|T_\pi V - T_\pi V'\|_\infty \leq \gamma \|V - V'\|_\infty$$

**推论**：
- $T_\pi$ 有唯一不动点 $V^\pi$
- 迭代 $V_{k+1} = T_\pi V_k$ 收敛到 $V^\pi$

### 2.3 完整实现

```python
import numpy as np
from typing import Tuple, Dict, List

class MDP:
    """马尔可夫决策过程"""
    
    def __init__(
        self, 
        states: List, 
        actions: List,
        transitions: Dict,
        rewards: Dict,
        gamma: float = 0.9,
        terminal_states: List = None
    ):
        self.states = states
        self.actions = actions
        self.P = transitions  # P[s][a][s'] = 概率
        self.R = rewards      # R[s][a][s'] = 奖励
        self.gamma = gamma
        self.terminal_states = terminal_states or []
    
    def get_next_states(self, state, action) -> List[Tuple]:
        """获取 (下一状态, 概率, 奖励) 列表"""
        result = []
        for next_state in self.states:
            prob = self.P[state][action].get(next_state, 0)
            if prob > 0:
                reward = self.R[state][action].get(next_state, 0)
                result.append((next_state, prob, reward))
        return result
    
    def is_terminal(self, state) -> bool:
        return state in self.terminal_states


def policy_evaluation(
    mdp: MDP, 
    policy: Dict, 
    theta: float = 1e-6, 
    max_iterations: int = 1000
) -> Tuple[Dict, int]:
    """
    迭代策略评估
    
    参数：
        mdp: MDP 环境
        policy: 策略，policy[s][a] = 选择动作 a 的概率
        theta: 收敛阈值
        max_iterations: 最大迭代次数
    
    返回：
        V: 状态价值函数
        iterations: 实际迭代次数
    """
    # 初始化价值函数
    V = {s: 0.0 for s in mdp.states}
    
    for iteration in range(max_iterations):
        delta = 0
        
        for state in mdp.states:
            if mdp.is_terminal(state):
                continue
            
            v_old = V[state]
            v_new = 0
            
            # 贝尔曼期望方程
            for action in mdp.actions:
                action_prob = policy[state][action]
                
                for next_state, trans_prob, reward in mdp.get_next_states(state, action):
                    v_new += action_prob * trans_prob * (reward + mdp.gamma * V[next_state])
            
            V[state] = v_new
            delta = max(delta, abs(v_new - v_old))
        
        if delta < theta:
            print(f"策略评估收敛，迭代 {iteration + 1} 次")
            return V, iteration + 1
    
    print(f"策略评估未收敛，达到最大迭代次数 {max_iterations}")
    return V, max_iterations
```

### 2.4 示例：网格世界

```python
def create_gridworld(size=4):
    """创建一个简单的网格世界 MDP"""
    states = [(i, j) for i in range(size) for j in range(size)]
    actions = ['up', 'down', 'left', 'right']
    
    # 目标和障碍
    goal = (0, size - 1)
    obstacles = [(1, 1)]
    terminal_states = [goal]
    
    # 转移概率（确定性）
    transitions = {}
    rewards = {}
    
    for s in states:
        transitions[s] = {}
        rewards[s] = {}
        
        for a in actions:
            transitions[s][a] = {}
            rewards[s][a] = {}
            
            # 计算下一位置
            i, j = s
            if a == 'up':
                next_s = (max(0, i - 1), j)
            elif a == 'down':
                next_s = (min(size - 1, i + 1), j)
            elif a == 'left':
                next_s = (i, max(0, j - 1))
            else:  # right
                next_s = (i, min(size - 1, j + 1))
            
            # 碰到障碍物，原地不动
            if next_s in obstacles:
                next_s = s
            
            # 设置转移和奖励
            transitions[s][a][next_s] = 1.0
            
            if next_s == goal:
                rewards[s][a][next_s] = 1.0  # 到达目标奖励
            elif next_s == s and s not in terminal_states:
                rewards[s][a][next_s] = -0.1  # 碰墙惩罚
            else:
                rewards[s][a][next_s] = 0.0
    
    return MDP(states, actions, transitions, rewards, gamma=0.9, terminal_states=terminal_states)


# 创建环境并运行策略评估
mdp = create_gridworld(size=4)

# 定义随机策略
random_policy = {
    s: {a: 1.0 / len(mdp.actions) for a in mdp.actions} 
    for s in mdp.states
}

# 策略评估
V, iters = policy_evaluation(mdp, random_policy)

# 可视化
def print_value_function(V, size=4):
    print("价值函数 V^π:")
    for i in range(size):
        row = [f"{V[(i, j)]:6.3f}" for j in range(size)]
        print(" ".join(row))

print_value_function(V, size=4)
```

## 3. 策略改进与策略迭代

### 3.1 策略改进定理

**定理**：设 π 和 π' 是两个策略，如果对所有状态 s：

$$Q^\pi(s, \pi'(s)) \geq V^\pi(s)$$

则 π' 优于或等于 π，即：

$$V^{\pi'}(s) \geq V^\pi(s), \forall s$$

如果至少在一个状态上严格不等，则 π' 严格优于 π。

**证明思路**：
- 从 Q 的不等式出发
- 反复应用贝尔曼方程展开
- 利用折扣因子的性质

**策略改进方法**：贪婪改进

$$\pi'(s) = \arg\max_a Q^\pi(s, a) = \arg\max_a \sum_{s'} P(s'|s,a) [R(s,a,s') + \gamma V^\pi(s')]$$

### 3.2 策略迭代算法

**思想**：交替进行策略评估和策略改进。

```
初始化：任意策略 π₀

循环 k = 0, 1, 2, ...
    1. 策略评估：计算 V^{π_k}
    2. 策略改进：
       π_{k+1}(s) = argmax_a Σ_s' P(s'|s,a)[R + γ V^{π_k}(s')]
    
    如果 π_{k+1} = π_k，停止

输出：π*
```

**收敛性**：
- 每次改进，价值函数严格增加（或不变）
- 有限 MDP 只有有限个确定性策略
- 因此有限步内收敛

### 3.3 完整实现

```python
def policy_improvement(mdp: MDP, V: Dict) -> Tuple[Dict, bool]:
    """
    策略改进
    
    参数：
        mdp: MDP 环境
        V: 状态价值函数
    
    返回：
        policy: 改进后的策略
        stable: 策略是否稳定（未改变）
    """
    policy = {s: {a: 0.0 for a in mdp.actions} for s in mdp.states}
    stable = True
    
    for state in mdp.states:
        if mdp.is_terminal(state):
            continue
        
        # 计算每个动作的 Q 值
        action_values = {}
        for action in mdp.actions:
            q = 0
            for next_state, trans_prob, reward in mdp.get_next_states(state, action):
                q += trans_prob * (reward + mdp.gamma * V[next_state])
            action_values[action] = q
        
        # 找到最优动作
        max_q = max(action_values.values())
        best_actions = [a for a, q in action_values.items() if q == max_q]
        
        # 设置新策略（等概率选择最优动作）
        for action in mdp.actions:
            new_prob = 1.0 / len(best_actions) if action in best_actions else 0.0
            policy[state][action] = new_prob
        
        # 检查策略是否改变
        # （需要比较新旧策略，这里简化处理）
    
    return policy, stable


def policy_iteration(
    mdp: MDP, 
    theta: float = 1e-6, 
    max_iterations: int = 100
) -> Tuple[Dict, Dict, int]:
    """
    策略迭代
    
    参数：
        mdp: MDP 环境
        theta: 策略评估收敛阈值
        max_iterations: 最大迭代轮数
    
    返回：
        V: 最优价值函数
        policy: 最优策略
        iterations: 迭代轮数
    """
    # 初始化随机策略
    policy = {
        s: {a: 1.0 / len(mdp.actions) for a in mdp.actions} 
        for s in mdp.states
    }
    
    for iteration in range(max_iterations):
        # 策略评估
        V, _ = policy_evaluation(mdp, policy, theta)
        
        # 策略改进
        new_policy, stable = policy_improvement(mdp, V)
        
        # 检查策略是否改变
        policy_changed = False
        for s in mdp.states:
            for a in mdp.actions:
                if abs(policy[s][a] - new_policy[s][a]) > 1e-10:
                    policy_changed = True
                    break
            if policy_changed:
                break
        
        policy = new_policy
        
        print(f"策略迭代第 {iteration + 1} 轮")
        
        if not policy_changed:
            print(f"策略迭代收敛，共 {iteration + 1} 轮")
            return V, policy, iteration + 1
    
    return V, policy, max_iterations


# 运行策略迭代
V_pi, policy_pi, iters_pi = policy_iteration(mdp)

print("\n最优价值函数：")
print_value_function(V_pi, size=4)

print("\n最优策略：")
def print_policy(policy, size=4):
    action_symbols = {'up': '↑', 'down': '↓', 'left': '←', 'right': '→'}
    for i in range(size):
        row = []
        for j in range(size):
            s = (i, j)
            if s == (0, 3):  # 目标
                row.append('G')
            elif s == (1, 1):  # 障碍
                row.append('█')
            else:
                best_action = max(policy[s].items(), key=lambda x: x[1])[0]
                row.append(action_symbols[best_action])
        print(" ".join(row))

print_policy(policy_pi, size=4)
```

## 4. 价值迭代

### 4.1 从策略迭代到价值迭代

**策略迭代的问题**：
- 每轮需要完整的策略评估
- 策略评估可能需要很多次迭代

**关键洞察**：
- 不需要策略评估完全收敛
- 只需要一次贝尔曼更新，然后立即改进策略

**价值迭代**：直接迭代应用贝尔曼最优方程。

### 4.2 价值迭代算法

```
初始化：V(s) = 0 对所有 s ∈ S

重复：
    对于每个状态 s：
        V_new(s) = max_a Σ_s' P(s'|s,a)[R(s,a,s') + γ V(s')]
    
    δ = max_s |V_new(s) - V(s)|
    V = V_new
    
直到 δ < θ

输出策略：π*(s) = argmax_a Σ_s' P(s'|s,a)[R(s,a,s') + γ V*(s')]
```

### 4.3 完整实现

```python
def value_iteration(
    mdp: MDP, 
    theta: float = 1e-6, 
    max_iterations: int = 1000
) -> Tuple[Dict, Dict, int]:
    """
    价值迭代
    
    参数：
        mdp: MDP 环境
        theta: 收敛阈值
        max_iterations: 最大迭代次数
    
    返回：
        V: 最优价值函数
        policy: 最优策略
        iterations: 实际迭代次数
    """
    # 初始化价值函数
    V = {s: 0.0 for s in mdp.states}
    
    for iteration in range(max_iterations):
        delta = 0
        
        for state in mdp.states:
            if mdp.is_terminal(state):
                continue
            
            v_old = V[state]
            
            # 贝尔曼最优方程
            action_values = []
            for action in mdp.actions:
                q = 0
                for next_state, trans_prob, reward in mdp.get_next_states(state, action):
                    q += trans_prob * (reward + mdp.gamma * V[next_state])
                action_values.append(q)
            
            V[state] = max(action_values)
            delta = max(delta, abs(V[state] - v_old))
        
        if delta < theta:
            print(f"价值迭代收敛，迭代 {iteration + 1} 次")
            break
    
    # 提取最优策略
    policy = {s: {a: 0.0 for a in mdp.actions} for s in mdp.states}
    
    for state in mdp.states:
        if mdp.is_terminal(state):
            continue
        
        action_values = {}
        for action in mdp.actions:
            q = 0
            for next_state, trans_prob, reward in mdp.get_next_states(state, action):
                q += trans_prob * (reward + mdp.gamma * V[next_state])
            action_values[action] = q
        
        max_q = max(action_values.values())
        best_actions = [a for a, q in action_values.items() if q == max_q]
        
        for action in mdp.actions:
            policy[state][action] = 1.0 / len(best_actions) if action in best_actions else 0.0
    
    return V, policy, iteration + 1


# 运行价值迭代
V_vi, policy_vi, iters_vi = value_iteration(mdp)

print("\n最优价值函数（价值迭代）：")
print_value_function(V_vi, size=4)

print("\n最优策略（价值迭代）：")
print_policy(policy_vi, size=4)
```

### 4.4 价值迭代的收敛性

**定理**：价值迭代以速率 γ 收敛：

$$\|V_{k+1} - V^*\|_\infty \leq \gamma \|V_k - V^*\|_\infty$$

**达到 ε-最优的迭代次数**：

$$k \geq \frac{\log(\epsilon(1-\gamma)/\|V_1\|)}{\log(\gamma)}$$

**例子**：
- γ = 0.9，需要约 60 次迭代达到 0.01 精度
- γ = 0.99，需要约 600 次迭代

## 5. 异步动态规划

### 5.1 动态规划的效率问题

**标准 DP 的问题**：
- 每次迭代要遍历所有状态
- 状态空间大时，单次迭代就很慢

**异步 DP 的思想**：
- 不需要每次更新所有状态
- 只更新部分状态，按任意顺序
- 只要保证每个状态被无限次访问，仍然收敛

### 5.2 原位动态规划（In-Place DP）

**标准价值迭代**：

```python
# 需要两个数组
V_new = {}
for s in states:
    V_new[s] = max_a Σ P(s'|s,a)[R + γ V(s')]
V = V_new
```

**原位更新**：

```python
# 只需要一个数组
for s in states:
    V[s] = max_a Σ P(s'|s,a)[R + γ V(s')]  # 直接更新 V
```

**优点**：
- 节省内存
- 收敛更快（新值立即可用）

### 5.3 优先级扫描（Prioritized Sweeping）

**思想**：优先更新"影响大"的状态。

**算法**：

1. 维护一个优先级队列
2. 更新某状态后，计算对前驱状态的影响
3. 如果影响大，将前驱加入队列

```python
def prioritized_sweeping(mdp, theta=1e-6, max_iterations=1000):
    """优先级扫描价值迭代"""
    import heapq
    
    V = {s: 0.0 for s in mdp.states}
    priority_queue = []
    
    # 初始化：所有状态加入队列
    for s in mdp.states:
        heapq.heappush(priority_queue, (0, s))
    
    for iteration in range(max_iterations):
        if not priority_queue:
            break
        
        # 取出优先级最高的状态
        _, state = heapq.heappop(priority_queue)
        
        if mdp.is_terminal(state):
            continue
        
        # 计算新的价值
        v_old = V[state]
        action_values = []
        for action in mdp.actions:
            q = 0
            for next_state, trans_prob, reward in mdp.get_next_states(state, action):
                q += trans_prob * (reward + mdp.gamma * V[next_state])
            action_values.append(q)
        V[state] = max(action_values)
        
        # 计算变化
        delta = abs(V[state] - v_old)
        
        # 更新前驱状态的优先级
        for pred_state in mdp.states:
            for action in mdp.actions:
                if mdp.P[pred_state][action].get(state, 0) > 0:
                    # pred_state 是 state 的前驱
                    priority = -delta * mdp.P[pred_state][action][state]
                    heapq.heappush(priority_queue, (priority, pred_state))
        
        if delta < theta:
            break
    
    return V
```

### 5.4 实时动态规划（Real-Time DP）

**思想**：只更新智能体实际访问的状态。

**算法**：
1. 智能体与环境交互
2. 每步更新当前状态的价值
3. 不需要遍历所有状态

```python
def real_time_dp(mdp, episodes=1000, theta=1e-6):
    """实时动态规划"""
    V = {s: 0.0 for s in mdp.states}
    
    for episode in range(episodes):
        state = np.random.choice(mdp.states)  # 随机起始状态
        
        while not mdp.is_terminal(state):
            # 选择动作（可以使用任何探索策略）
            action = np.random.choice(mdp.actions)
            
            # 执行动作
            next_states = mdp.get_next_states(state, action)
            next_state = np.random.choice(
                [s for s, _, _ in next_states],
                p=[p for _, p, _ in next_states]
            )
            reward = [r for s, p, r in next_states if s == next_state][0]
            
            # 更新当前状态的价值
            v_old = V[state]
            action_values = []
            for a in mdp.actions:
                q = 0
                for ns, trans_prob, r in mdp.get_next_states(state, a):
                    q += trans_prob * (r + mdp.gamma * V[ns])
                action_values.append(q)
            V[state] = max(action_values)
            
            # 移动到下一状态
            state = next_state
    
    return V
```

## 6. 维度灾难与解决方案

### 6.1 维度灾难

**问题**：状态空间随变量数量指数增长。

**例子**：
- 国际象棋：约 10^47 个状态
- 围棋：约 10^170 个状态
- 机器人控制：连续状态空间

**DP 的复杂度**：

| 算法 | 时间复杂度（每次迭代） |
|------|------------------------|
| 策略评估 | O(\|S\|²\|A\|) |
| 策略迭代 | O(\|S\|²\|A\| × 轮数) |
| 价值迭代 | O(\|S\|²\|A\| × 迭代次数) |

**问题**：当 \|S\| 很大时，DP 不可行。

### 6.2 解决方案概览

| 方法 | 思想 | 适用场景 |
|------|------|----------|
| **函数逼近** | 用参数化函数表示 V 或 Q | 大规模或连续状态空间 |
| **采样方法** | 只访问实际到达的状态 | 无模型场景 |
| **分层 DP** | 分解问题，分层求解 | 结构化问题 |
| **抽象/聚合** | 合并相似状态 | 可抽象的问题 |

**后续课程**：
- Day 5-6：无模型方法（不需要环境模型）
- Day 7：深度 Q 网络（函数逼近）

def state_aggregation_dp(mdp, aggregation_func, theta=1e-6):

    """

    state_groups = {}
    for s in mdp.states:
        group = aggregation_func(s)
        if group not in state_groups:

    P_agg = {g: {a: {} for a in mdp.actions} for g in groups}

        for a in mdp.actions:
            for g_next in groups:
                # 计算聚合转移概率
                prob = 0
                reward = 0
                count = 0
                
                for s in state_groups[g]:
                    for s_next in state_groups[g_next]:

                        r = mdp.R[s][a].get(s_next, 0)

                        reward += p * r
                        count += p
                
                if prob > 0:
                    P_agg[g][a][g_next] = prob / len(state_groups[g])
                    R_agg[g][a][g_next] = reward / prob
    
    # 在聚合 MDP 上运行价值迭代
    # ...
    
    return P_agg, R_agg

def grid_aggregation(state, grid_size=2):

## 7. 算法比较与选择

### 7.1 策略迭代 vs 价值迭代

| 特性 | 策略迭代 | 价值迭代 |
|------|----------|----------|
| **每次迭代复杂度** | O(\|S\|³\|A\|)（求解线性方程组） | O(\|S\|²\|A\|) |
| **迭代次数** | 少（策略改进快速收敛） | 多（价值收敛慢） |
| **实现复杂度** | 较高（需要求解方程组） | 低（直接迭代） |
| **适用场景** | 小规模 MDP | 大规模 MDP |

**经验法则**：
- 状态空间小（< 1000）：策略迭代更快
- 状态空间大：价值迭代更实用
- 需要精确策略：策略迭代
- 需要快速近似：价值迭代

| 算法 | 核心 | 每次迭代 | 迭代次数 | 适用场景 |
|------|------|----------|----------|----------|
| 策略评估 | 贝尔曼期望方程 | O(\|S\|²\|A\|) | 多 | 计算策略价值 |
| 策略迭代 | 评估→改进循环 | O(\|S\|³\|A\|) | 少 | 小规模精确求解 |
| 价值迭代 | 贝尔曼最优方程 | O(\|S\|²\|A\|) | 多 | 大规模近似求解 |

**核心公式速查**：

| 公式 | 表达式 |
|------|--------|
| 贝尔曼期望方程 | $V^\pi(s) = \sum_a \pi(a|s) \sum_{s'} P(s'|s,a)[R + \gamma V^\pi(s')]$ |
| 贝尔曼最优方程 | $V^*(s) = \max_a \sum_{s'} P(s'|s,a)[R + \gamma V^*(s')]$ |
| 策略改进 | $\pi'(s) = \arg\max_a Q^\pi(s, a)$ |
| 压缩映射 | $\|TV - TV'\|_\infty \leq \gamma \|V - V'\|_\infty$ |

**一句话总结**：动态规划利用贝尔曼方程的递归结构，通过迭代求解最优价值函数和最优策略，是强化学习中最基本的算法范式。
