# Day 10：多智能体RL与前沿方向

> **核心问题**：当多个智能体同时学习和交互时，会发生什么？

## 引言：从围棋到星际争霸

2016年，AlphaGo 击败李世石，标志着单智能体 RL 的巅峰。但现实世界远比围棋复杂：

- **自动驾驶**：多辆车需要协调
- **电商定价**：多个商家博弈
- **网络路由**：多个节点协作
- **人工智能安全**：AI 与人类共存

这些场景的核心问题：**多个智能体同时学习、同时决策**。

**多智能体强化学习（Multi-Agent RL, MARL）** 研究的就是这类问题：

- **合作**：多智能体共同最大化团队奖励
- **竞争**：零和博弈，一方收益等于另一方损失
- **混合**：既有合作又有竞争

今天，我们探索 MARL 的核心理论和前沿方向。

---

## 1. 多智能体强化学习基础

### 1.1 从MDP到马尔可夫博弈

**单智能体 MDP**：

$$\text{MDP} = (\mathcal{S}, \mathcal{A}, P, R, \gamma)$$

**多智能体马尔可夫博弈（Markov Game）**：

$$\text{MG} = (\mathcal{S}, \{\mathcal{A}_i\}_{i=1}^N, P, \{R_i\}_{i=1}^N, \gamma)$$

其中：
- $\mathcal{S}$：状态空间
- $\mathcal{A}_i$：智能体 i 的动作空间
- $P(s'|s, a_1, ..., a_N)$：状态转移概率，依赖于所有智能体的动作
- $R_i(s, a_1, ..., a_N)$：智能体 i 的奖励函数

**关键差异**：状态转移和奖励依赖于所有智能体的动作！

### 1.2 博弈论基础

**纳什均衡**：在纳什均衡中，没有任何智能体可以通过单方面改变策略来提高收益。

$$\pi_i^* \in \arg\max_{\pi_i} V_i(\pi_i, \pi_{-i}^*)$$

其中 $\pi_{-i}$ 表示除智能体 i 外所有智能体的策略。

**经典博弈例子**：

**囚徒困境**：

```
              囚徒B
           合作      背叛
囚徒A  合作  (-1,-1)  (-3, 0)
       背叛  ( 0,-3)  (-2,-2)
```

纳什均衡：（背叛，背叛），但不是帕累托最优！

**性别战争**：

```
             伴侣B
         足球    购物
伴侣A 足球  (2,1)  (0,0)
      购物  (0,0)  (1,2)
```

两个纳什均衡：（足球，足球）和（购物，购物）

### 1.3 MARL的分类

| 类型 | 描述 | 例子 |
|------|------|------|
| **完全合作** | 所有智能体共享奖励 | 团队游戏 |
| **完全竞争** | 零和博弈 | 棋类游戏 |
| **混合动机** | 合作与竞争并存 | 市场竞争 |
| **完全信息** | 所有智能体知道完整状态 | 国际象棋 |
| **不完全信息** | 智能体只能观测部分状态 | 扑克牌 |

### 1.4 MARL的挑战

**挑战一：非平稳性（Non-stationarity）**

单智能体 RL 假设环境是稳定的（马尔可夫性）。但在 MARL 中，其他智能体也在学习和改变策略，从某个智能体的视角看，环境是不断变化的。

**挑战二：维度灾难**

联合动作空间随智能体数量指数增长：

$$|\mathcal{A}| = \prod_{i=1}^N |\mathcal{A}_i|$$

**挑战三：信用分配**

在合作场景中，团队奖励如何分配给每个智能体？谁贡献更大？

**挑战四：均衡选择**

多个纳什均衡时，智能体如何协调选择同一个均衡？

## 2. 独立学习与合作学习

### 2.1 独立Q学习（IQL）

**最简单的 MARL 方法**：每个智能体独立学习，忽略其他智能体的存在。

$$Q_i(s, a_i) \leftarrow Q_i(s, a_i) + \alpha [r_i + \gamma \max_{a_i'} Q_i(s', a_i') - Q_i(s, a_i)]$$

**优点**：
- 简单，易实现
- 可扩展性好

**缺点**：
- 忽略其他智能体的影响
- 环境非平稳，收敛性无保证

```python
class IndependentQLearning:
    """独立 Q 学习"""
    
    def __init__(self, n_agents, state_dim, action_dims, alpha=0.1, gamma=0.99, epsilon=0.1):
        self.n_agents = n_agents
        self.action_dims = action_dims
        
        # 每个智能体独立 Q 表
        self.Q_tables = [
            np.zeros((state_dim, action_dim)) 
            for action_dim in action_dims
        ]
        
        self.alpha = alpha
        self.gamma = gamma
        self.epsilon = epsilon
    
    def select_actions(self, state):
        """每个智能体独立选择动作"""
        actions = []
        for i in range(self.n_agents):
            if np.random.random() < self.epsilon:
                action = np.random.randint(self.action_dims[i])
            else:
                action = np.argmax(self.Q_tables[i][state])
            actions.append(action)
        return actions
    
    def update(self, state, actions, rewards, next_state, dones):
        """每个智能体独立更新"""
        for i in range(self.n_agents):
            # Q 学习更新
            best_next = np.max(self.Q_tables[i][next_state])
            td_target = rewards[i] + self.gamma * best_next * (1 - dones[i])
            td_error = td_target - self.Q_tables[i][state, actions[i]]
            self.Q_tables[i][state, actions[i]] += self.alpha * td_error
```

### 2.2 联合动作学习（JAL）

**思想**：学习联合动作的价值。

$$Q(s, \mathbf{a}) \leftarrow Q(s, \mathbf{a}) + \alpha [r + \gamma \max_{\mathbf{a}'} Q(s', \mathbf{a}') - Q(s, \mathbf{a})]$$

其中 $\mathbf{a} = (a_1, a_2, ..., a_N)$ 是联合动作。

**优点**：
- 考虑了智能体间的交互
- 可以找到最优联合策略

**缺点**：
- 维度灾难：联合动作空间指数增长
- 不可扩展到多智能体

```python
class JointActionQLearning:
    """联合动作 Q 学习"""
    
    def __init__(self, n_agents, state_dim, action_dims, alpha=0.1, gamma=0.99):
        self.n_agents = n_agents
        self.action_dims = action_dims
        self.joint_action_dim = np.prod(action_dims)
        
        # 联合 Q 表
        self.Q_table = np.zeros((state_dim, self.joint_action_dim))
        
        self.alpha = alpha
        self.gamma = gamma
    
    def action_to_index(self, actions):
        """联合动作转索引"""
        index = 0
        for i, a in enumerate(actions):
            index = index * self.action_dims[i] + a
        return index
    
    def index_to_action(self, index):
        """索引转联合动作"""
        actions = []
        for i in reversed(range(self.n_agents)):
            actions.append(index % self.action_dims[i])
            index //= self.action_dims[i]
        return actions[::-1]
    
    def select_joint_action(self, state):
        """选择联合动作"""
        joint_index = np.argmax(self.Q_table[state])
        return self.index_to_action(joint_index)
    
    def update(self, state, actions, reward, next_state, done):
        """更新联合 Q 值"""
        action_index = self.action_to_index(actions)
        best_next = np.max(self.Q_table[next_state])
        td_target = reward + self.gamma * best_next * (1 - done)
        self.Q_table[state, action_index] += self.alpha * (td_target - self.Q_table[state, action_index])
```

### 2.3 价值分解（Value Decomposition）

**问题**：如何结合 IQL 的可扩展性和 JAL 的全局最优？

**价值分解的思想**：将联合价值分解为个体价值的组合。

**线性分解**：

$$Q_{tot}(s, \mathbf{a}) = \sum_{i=1}^N Q_i(s, a_i)$$

**VDN（Value Decomposition Networks）**：

$$Q_{tot}(s, \mathbf{a}) \approx \sum_{i=1}^N Q_i(\tau_i, a_i)$$

其中 $\tau_i$ 是智能体 i 的观测历史。

```python
class VDN:
    """Value Decomposition Networks"""
    
    def __init__(self, n_agents, obs_dim, action_dim, hidden_dim=64):
        self.n_agents = n_agents
        
        # 每个智能体的 Q 网络
        self.q_networks = [
            nn.Sequential(
                nn.Linear(obs_dim, hidden_dim),
                nn.ReLU(),
                nn.Linear(hidden_dim, action_dim)
            ) for _ in range(n_agents)
        ]
        
        # 共享优化器
        self.optimizer = optim.Adam(
            sum([list(net.parameters()) for net in self.q_networks], []),
            lr=1e-3
        )
    
    def get_q_values(self, observations):
        """获取所有智能体的 Q 值"""
        q_values = []
        for i, obs in enumerate(observations):
            q = self.q_networks[i](torch.FloatTensor(obs))
            q_values.append(q)
        return q_values
    
    def get_joint_q(self, observations, actions):
        """计算联合 Q 值"""
        q_values = self.get_q_values(observations)
        
        # 选择对应动作的 Q 值
        selected_q = []
        for i, (q, a) in enumerate(zip(q_values, actions)):
            selected_q.append(q[a])
        
        # 求和得到联合 Q 值
        joint_q = sum(selected_q)
        return joint_q
    
    def update(self, batch):
        """更新网络"""
        obs, actions, rewards, next_obs, dones = batch
        
        # 当前 Q 值
        current_q = self.get_joint_q(obs, actions)
        
        # 目标 Q 值
        with torch.no_grad():
            next_q_values = self.get_q_values(next_obs)
            max_next_q = sum([q.max() for q in next_q_values])
            target_q = rewards.sum() + 0.99 * max_next_q * (1 - dones[0])
        
        # 损失
        loss = nn.MSELoss()(current_q, target_q)
        
        self.optimizer.zero_grad()
        loss.backward()
        self.optimizer.step()
        
        return loss.item()
```

### 2.4 QMIX：非线性价值分解

**VDN 的局限**：线性分解假设太强，限制了表达能力。

**QMIX**：使用神经网络学习更复杂的分解。

$$Q_{tot}(s, \mathbf{a}) = f(Q_1, Q_2, ..., Q_N; s)$$

关键约束：**单调性**

$$\frac{\partial Q_{tot}}{\partial Q_i} \geq 0, \forall i$$

这保证了：如果个体 Q 值增加，联合 Q 值也增加，从而可以使用贪婪策略。

```python
class QMIXNetwork(nn.Module):
    """QMIX 混合网络"""
    
    def __init__(self, n_agents, state_dim, hidden_dim=64):
        super().__init__()
        self.n_agents = n_agents
        self.hidden_dim = hidden_dim
        
        # 超网络：从状态生成混合权重
        self.hyper_w1 = nn.Sequential(
            nn.Linear(state_dim, hidden_dim),
            nn.ReLU(),
            nn.Linear(hidden_dim, n_agents * hidden_dim)
        )
        
        self.hyper_w2 = nn.Sequential(
            nn.Linear(state_dim, hidden_dim),
            nn.ReLU(),
            nn.Linear(hidden_dim, hidden_dim * 1)
        )
        
        # 超网络：从状态生成偏置
        self.hyper_b1 = nn.Linear(state_dim, hidden_dim)
        self.hyper_b2 = nn.Sequential(
            nn.Linear(state_dim, hidden_dim),
            nn.ReLU(),
            nn.Linear(hidden_dim, 1)
        )
    
    def forward(self, q_values, state):
        """
        q_values: [batch, n_agents] 个体 Q 值
        state: [batch, state_dim] 全局状态
        """
        batch_size = q_values.size(0)
        
        # 生成混合权重（保证非负）
        w1 = torch.abs(self.hyper_w1(state))  # [batch, n_agents * hidden_dim]
        w2 = torch.abs(self.hyper_w2(state))  # [batch, hidden_dim]
        
        # 重塑
        w1 = w1.view(batch_size, self.n_agents, self.hidden_dim)
        w2 = w2.view(batch_size, self.hidden_dim, 1)
        
        # 偏置
        b1 = self.hyper_b1(state)  # [batch, hidden_dim]
        b2 = self.hyper_b2(state)  # [batch, 1]
        
        # 两层混合网络
        hidden = torch.relu(torch.bmm(q_values.unsqueeze(1), w1).squeeze(1) + b1)
        q_tot = torch.bmm(hidden.unsqueeze(1), w2).squeeze(1) + b2
        
        return q_tot


class QMIX:
    """QMIX 算法"""
    
    def __init__(self, n_agents, obs_dim, state_dim, action_dim, hidden_dim=64):
        self.n_agents = n_agents
        
        # 个体 Q 网络
        self.agent_q = AgentQNetwork(obs_dim, action_dim, hidden_dim)
        self.target_agent_q = AgentQNetwork(obs_dim, action_dim, hidden_dim)
        self.target_agent_q.load_state_dict(self.agent_q.state_dict())
        
        # 混合网络
        self.mixer = QMIXNetwork(n_agents, state_dim, hidden_dim)
        self.target_mixer = QMIXNetwork(n_agents, state_dim, hidden_dim)
        self.target_mixer.load_state_dict(self.mixer.state_dict())
        
        self.optimizer = optim.Adam(
            list(self.agent_q.parameters()) + list(self.mixer.parameters()),
            lr=1e-3
        )
    
    def select_actions(self, observations, epsilon=0.1):
        """ε-贪心动作选择"""
        actions = []
        for i, obs in enumerate(observations):
            q = self.agent_q(torch.FloatTensor(obs))
            if np.random.random() < epsilon:
                action = np.random.randint(q.size(-1))
            else:
                action = q.argmax().item()
            actions.append(action)
        return actions
    
    def update(self, batch):
        """更新网络"""
        obs, actions, rewards, next_obs, next_state, dones, state = batch
        
        # 当前 Q 值
        current_qs = []
        for i in range(self.n_agents):
            q = self.agent_q(obs[:, i, :])
            q_a = q.gather(1, actions[:, i].unsqueeze(1))
            current_qs.append(q_a)
        current_qs = torch.cat(current_qs, dim=1)
        current_q_tot = self.mixer(current_qs, state)
        
        # 目标 Q 值
        with torch.no_grad():
            next_qs = []
            for i in range(self.n_agents):
                q = self.target_agent_q(next_obs[:, i, :])
                q_a = q.max(dim=1, keepdim=True)[0]
                next_qs.append(q_a)
            next_qs = torch.cat(next_qs, dim=1)
            next_q_tot = self.target_mixer(next_qs, next_state)
            target_q_tot = rewards + 0.99 * next_q_tot * (1 - dones)
        
        # 损失
        loss = nn.MSELoss()(current_q_tot, target_q_tot)
        
        self.optimizer.zero_grad()
        loss.backward()
        self.optimizer.step()
        
        return loss.item()
```

## 3. 博弈论与MARL

### 3.1 纳什均衡计算

**纳什 Q 学习**：学习纳什均衡策略。

**思想**：假设其他智能体玩纳什均衡策略，学习自己的最优回应。

```python
class NashQLearning:
    """纳什 Q 学习"""
    
    def __init__(self, n_agents, state_dim, action_dims, alpha=0.1, gamma=0.99):
        self.n_agents = n_agents
        self.action_dims = action_dims
        
        # 联合 Q 表（每个智能体维护）
        self.Q_tables = [
            np.zeros((state_dim, np.prod(action_dims)))
            for _ in range(n_agents)
        ]
        
        self.alpha = alpha
        self.gamma = gamma
    
    def compute_nash_equilibrium(self, state):
        """计算纳什均衡（简化：双智能体零和博弈）"""
        if self.n_agents != 2:
            raise NotImplementedError("只支持双智能体")
        
        # 获取 Q 值矩阵
        Q1 = self.Q_tables[0][state].reshape(self.action_dims)
        Q2 = self.Q_tables[1][state].reshape(self.action_dims)
        
        # 零和博弈：Q2 = -Q1
        # 简化：使用线性规划求解
        # 这里用随机策略近似
        nash_policy_1 = np.ones(self.action_dims[0]) / self.action_dims[0]
        nash_policy_2 = np.ones(self.action_dims[1]) / self.action_dims[1]
        
        return nash_policy_1, nash_policy_2
    
    def update(self, state, actions, rewards, next_state):
        """更新 Q 值"""
        action_index = self.action_to_index(actions)
        
        # 计算纳什均衡值
        nash_policies = self.compute_nash_equilibrium(next_state)
        
        for i in range(self.n_agents):
            # 纳什均衡价值
            nash_value = self.compute_nash_value(i, next_state, nash_policies)
            
            # 更新 Q 值
            td_target = rewards[i] + self.gamma * nash_value
            self.Q_tables[i][state, action_index] += self.alpha * (
                td_target - self.Q_tables[i][state, action_index]
            )
    
    def compute_nash_value(self, agent_id, state, nash_policies):
        """计算纳什均衡价值"""
        Q = self.Q_tables[agent_id][state].reshape(self.action_dims)
        
        if agent_id == 0:
            return np.sum(nash_policies[0][:, None] * nash_policies[1][None, :] * Q)
        else:
            return np.sum(nash_policies[0][:, None] * nash_policies[1][None, :] * Q)
```

### 3.2 自博弈（Self-Play）

**自博弈**：智能体与自己或过去的版本对战。

**应用**：
- AlphaGo Zero：自我对弈
- AlphaStar：人口训练
- OpenAI Five：团队自博弈

```python
class SelfPlay:
    """自博弈训练"""
    
    def __init__(self, agent_class, env, buffer_size=100000):
        self.env = env
        self.agent = agent_class()
        self.buffer = ReplayBuffer(buffer_size)
        
        # 历史策略池
        self.policy_pool = []
        self.pool_size = 10
    
    def self_play_episode(self):
        """自博弈一局"""
        state = self.env.reset()
        trajectory = []
        
        while not self.env.is_terminal():
            # 双方使用相同策略
            action = self.agent.select_action(state)
            next_state, reward, done = self.env.step(action)
            
            trajectory.append((state, action, reward, next_state, done))
            state = next_state
        
        return trajectory
    
    def train(self, n_episodes=10000, update_freq=100):
        """训练循环"""
        for episode in range(n_episodes):
            # 自博弈
            trajectory = self.self_play_episode()
            self.buffer.extend(trajectory)
            
            # 更新策略
            if len(self.buffer) > 1000:
                batch = self.buffer.sample(32)
                self.agent.update(batch)
            
            # 更新策略池
            if episode % update_freq == 0:
                self.policy_pool.append(self.agent.copy())
                if len(self.policy_pool) > self.pool_size:
                    self.policy_pool.pop(0)
                
                print(f"Episode {episode}, Pool size: {len(self.policy_pool)}")
        
        return self.agent
```

## 4. 前沿方向

### 4.1 离线强化学习（Offline RL）

**问题**：从固定数据集学习策略，不能与环境交互。

**挑战**：
- 分布偏移：策略产生的数据分布与数据集分布不同
- 外推误差：Q 值在数据集外的区域被高估

**方法**：

**保守 Q 学习（CQL）**：

$$\mathcal{L}_{CQL} = \mathcal{L}_{TD} + \alpha \left( \mathbb{E}_{s \sim D, a \sim \pi}[Q(s,a)] - \mathbb{E}_{s,a \sim D}[Q(s,a)] \right)$$

惩罚策略选择的动作 Q 值，鼓励保守估计。

```python
class ConservativeQLearning:
    """保守 Q 学习"""
    
    def __init__(self, state_dim, action_dim, hidden_dim=256, alpha=1.0):
        self.q_network = QNetwork(state_dim, action_dim, hidden_dim)
        self.target_q = QNetwork(state_dim, action_dim, hidden_dim)
        self.target_q.load_state_dict(self.q_network.state_dict())
        
        self.optimizer = optim.Adam(self.q_network.parameters(), lr=1e-3)
        self.alpha = alpha
    
    def update(self, batch, policy):
        """CQL 更新"""
        states, actions, rewards, next_states, dones = batch
        
        # 标准 TD 损失
        current_q = self.q_network(states).gather(1, actions.unsqueeze(1))
        with torch.no_grad():
            next_q = self.target_q(next_states).max(dim=1)[0]
            target_q = rewards + 0.99 * next_q * (1 - dones)
        td_loss = nn.MSELoss()(current_q.squeeze(), target_q)
        
        # 保守正则化
        # 从数据集采样的动作
        dataset_q = self.q_network(states).gather(1, actions.unsqueeze(1)).mean()
        
        # 从策略采样的动作
        policy_actions = policy.sample(states)
        policy_q = self.q_network(states).gather(1, policy_actions.unsqueeze(1)).mean()
        
        cql_loss = self.alpha * (policy_q - dataset_q)
        
        # 总损失
        loss = td_loss + cql_loss
        
        self.optimizer.zero_grad()
        loss.backward()
        self.optimizer.step()
        
        return loss.item()
```

### 4.2 元强化学习（Meta-RL）

**目标**：学习如何快速学习新任务。

**MAML（Model-Agnostic Meta-Learning）**：

$$\theta^* = \arg\min_\theta \sum_{T_i} \mathcal{L}_{T_i}(f_{\theta_i'})$$

其中 $\theta_i' = \theta - \alpha \nabla_\theta \mathcal{L}_{T_i}(f_\theta)$ 是任务 $T_i$ 上的一步梯度更新。

```python
class MAML:
    """Model-Agnostic Meta-Learning for RL"""
    
    def __init__(self, policy, inner_lr=0.1, outer_lr=0.001):
        self.policy = policy
        self.inner_lr = inner_lr
        self.outer_lr = outer_lr
        self.optimizer = optim.Adam(policy.parameters(), lr=outer_lr)
    
    def inner_update(self, task, n_steps=1):
        """内循环：在单个任务上更新"""
        # 复制参数
        theta_i = {k: v.clone() for k, v in self.policy.state_dict().items()}
        
        for _ in range(n_steps):
            # 采样任务数据
            trajectories = task.sample_trajectories(self.policy, theta_i)
            
            # 计算梯度
            loss = self.policy.compute_loss(trajectories, theta_i)
            grads = torch.autograd.grad(loss, self.policy.parameters())
            
            # 更新参数
            for (k, v), g in zip(theta_i.items(), grads):
                theta_i[k] = v - self.inner_lr * g
        
        return theta_i
    
    def meta_update(self, tasks):
        """外循环：元更新"""
        meta_loss = 0
        
        for task in tasks:
            # 内循环更新
            theta_i = self.inner_update(task)
            
            # 在新数据上评估
            test_trajectories = task.sample_trajectories(self.policy, theta_i)
            task_loss = self.policy.compute_loss(test_trajectories, theta_i)
            
            meta_loss += task_loss
        
        meta_loss /= len(tasks)
        
        # 更新元参数
        self.optimizer.zero_grad()
        meta_loss.backward()
        self.optimizer.step()
        
        return meta_loss.item()
```

### 4.3 层次化强化学习（HRL）

**思想**：将复杂任务分解为层次化的子任务。

**选项框架（Options Framework）**：

一个选项 $o = (\mathcal{I}, \pi, \beta)$ 包含：
- $\mathcal{I}$：初始条件
- $\pi$：内部策略
- $\beta$：终止条件

```python
class Option:
    """选项"""
    
    def __init__(self, initiation, policy, termination, name="option"):
        self.initiation = initiation  # 初始条件函数
        self.policy = policy          # 内部策略
        self.termination = termination  # 终止条件函数
        self.name = name
    
    def can_initiate(self, state):
        """检查选项是否可以开始"""
        return self.initiation(state)
    
    def select_action(self, state):
        """选择动作"""
        return self.policy(state)
    
    def should_terminate(self, state):
        """检查是否应该终止"""
        return self.termination(state)


class HierarchicalAgent:
    """层次化智能体"""
    
    def __init__(self, options, meta_policy):
        self.options = options          # 选项集
        self.meta_policy = meta_policy  # 元策略（选择选项）
        self.current_option = None
    
    def select_option(self, state):
        """选择选项"""
        # 找出可以开始的选项
        available_options = [
            o for o in self.options 
            if o.can_initiate(state)
        ]
        
        if not available_options:
            return None
        
        # 元策略选择
        option_idx = self.meta_policy(state, available_options)
        return available_options[option_idx]
    
    def act(self, state):
        """选择动作"""
        # 如果没有当前选项或选项终止
        if self.current_option is None or self.current_option.should_terminate(state):
            self.current_option = self.select_option(state)
        
        if self.current_option is None:
            return None  # 无可用动作
        
        return self.current_option.select_action(state)
```

### 4.4 探索与好奇心

**内在动机**：在缺乏外部奖励时，智能体如何探索？

**好奇心驱动探索**：

$$r_{intrinsic} = \| \phi(s_{t+1}) - \hat{\phi}(s_{t+1}) \|^2$$

其中 $\hat{\phi}$ 是预测下一状态的模型，预测误差作为内在奖励。

```python
class CuriosityDrivenAgent:
    """好奇心驱动的智能体"""
    
    def __init__(self, state_dim, action_dim, hidden_dim=256):
        # 策略网络
        self.policy = PolicyNetwork(state_dim, action_dim, hidden_dim)
        
        # 前向模型：预测下一状态
        self.forward_model = nn.Sequential(
            nn.Linear(state_dim + action_dim, hidden_dim),
            nn.ReLU(),
            nn.Linear(hidden_dim, state_dim)
        )
        
        # 逆向模型：预测动作
        self.inverse_model = nn.Sequential(
            nn.Linear(state_dim * 2, hidden_dim),
            nn.ReLU(),
            nn.Linear(hidden_dim, action_dim)
        )
        
        self.optimizer = optim.Adam(
            list(self.policy.parameters()) + 
            list(self.forward_model.parameters()) +
            list(self.inverse_model.parameters()),
            lr=1e-3
        )
    
    def compute_intrinsic_reward(self, state, action, next_state):
        """计算内在奖励（预测误差）"""
        # 前向模型预测
        input_tensor = torch.cat([state, action], dim=-1)
        predicted_next_state = self.forward_model(input_tensor)
        
        # 预测误差
        error = nn.MSELoss()(predicted_next_state, next_state)
        
        return error.item()
    
    def update(self, batch):
        """更新"""
        states, actions, rewards, next_states, dones = batch
        
        # 内在奖励
        intrinsic_rewards = []
        for s, a, ns in zip(states, actions, next_states):
            r_int = self.compute_intrinsic_reward(s, a, ns)
            intrinsic_rewards.append(r_int)
        intrinsic_rewards = torch.tensor(intrinsic_rewards)
        
        # 总奖励
        total_rewards = rewards + 0.01 * intrinsic_rewards
        
        # 策略更新（使用 PPO 等）
        policy_loss = self.policy.update(states, actions, total_rewards)
        
        # 前向模型更新
        input_tensor = torch.cat([states, actions], dim=-1)
        pred_next = self.forward_model(input_tensor)
        forward_loss = nn.MSELoss()(pred_next, next_states)
        
        # 逆向模型更新
        state_action_pair = torch.cat([states, next_states], dim=-1)
        pred_actions = self.inverse_model(state_action_pair)
        inverse_loss = nn.CrossEntropyLoss()(pred_actions, actions)
        
        # 总损失
        loss = policy_loss + forward_loss + inverse_loss
        
        self.optimizer.zero_grad()
        loss.backward()
        self.optimizer.step()
```

class LLMAgent:

"""
        return prompt

```python
class CooperativeNavigation:
    """合作导航环境"""
    
    def __init__(self, n_agents=3, n_landmarks=3, world_size=1.0):
        self.n_agents = n_agents
        self.n_landmarks = n_landmarks
        self.world_size = world_size
        
        # 动作：[no_action, move_left, move_right, move_up, move_down]
        self.action_dim = 5
        
        # 状态：每个智能体的位置 + 所有地标位置
        self.obs_dim = 2 + 2 * n_landmarks
        
        self.reset()
    
    def reset(self):
        """重置环境"""
        # 随机初始化智能体位置
        self.agent_positions = np.random.rand(self.n_agents, 2) * self.world_size
        
        # 随机初始化地标位置
        self.landmark_positions = np.random.rand(self.n_landmarks, 2) * self.world_size
        
        return self._get_observations()
    
    def _get_observations(self):
        """获取观测"""
        observations = []
        for i in range(self.n_agents):
            # 自己的位置
            obs = list(self.agent_positions[i])
            
            # 地标的相对位置
            for lm_pos in self.landmark_positions:
                relative_pos = lm_pos - self.agent_positions[i]
                obs.extend(relative_pos)
            
            observations.append(obs)
        
        return observations
    
    def step(self, actions):
        """执行动作"""
        # 移动智能体
        for i, action in enumerate(actions):
            if action == 1:  # left
                self.agent_positions[i][0] -= 0.1
            elif action == 2:  # right
                self.agent_positions[i][0] += 0.1
            elif action == 3:  # up
                self.agent_positions[i][1] += 0.1
            elif action == 4:  # down
                self.agent_positions[i][1] -= 0.1
            
            # 边界约束
            self.agent_positions[i] = np.clip(
                self.agent_positions[i], 0, self.world_size
            )
        
        # 计算奖励（所有智能体共享）
        reward = self._compute_reward()
        
        # 检查是否结束
        done = self._check_done()
        
        return self._get_observations(), reward, done
    
    def _compute_reward(self):
        """计算奖励"""
        # 每个地标需要被至少一个智能体覆盖
        total_reward = 0
        agent_radius = 0.1
        
        for lm_pos in self.landmark_positions:
            # 检查是否有智能体覆盖这个地标
            covered = False
            for agent_pos in self.agent_positions:
                distance = np.linalg.norm(agent_pos - lm_pos)
                if distance < agent_radius:
                    covered = True
                    break
            
            if covered:
                total_reward += 1.0
            else:
                # 距离惩罚
                min_dist = min(
                    np.linalg.norm(agent_pos - lm_pos)
                    for agent_pos in self.agent_positions
                )
                total_reward -= min_dist
        
        # 碰撞惩罚
        for i in range(self.n_agents):
            for j in range(i + 1, self.n_agents):
                dist = np.linalg.norm(
                    self.agent_positions[i] - self.agent_positions[j]
                )
                if dist < 0.1:
                    total_reward -= 1.0
        
        return total_reward
    
    def _check_done(self):
        """检查是否结束"""
        # 所有地标被覆盖
        agent_radius = 0.1
        all_covered = True
        
        for lm_pos in self.landmark_positions:
            covered = False
            for agent_pos in self.agent_positions:
                if np.linalg.norm(agent_pos - lm_pos) < agent_radius:
                    covered = True
                    break
            if not covered:
                all_covered = False
                break
        
        return all_covered


def train_cooperative_navigation():
    """训练合作导航"""
    env = CooperativeNavigation(n_agents=3, n_landmarks=3)
    agent = QMIX(
        n_agents=3,
        obs_dim=8,
        state_dim=12,
        action_dim=5,
        hidden_dim=64
    )
    
    n_episodes = 10000
    epsilon_start = 1.0
    epsilon_end = 0.05
    epsilon_decay = 0.995
    
    epsilon = epsilon_start
    rewards_history = []
    
    for episode in range(n_episodes):
        observations = env.reset()
        episode_reward = 0
        done = False
        
        while not done:
            # 选择动作
            actions = agent.select_actions(observations, epsilon)
            
            # 执行动作
            next_observations, reward, done = env.step(actions)
            episode_reward += reward
            
            # 存储经验
            # ...（存储到经验回放）
            
            observations = next_observations
        
        rewards_history.append(episode_reward)
        epsilon = max(epsilon_end, epsilon * epsilon_decay)
        
        if (episode + 1) % 100 == 0:
            avg_reward = np.mean(rewards_history[-100:])
            print(f"Episode {episode + 1}, Avg Reward: {avg_reward:.2f}, Epsilon: {epsilon:.3f}")
    
    return agent, rewards_history
```

| 主题 | 核心公式 |
|------|----------|
| 贝尔曼方程 | $V^\pi(s) = \sum_a \pi(a\|s)\sum_{s'}P(s'\|s,a)[R + \gamma V^\pi(s')]$ |
| Q-Learning | $Q(s,a) \leftarrow Q(s,a) + \alpha[r + \gamma\max_{a'}Q(s',a') - Q(s,a)]$ |
| 策略梯度 | $\nabla_\theta J = \mathbb{E}[\nabla_\theta\log\pi_\theta(a\|s) \cdot A]$ |

| DPO | $\mathcal{L} = -\mathbb{E}[\log\sigma(\beta(\log\frac{\pi(y_w\|x)}{\pi_{ref}(y_w\|x)} - \log\frac{\pi(y_l\|x)}{\pi_{ref}(y_l\|x)}))]$ |
