# Day 5：无模型学习——Q-Learning与SARSA

> **核心问题**：当我们不知道环境模型时，如何从经验中学习最优策略？

## 引言：黑盒世界的挑战

想象你被困在一个陌生的迷宫中：

**动态规划的方式**：你有一张详细的地图，标注了每个位置、每条通道和奖励位置。你可以坐在家里，仔细规划最优路径。

**现实情况**：你没有地图。你只能走进迷宫，尝试不同的路，记住哪些路通向奖励，哪些路通向死胡同。

这就是**无模型学习（Model-Free Learning）**面临的问题：**不知道状态转移概率 P(s'|s,a)，不知道奖励函数 R(s,a,s')，只能通过与环境交互获取样本。**

Day 4 我们学习了动态规划——假设完全知道环境模型。今天，我们抛开这个假设，进入更接近现实的场景：**从经验中学习**。

---

## 1. 从有模型到无模型

### 1.1 为什么需要无模型方法？

**现实世界的特点**：

| 场景 | 已知模型？ | 原因 |
|------|-----------|------|
| 棋类游戏 | ✓ 已知 | 规则完全确定 |
| 机器人控制 | ✗ 未知 | 物理模型复杂，难以精确建模 |
| 推荐系统 | ✗ 未知 | 用户偏好未知，需要探索 |
| 自动驾驶 | ✗ 未知 | 环境高度不确定 |
| 游戏AI | 部分已知 | 规则已知，但对手策略未知 |

**核心差异**：

| 特性 | 有模型（DP） | 无模型（TD/MC） |
|------|-------------|-----------------|
| 需要 P(s'\|s,a) | ✓ | ✗ |
| 需要 R(s,a,s') | ✓ | ✗ |
| 数据来源 | 计算所有转移 | 从交互中采样 |
| 样本效率 | 高（充分利用模型） | 低（需要大量样本） |
| 适用范围 | 模型已知的场景 | 通用场景 |

### 1.2 无模型学习的两条路径

**路径一：蒙特卡洛方法（Monte Carlo, MC）**
- 从完整轨迹学习
- 需要等一个 episode 结束才能更新
- 无偏估计，但方差大

**路径二：时序差分学习（Temporal Difference, TD）**
- 从单步转移学习
- 每一步都能更新
- 有偏估计，但方差小

两条路径都能达到目标，各有优劣。让我们逐一深入。

## 2. 蒙特卡洛方法

### 2.1 核心思想

**蒙特卡洛方法**：用样本平均估计期望值。

**回顾价值函数定义**：

$$V^\pi(s) = \mathbb{E}_\pi[G_t | S_t = s] = \mathbb{E}_\pi\left[\sum_{k=0}^{\infty} \gamma^k R_{t+k+1} \bigg| S_t = s\right]$$

**蒙特卡洛估计**：收集从状态 s 出发的多条轨迹，计算回报的平均值。

$$V(s) \approx \frac{1}{N} \sum_{i=1}^{N} G_i^{(s)}$$

其中 $G_i^{(s)}$ 是第 i 条从 s 出发的轨迹的回报。

### 2.2 首次访问 vs 每次访问

**首次访问蒙特卡洛（First-Visit MC）**：
- 只统计每个 episode 中第一次访问某状态的回报
- 无偏估计

**每次访问蒙特卡洛（Every-Visit MC）**：
- 统计所有访问的回报
- 有偏，但方差更小

```python
def first_visit_mc_prediction(env, policy, episodes=1000, gamma=0.9):
    """
    首次访问蒙特卡洛策略评估
    
    参数：
        env: 环境，需要 step(), reset(), is_terminal() 方法
        policy: 策略，policy(state) 返回动作
        episodes: episode 数量
        gamma: 折扣因子
    
    返回：
        V: 状态价值函数
    """
    # 初始化
    returns = {s: [] for s in env.states}  # 每个状态的回报列表
    V = {s: 0.0 for s in env.states}       # 价值函数
    
    for episode in range(episodes):
        # 生成一条轨迹
        trajectory = []
        state = env.reset()
        
        while not env.is_terminal(state):
            action = policy(state)
            next_state, reward, done = env.step(action)
            trajectory.append((state, action, reward))
            state = next_state
            if done:
                break
        
        # 计算回报并更新
        G = 0
        visited = set()
        
        # 从后往前计算回报
        for t in reversed(range(len(trajectory))):
            state, action, reward = trajectory[t]
            G = gamma * G + reward
            
            # 首次访问：只在第一次遇到时更新
            if state not in visited:
                returns[state].append(G)
                V[state] = sum(returns[state]) / len(returns[state])
                visited.add(state)
    
    return V
```

### 2.3 蒙特卡洛控制

**目标**：在不知道模型的情况下找到最优策略。

**思路**：估计 Q(s, a) 而非 V(s)，然后贪婪改进。

**算法**：

```
初始化 Q(s, a) = 0，π(s) = 随机动作

循环：
    1. 使用 π 生成一条轨迹
    2. 计算每个 (s, a) 的回报 G
    3. 更新 Q(s, a) = average(G)
    4. π(s) = argmax_a Q(s, a)（贪婪改进）
```

**问题**：纯贪婪策略会陷入局部最优，需要探索！

**解决方案**：ε-贪心探索

$$\pi(a|s) = \begin{cases} 
1 - \epsilon + \frac{\epsilon}{|\mathcal{A}|} & \text{if } a = \arg\max_a Q(s,a) \\
\frac{\epsilon}{|\mathcal{A}|} & \text{otherwise}
\end{cases}$$

### 2.4 完整实现

```python
import numpy as np
from collections import defaultdict

class MonteCarloControl:
    """蒙特卡洛控制（ε-贪心探索）"""
    
    def __init__(self, n_states, n_actions, epsilon=0.1, gamma=0.9):
        self.n_states = n_states
        self.n_actions = n_actions
        self.epsilon = epsilon
        self.gamma = gamma
        
        # Q 表和访问次数
        self.Q = defaultdict(lambda: np.zeros(n_actions))
        self.N = defaultdict(lambda: np.zeros(n_actions))
    
    def select_action(self, state):
        """ε-贪心动作选择"""
        if np.random.random() < self.epsilon:
            return np.random.randint(self.n_actions)
        else:
            return np.argmax(self.Q[state])
    
    def update(self, trajectory):
        """
        从轨迹更新 Q 值
        
        参数：
            trajectory: [(state, action, reward), ...] 列表
        """
        G = 0
        visited = set()
        
        for t in reversed(range(len(trajectory))):
            state, action, reward = trajectory[t]
            G = self.gamma * G + reward
            
            # 首次访问更新
            if (state, action) not in visited:
                self.N[state][action] += 1
                # 增量式均值更新
                self.Q[state][action] += (G - self.Q[state][action]) / self.N[state][action]
                visited.add((state, action))
    
    def get_policy(self):
        """获取当前策略"""
        policy = {}
        for state in self.Q:
            policy[state] = np.argmax(self.Q[state])
        return policy
```

### 2.5 蒙特卡洛的优缺点

**优点**：
- 无偏估计
- 不需要环境模型
- 适用于 episode 任务

**缺点**：
- 必须等 episode 结束才能更新
- 方差大（单条轨迹的回报波动大）
- 不适用于连续任务

## 3. 时序差分学习

### 3.1 核心思想

**时序差分（TD）**：结合了蒙特卡洛和动态规划的优点。

**回顾两种方法的更新公式**：

| 方法 | 更新公式 |
|------|----------|
| **蒙特卡洛** | $V(S_t) \leftarrow V(S_t) + \alpha [G_t - V(S_t)]$ |
| **动态规划** | $V(S_t) \leftarrow \mathbb{E}[R_{t+1} + \gamma V(S_{t+1})]$ |

**TD 的创新**：用估计值代替真实回报

$$V(S_t) \leftarrow V(S_t) + \alpha [R_{t+1} + \gamma V(S_{t+1}) - V(S_t)]$$

**关键概念**：

**TD 误差（TD Error）**：

$$\delta_t = R_{t+1} + \gamma V(S_{t+1}) - V(S_t)$$

**直觉理解**：
- $R_{t+1} + \gamma V(S_{t+1})$ 是"新的估计"（基于当前转移）
- $V(S_t)$ 是"旧的估计"
- TD 误差反映了估计的不一致性

### 3.2 TD(0) 策略评估

**TD(0)**：最简单的时序差分方法。

```python
def td0_prediction(env, policy, episodes=1000, alpha=0.1, gamma=0.9):
    """
    TD(0) 策略评估
    
    参数：
        env: 环境
        policy: 策略
        episodes: episode 数量
        alpha: 学习率
        gamma: 折扣因子
    
    返回：
        V: 状态价值函数
    """
    V = {s: 0.0 for s in env.states}
    
    for episode in range(episodes):
        state = env.reset()
        
        while not env.is_terminal(state):
            action = policy(state)
            next_state, reward, done = env.step(action)
            
            # TD 更新
            td_target = reward + gamma * V[next_state]
            td_error = td_target - V[state]
            V[state] += alpha * td_error
            
            state = next_state
            if done:
                break
    
    return V
```

### 3.3 TD vs 蒙特卡洛

| 特性 | 蒙特卡洛 | TD(0) |
|------|----------|-------|
| **更新时机** | episode 结束后 | 每一步 |
| **目标** | $G_t$（真实回报） | $R_{t+1} + \gamma V(S_{t+1})$（估计） |
| **偏差** | 无偏 | 有偏 |
| **方差** | 高 | 低 |
| **适用场景** | episode 任务 | episode 和连续任务 |
| **样本效率** | 低 | 高 |

**例子**开车回家**：

- **MC**：每次开车回家，记录总时间。100 次后，对"从公司到家"的时间有准确估计。
- **TD**：走到一半发现比平时慢，立即更新"从当前位置到家"的时间估计。

TD 更快适应变化！

### 3.4 TD(λ)：融合 MC 和 TD

**核心思想**：结合不同时间尺度的回报。

**n 步回报**：

$$G_t^{(n)} = R_{t+1} + \gamma R_{t+2} + \cdots + \gamma^{n-1} R_{t+n} + \gamma^n V(S_{t+n})$$

- n=1：TD(0)，单步
- n=∞：蒙特卡洛，完整轨迹

**TD(λ) 前向视角**：

$$G_t^\lambda = (1-\lambda) \sum_{n=1}^{\infty} \lambda^{n-1} G_t^{(n)}$$

**TD(λ) 反向视角（资格迹）**：

$$E_t(s) = \begin{cases} 
\gamma \lambda E_{t-1}(s) + 1 & \text{if } s = S_t \\
\gamma \lambda E_{t-1}(s) & \text{otherwise}
\end{cases}$$

$$V(s) \leftarrow V(s) + \alpha \delta_t E_t(s)$$

## 4. Q-Learning：离策略时序差分控制

### 4.1 核心算法

**Q-Learning** 是最著名的无模型控制算法之一。

**更新公式**：

$$Q(S_t, A_t) \leftarrow Q(S_t, A_t) + \alpha \left[ R_{t+1} + \gamma \max_a Q(S_{t+1}, a) - Q(S_t, A_t) \right]$$

**关键特点**：
- 直接学习最优 Q 值
- **离策略（Off-Policy）**：学习策略和行为策略可以不同
- 使用 max，因此是贪婪的（相对于 Q 值）

### 4.2 算法流程

```
初始化 Q(s, a) = 0 对所有 s ∈ S, a ∈ A

对于每个 episode：
    初始化状态 s
    
    对于每一步：
        使用行为策略（如 ε-贪心）选择动作 a
        执行 a，观察到 r, s'
        
        # Q-Learning 更新
        Q(s, a) ← Q(s, a) + α[r + γ max_a' Q(s', a') - Q(s, a)]
        
        s ← s'
    
    直到 s 是终止状态
```

### 4.3 完整实现

```python
import numpy as np
from collections import defaultdict

class QLearning:
    """Q-Learning 算法"""
    
    def __init__(
        self, 
        n_states, 
        n_actions, 
        learning_rate=0.1,
        discount_factor=0.9,
        epsilon=0.1
    ):
        self.n_states = n_states
        self.n_actions = n_actions
        self.alpha = learning_rate
        self.gamma = discount_factor
        self.epsilon = epsilon
        
        # Q 表
        self.Q = defaultdict(lambda: np.zeros(n_actions))
    
    def select_action(self, state, training=True):
        """ε-贪心动作选择"""
        if training and np.random.random() < self.epsilon:
            return np.random.randint(self.n_actions)
        else:
            return np.argmax(self.Q[state])
    
    def update(self, state, action, reward, next_state, done):
        """
        Q-Learning 更新
        
        参数：
            state: 当前状态
            action: 执行的动作
            reward: 获得的奖励
            next_state: 下一状态
            done: 是否终止
        """
        # 计算目标
        if done:
            target = reward
        else:
            target = reward + self.gamma * np.max(self.Q[next_state])
        
        # TD 更新
        td_error = target - self.Q[state][action]
        self.Q[state][action] += self.alpha * td_error
    
    def train(self, env, episodes=1000, max_steps=1000):
        """训练"""
        rewards_history = []
        
        for episode in range(episodes):
            state = env.reset()
            total_reward = 0
            
            for step in range(max_steps):
                # 选择动作
                action = self.select_action(state, training=True)
                
                # 执行动作
                next_state, reward, done = env.step(action)
                total_reward += reward
                
                # 更新 Q 值
                self.update(state, action, reward, next_state, done)
                
                state = next_state
                
                if done:
                    break
            
            rewards_history.append(total_reward)
            
            # 衰减探索率
            self.epsilon = max(0.01, self.epsilon * 0.995)
        
        return rewards_history
    
    def get_policy(self):
        """获取贪婪策略"""
        return {s: np.argmax(self.Q[s]) for s in self.Q}
    
    def get_value_function(self):
        """获取价值函数"""
        return {s: np.max(self.Q[s]) for s in self.Q}
```

### 4.4 Q-Learning 的收敛性

**定理**：在满足以下条件时，Q-Learning 收敛到最优 Q*：

1. **所有状态-动作对被无限次访问**
2. **学习率 α_t 满足**：
   - $\sum_{t=0}^{\infty} \alpha_t = \infty$（足够大）
   - $\sum_{t=0}^{\infty} \alpha_t^2 < \infty$（足够小）

**实践中的设置**：
- $\alpha_t = \frac{1}{t}$ 或 $\alpha_t = \frac{1}{1 + t \cdot \text{decay}}$
- 常用常数学习率 α = 0.1 也往往有效

## 5. SARSA：在策略时序差分控制

### 5.1 核心算法

**SARSA**（State-Action-Reward-State-Action）是另一种 TD 控制算法。

**更新公式**：

$$Q(S_t, A_t) \leftarrow Q(S_t, A_t) + \alpha \left[ R_{t+1} + \gamma Q(S_{t+1}, A_{t+1}) - Q(S_t, A_t) \right]$$

**关键特点**：
- **在策略（On-Policy）**：学习策略和行为策略相同
- 使用实际选择的动作 $A_{t+1}$，而非 max

### 5.2 Q-Learning vs SARSA

| 特性 | Q-Learning | SARSA |
|------|------------|-------|
| **策略类型** | 离策略（Off-Policy） | 在策略（On-Policy） |
| **更新目标** | $\max_a Q(s', a)$ | $Q(s', a')$ |
| **学习内容** | 最优策略的价值 | 当前策略的价值 |
| **探索策略** | 影响学习过程 | 参与学习过程 |
| **收敛性** | 收敛到最优 Q* | 收敛到当前策略的 Q^π |

### 5.3 完整实现

```python
class SARSA:
    """SARSA 算法"""
    
    def __init__(
        self, 
        n_states, 
        n_actions, 
        learning_rate=0.1,
        discount_factor=0.9,
        epsilon=0.1
    ):
        self.n_states = n_states
        self.n_actions = n_actions
        self.alpha = learning_rate
        self.gamma = discount_factor
        self.epsilon = epsilon
        
        self.Q = defaultdict(lambda: np.zeros(n_actions))
    
    def select_action(self, state, training=True):
        """ε-贪心动作选择"""
        if training and np.random.random() < self.epsilon:
            return np.random.randint(self.n_actions)
        else:
            return np.argmax(self.Q[state])
    
    def update(self, state, action, reward, next_state, next_action, done):
        """
        SARSA 更新
        
        参数：
            state: 当前状态
            action: 执行的动作
            reward: 获得的奖励
            next_state: 下一状态
            next_action: 下一动作
            done: 是否终止
        """
        if done:
            target = reward
        else:
            target = reward + self.gamma * self.Q[next_state][next_action]
        
        td_error = target - self.Q[state][action]
        self.Q[state][action] += self.alpha * td_error
    
    def train(self, env, episodes=1000, max_steps=1000):
        """训练"""
        rewards_history = []
        
        for episode in range(episodes):
            state = env.reset()
            action = self.select_action(state, training=True)
            total_reward = 0
            
            for step in range(max_steps):
                # 执行动作
                next_state, reward, done = env.step(action)
                total_reward += reward
                
                # 选择下一动作
                next_action = self.select_action(next_state, training=True)
                
                # 更新 Q 值
                self.update(state, action, reward, next_state, next_action, done)
                
                state = next_state
                action = next_action
                
                if done:
                    break
            
            rewards_history.append(total_reward)
            self.epsilon = max(0.01, self.epsilon * 0.995)
        
        return rewards_history
    
    def get_policy(self):
        return {s: np.argmax(self.Q[s]) for s in self.Q}
```

### 5.4 一个经典例子：悬崖行走

**环境设置**：

```
┌───┬───┬───┬───┬───┬───┬───┬───┬───┬───┬───┬───┐
│ S │   │   │   │   │   │   │   │   │   │   │   │  起点 S
├───┼───┼───┼───┼───┼───┼───┼───┼───┼───┼───┼───┤  目标 G
│   │   │   │   │   │   │   │   │   │   │   │   │  悬崖 C
├───┼───┼───┼───┼───┼───┼───┼───┼───┼───┼───┼───┤  
│   │   │   │   │   │   │   │   │   │   │   │   │  奖励：
├───┼───┼───┼───┼───┼───┼───┼───┼───┼───┼───┼───┤  -1 每步
│   │ C │ C │ C │ C │ C │ C │ C │ C │ C │ C │ G │  -100 掉入悬崖
└───┴───┴───┴───┴───┴───┴───┴───┴───┴───┴───┴───┘
```

**Q-Learning vs SARSA 的行为差异**：

- **Q-Learning**：学习最优路径（紧贴悬崖），但因为探索，实际执行时可能掉入悬崖
- **SARSA**：学习安全路径（远离悬崖），考虑到探索的代价

```python
class CliffWalking:
    """悬崖行走环境"""
    
    def __init__(self, width=12, height=4):
        self.width = width
        self.height = height
        self.start = (height - 1, 0)
        self.goal = (height - 1, width - 1)
        self.cliff = [(height - 1, j) for j in range(1, width - 1)]
        self.actions = ['up', 'down', 'left', 'right']
        self.action_effects = {
            'up': (-1, 0),
            'down': (1, 0),
            'left': (0, -1),
            'right': (0, 1)
        }
        self.state = None
        self.reset()
    
    @property
    def states(self):
        return [(i, j) for i in range(self.height) for j in range(self.width)]
    
    @property
    def n_actions(self):
        return len(self.actions)
    
    def reset(self):
        self.state = self.start
        return self.state
    
    def is_terminal(self, state=None):
        state = state or self.state
        return state == self.goal
    
    def step(self, action):
        if isinstance(action, int):
            action = self.actions[action]
        
        # 计算新位置
        di, dj = self.action_effects[action]
        new_i = max(0, min(self.height - 1, self.state[0] + di))
        new_j = max(0, min(self.width - 1, self.state[1] + dj))
        new_state = (new_i, new_j)
        
        # 检查悬崖
        if new_state in self.cliff:
            self.state = self.start
            return self.start, -100, False  # 回到起点
        
        # 检查目标
        if new_state == self.goal:
            self.state = new_state
            return new_state, 0, True  # 到达目标
        
        self.state = new_state
        return new_state, -1, False  # 普通移动
```

## 6. 期望SARSA与双Q学习

### 6.1 期望SARSA

**动机**：减少 SARSA 的方差。

**更新公式**：

$$Q(S_t, A_t) \leftarrow Q(S_t, A_t) + \alpha \left[ R_{t+1} + \gamma \sum_a \pi(a|S_{t+1}) Q(S_{t+1}, a) - Q(S_t, A_t) \right]$$

**区别**：
- SARSA 使用单个样本 $Q(S_{t+1}, A_{t+1})$
- 期望 SARSA 使用期望 $\sum_a \pi(a|S_{t+1}) Q(S_{t+1}, a)$

```python
class ExpectedSARSA:
    """期望 SARSA 算法"""
    
    def __init__(
        self, 
        n_states, 
        n_actions, 
        learning_rate=0.1,
        discount_factor=0.9,
        epsilon=0.1
    ):
        self.n_states = n_states
        self.n_actions = n_actions
        self.alpha = learning_rate
        self.gamma = discount_factor
        self.epsilon = epsilon
        
        self.Q = defaultdict(lambda: np.zeros(n_actions))
    
    def get_action_probs(self, state):
        """获取动作概率（ε-贪心）"""
        probs = np.ones(self.n_actions) * self.epsilon / self.n_actions
        best_action = np.argmax(self.Q[state])
        probs[best_action] += 1 - self.epsilon
        return probs
    
    def select_action(self, state, training=True):
        if training and np.random.random() < self.epsilon:
            return np.random.randint(self.n_actions)
        else:
            return np.argmax(self.Q[state])
    
    def update(self, state, action, reward, next_state, done):
        if done:
            target = reward
        else:
            # 计算期望 Q 值
            probs = self.get_action_probs(next_state)
            expected_q = np.sum(probs * self.Q[next_state])
            target = reward + self.gamma * expected_q
        
        td_error = target - self.Q[state][action]
        self.Q[state][action] += self.alpha * td_error
```

### 6.2 双Q学习（Double Q-Learning）

**动机**：解决 Q-Learning 的**过高估计偏差**。

**问题**：

Q-Learning 使用 $\max_a Q(s', a)$，导致**正向偏差**：

$$\mathbb{E}[\max_a Q(s', a)] \geq \max_a \mathbb{E}[Q(s', a)]$$

**解决方案**：使用两个 Q 表，一个用于选择动作，另一个用于评估。

**更新公式**：

$$Q_1(S_t, A_t) \leftarrow Q_1(S_t, A_t) + \alpha \left[ R_{t+1} + \gamma Q_2(S_{t+1}, \arg\max_a Q_1(S_{t+1}, a)) - Q_1(S_t, A_t) \right]$$

以 0.5 概率更新 Q1 或 Q2。

```python
class DoubleQLearning:
    """双 Q 学习算法"""
    
    def __init__(
        self, 
        n_states, 
        n_actions, 
        learning_rate=0.1,
        discount_factor=0.9,
        epsilon=0.1
    ):
        self.n_states = n_states
        self.n_actions = n_actions
        self.alpha = learning_rate
        self.gamma = discount_factor
        self.epsilon = epsilon
        
        # 两个 Q 表
        self.Q1 = defaultdict(lambda: np.zeros(n_actions))
        self.Q2 = defaultdict(lambda: np.zeros(n_actions))
    
    def get_Q(self, state):
        """获取平均 Q 值"""
        return (self.Q1[state] + self.Q2[state]) / 2
    
    def select_action(self, state, training=True):
        if training and np.random.random() < self.epsilon:
            return np.random.randint(self.n_actions)
        else:
            return np.argmax(self.get_Q(state))
    
    def update(self, state, action, reward, next_state, done):
        if done:
            target = reward
        else:
            # 随机选择更新哪个 Q 表
            if np.random.random() < 0.5:
                # 用 Q1 选择动作，用 Q2 评估
                best_action = np.argmax(self.Q1[next_state])
                target = reward + self.gamma * self.Q2[next_state][best_action]
                td_error = target - self.Q1[state][action]
                self.Q1[state][action] += self.alpha * td_error
            else:
                # 用 Q2 选择动作，用 Q1 评估
                best_action = np.argmax(self.Q2[next_state])
                target = reward + self.gamma * self.Q1[next_state][best_action]
                td_error = target - self.Q2[state][action]
                self.Q2[state][action] += self.alpha * td_error
    
    def get_policy(self):
        return {s: np.argmax(self.get_Q(s)) for s in self.Q1}
```

## 7. 算法对比与选择

### 7.1 综合对比

| 算法 | 策略类型 | 更新目标 | 偏差 | 方差 | 适用场景 |
|------|----------|----------|------|------|----------|
| **MC** | 在策略 | $G_t$ | 无偏 | 高 | episode 任务 |
| **TD(0)** | 在策略 | $R + \gamma V(s')$ | 有偏 | 低 | 通用 |
| **SARSA** | 在策略 | $R + \gamma Q(s', a')$ | 有偏 | 中 | 考虑探索代价 |
| **Q-Learning** | 离策略 | $R + \gamma \max_a Q(s', a)$ | 正向偏差 | 低 | 学习最优策略 |
| **期望SARSA** | 在策略 | $R + \gamma \mathbb{E}[Q]$ | 有偏 | 低 | 方差敏感 |
| **Double Q** | 离策略 | $R + \gamma Q_2(s', \arg\max Q_1)$ | 无偏差 | 低 | 高估问题严重 |

### 7.2 选择指南

```
选择决策树：

是否需要 episode 完整轨迹？
├── 是 → 蒙特卡洛
└── 否 → TD 方法

是否关心探索的实际代价？
├── 是 → SARSA 或 期望SARSA
└── 否 → Q-Learning 或 Double Q

是否担心过高估计？
├── 是 → Double Q-Learning
└── 否 → 标准 Q-Learning
```

### 8.1 通用智能体接口

```python
from abc import ABC, abstractmethod
from collections import defaultdict
import numpy as np

class BaseAgent(ABC):
    """强化学习智能体基类"""
    
    def __init__(self, n_actions, learning_rate=0.1, discount_factor=0.9, epsilon=0.1):
        self.n_actions = n_actions
        self.alpha = learning_rate
        self.gamma = discount_factor
        self.epsilon = epsilon
        self.Q = defaultdict(lambda: np.zeros(n_actions))
    
    @abstractmethod
    def update(self, state, action, reward, next_state, done, **kwargs):
        """更新 Q 值"""
        pass
    
    def select_action(self, state, training=True):
        """ε-贪心动作选择"""
        if training and np.random.random() < self.epsilon:
            return np.random.randint(self.n_actions)
        return np.argmax(self.Q[state])
    
    def save(self, filepath):
        """保存 Q 表"""
        import pickle
        with open(filepath, 'wb') as f:
            pickle.dump(dict(self.Q), f)
    
    def load(self, filepath):
        """加载 Q 表"""
        import pickle
        with open(filepath, 'rb') as f:
            self.Q = defaultdict(lambda: np.zeros(self.n_actions), pickle.load(f))


class QLearningAgent(BaseAgent):
    def update(self, state, action, reward, next_state, done, **kwargs):
        if done:
            target = reward
        else:
            target = reward + self.gamma * np.max(self.Q[next_state])
        self.Q[state][action] += self.alpha * (target - self.Q[state][action])


class SARSAgent(BaseAgent):
    def update(self, state, action, reward, next_state, done, next_action=None, **kwargs):
        if done:
            target = reward
        else:
            target = reward + self.gamma * self.Q[next_state][next_action]
        self.Q[state][action] += self.alpha * (target - self.Q[state][action])
```

### 8.2 训练与评估框架

```python
def train_and_evaluate(env, agent, episodes=1000, eval_interval=100, eval_episodes=10):
    """
    训练并定期评估智能体
    
    参数：
        env: 环境
        agent: 智能体
        episodes: 训练回合数
        eval_interval: 评估间隔
        eval_episodes: 每次评估的回合数
    
    返回：
        training_rewards: 训练奖励历史
        eval_rewards: 评估奖励历史
    """
    training_rewards = []
    eval_rewards = []
    
    for episode in range(episodes):
        state = env.reset()
        total_reward = 0
        
        while not env.is_terminal(state):
            action = agent.select_action(state, training=True)
            next_state, reward, done = env.step(action)
            
            # SARSA 需要下一个动作
            if isinstance(agent, SARSAgent):
                next_action = agent.select_action(next_state, training=True)
                agent.update(state, action, reward, next_state, done, next_action=next_action)
                action = next_action
            else:
                agent.update(state, action, reward, next_state, done)
            
            total_reward += reward
            state = next_state
        
        training_rewards.append(total_reward)
        
        # 定期评估
        if (episode + 1) % eval_interval == 0:
            avg_reward = evaluate(env, agent, eval_episodes)
            eval_rewards.append(avg_reward)
            print(f"Episode {episode + 1}, Eval Reward: {avg_reward:.2f}")
    
    return training_rewards, eval_rewards


def evaluate(env, agent, episodes=10):
    """评估智能体（不探索）"""
    total_rewards = []
    
    for _ in range(episodes):
        state = env.reset()
        total_reward = 0
        
        while not env.is_terminal(state):
            action = agent.select_action(state, training=False)
            next_state, reward, done = env.step(action)
            total_reward += reward
            state = next_state
        
        total_rewards.append(total_reward)
    
    return np.mean(total_rewards)
```

| **Double Q** | 消除过高估计偏差 | ⭐⭐⭐⭐ |

**算法对比速查**：

| 算法 | 类型 | 目标 | 更新公式 |
|------|------|------|----------|
| MC | 在策略 | $G_t$ | $Q \leftarrow Q + \alpha[G - Q]$ |
| SARSA | 在策略 | $R + \gamma Q(s', a')$ | $Q \leftarrow Q + \alpha[R + \gamma Q' - Q]$ |
| Q-Learning | 离策略 | $R + \gamma \max_a Q(s', a)$ | $Q \leftarrow Q + \alpha[R + \gamma \max Q' - Q]$ |
| 期望 SARSA | 在策略 | $R + \gamma \mathbb{E}[Q]$ | $Q \leftarrow Q + \alpha[R + \gamma \sum \pi Q' - Q]$ |
| Double Q | 离策略 | $R + \gamma Q_2(s', \arg\max Q_1)$ | 交叉更新两个 Q 表 |

**核心公式速查**：

| 公式 | 表达式 |
|------|--------|
| TD 误差 | $\delta_t = R_{t+1} + \gamma V(S_{t+1}) - V(S_t)$ |
| TD(0) 更新 | $V(S_t) \leftarrow V(S_t) + \alpha \delta_t$ |
| Q-Learning 更新 | $Q(s,a) \leftarrow Q(s,a) + \alpha[r + \gamma \max_{a'} Q(s',a') - Q(s,a)]$ |
| SARSA 更新 | $Q(s,a) \leftarrow Q(s,a) + \alpha[r + \gamma Q(s',a') - Q(s,a)]$ |

**一句话总结**：无模型方法通过与环境交互获取样本，利用 TD 误差高效地学习价值函数，是强化学习从理论走向实践的关键一步。
