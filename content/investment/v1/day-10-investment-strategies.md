# Day 10: 投资策略选择

> **核心问题**：你应该选择什么投资策略？主动还是被动？

---

## 一、主动投资 vs 被动投资

### 1.1 主动投资与被动投资的本质区别

**第一性原理**：

```
主动投资的假设：
1. 市场不是完全有效的
2. 通过研究和分析可以战胜市场
3. 信息优势或分析能力可以产生超额收益

被动投资的假设：
1. 市场是有效的或接近有效的
2. 战胜市场非常困难
3. 低成本和分散投资是关键

核心问题：
你能持续战胜市场吗？

如果"能" → 主动投资
如果"不能" → 被动投资
```

**两者的对比**：

| 维度 | 主动投资 | 被动投资 |
|------|---------|---------|
| 目标 | 战胜市场 | 获得市场平均收益 |
| 方法 | 选股、择时 | 持有指数 |
| 成本 | 高（研究成本、交易成本） | 低（管理费低、交易少） |
| 时间 | 需要大量时间 | 不需要太多时间 |
| 技能 | 需要专业技能 | 不需要专业技能 |
| 风险 | 可能跑输市场 | 获得市场收益 |
| 分散度 | 较低（集中持股） | 较高（持有整个市场） |

### 1.2 主动投资能战胜市场吗？

**实证数据**：

```
美国市场数据（1945-2020）：

主动基金战胜指数的比例：
- 1年期：约40%
- 3年期：约30%
- 5年期：约20%
- 10年期：约10%
- 20年期：约5%

结论：
时间越长，主动基金战胜指数的概率越低

原因：
1. 市场有效性：信息很快被定价
2. 成本劣势：主动基金费用高
3. 幸存者偏差：差的基金被关闭
4. 零和博弈：扣除费用后是负和博弈
```

**数学证明**：

```
市场平均收益 = 所有投资者的平均收益

主动投资者的收益 = 市场平均收益 + 超额收益 - 成本

所有主动投资者的平均超额收益 = 0（零和博弈）

主动投资者的平均收益 = 市场平均收益 - 成本

结论：
主动投资者的平均收益 < 市场平均收益

原因：
主动投资的成本（研究成本、交易成本、管理费）高于被动投资
```

**为什么还有人做主动投资？**

```
原因一：幸存者偏差
我们只看到成功的主动投资者
失败的主动投资者被遗忘

原因二：过度自信
大多数投资者认为自己能战胜市场
但实际只有少数人能做到

原因三：小部分人确实能战胜市场
巴菲特、彼得·林奇等确实长期战胜市场
但这是极少数

原因四：市场并非完全有效
有些市场、有些时期存在定价错误
可以被发现和利用

主动投资成功的条件：
1. 信息优势（比别人知道更多）
2. 分析优势（比别人分析更好）
3. 行为优势（比别人更理性）
4. 成本控制（费用和交易成本）
```

### 1.3 如何选择？

**决策框架**：

```
问题一：你有时间吗？
- 有充足时间研究 → 可以考虑主动
- 时间有限 → 被动

问题二：你有专业知识吗？
- 有专业知识和技能 → 可以考虑主动
- 没有专业知识 → 被动

问题三：你能控制情绪吗？
- 能理性决策 → 可以考虑主动
- 容易情绪化 → 被动

问题四：你有信息优势吗？
- 有独特信息来源 → 可以考虑主动
- 信息来源普通 → 被动

问题五：你的目标是什么？
- 追求超额收益 → 可以尝试主动
- 追求市场平均收益 → 被动

问题六：你能接受跑输市场吗？
- 能接受短期跑输 → 可以尝试主动
- 不能接受 → 被动

如果大多数答案倾向于"被动"，选择被动投资
如果大多数答案倾向于"主动"，可以尝试主动投资
```

**务实建议**：

```
对于大多数人：

核心组合：被动投资（70-90%）
- 指数基金
- 低成本
- 分散投资

卫星组合：主动投资（10-30%）
- 尝试战胜市场
- 限制金额，控制风险
- 学习和积累经验

好处：
1. 大部分资产获得市场平均收益
2. 小部分资产尝试超额收益
3. 即使主动投资失败，影响有限
```

## 二、主要投资策略

### 2.1 价值投资

**核心哲学**：

```
价值投资的三个核心理念：

1. 价格围绕价值波动
   内在价值是价格的锚
   价格有时高于价值，有时低于价值
   但长期会回归价值

2. 安全边际
   买入价格要显著低于内在价值
   为错误和不确定性留出空间

3. 能力圈
   只投资自己理解的公司
   不懂不投
```

**第一性原理推导**：

```
价值投资的逻辑：

前提一：公司有内在价值
内在价值 = 未来现金流的现值

前提二：价格有时偏离价值
原因：市场情绪、信息不对称、短期因素

前提三：长期价格会回归价值
原因：套利机制、公司回购、价值实现

结论：在价格低于价值时买入，等待价格回归

如何判断价格低于价值？
1. 估值方法：PE、PB、DCF
2. 相对历史估值
3. 相对同行业估值

安全边际的意义：
1. 为估值错误留空间
2. 为不确定性留空间
3. 降低损失风险
```

**价值投资的实践**：

```
选股标准：

1. 低估值
   PE < 行业平均
   PB < 行业平均
   PEG < 1

2. 安全边际
   价格 < 内在价值 × 0.7
   （至少30%安全边际）

3. 财务健康
   资产负债率 < 60%
   经营现金流 > 净利润
   ROE > 10%

4. 护城河
   品牌优势
   成本优势
   网络效应
   转换成本

5. 管理层
   诚信
   能力
   与股东利益一致
```

**价值投资的挑战**：

```
挑战一：价值陷阱
低估值可能是因为公司基本面恶化
需要区分：
- 暂时低估 vs 永久低估
- 周期性低估 vs 结构性低估

挑战二：等待时间
价格回归价值可能需要很长时间
"市场保持非理性的时间，可能比你保持偿付能力的时间更长"
需要耐心

挑战三：情绪控制
低估值的股票往往不被市场看好
需要有勇气逆向投资

挑战四：估值难度
内在价值难以精确计算
需要保守估计
```

**代表人物**：

```
本杰明·格雷厄姆：
- "证券分析"之父
- 强调安全边际
- "烟蒂"投资法（买入极度低估的股票）

沃伦·巴菲特：
- 格雷厄姆的学生
- 从"烟蒂"投资进化到"伟大公司"
- "以合理价格买入优秀公司，胜过以优秀价格买入平庸公司"

塞思·卡拉曼：
- 强调安全边际
- 绝对收益导向
- 现金等待机会
```

### 2.2 成长投资

**核心哲学**：

```
成长投资的核心：

1. 增长创造价值
   公司的内在价值来自未来增长
   高增长值得高估值

2. 寻找成长股
   收入增长快
   利润增长快
   市场扩张快

3. 持有成长期
   在成长期持有
   在成长放缓前卖出
```

**第一性原理推导**：

```
成长投资的逻辑：

前提一：公司价值来自未来现金流
DCF公式：V = Σ CFt / (1+r)^t

前提二：高增长带来高现金流
如果公司快速增长，未来现金流会大幅增加

前提三：高增长公司值得高估值
即使当前PE高，但增长会降低未来PE

结论：找到能持续高增长的公司，享受增长带来的价值

关键问题：增长是否可持续？

区分：
- 真成长：增长可持续
- 伪成长：增长不可持续
```

**成长投资的实践**：

```
选股标准：

1. 高增长
   收入增长率 > 20%
   利润增长率 > 20%
   增长可持续

2. 大市场
   目标市场大
   市场份额低
   有增长空间

3. 竞争优势
   技术领先
   网络效应
   规模优势

4. 好的管理层
   战略清晰
   执行力强
   资本配置能力强

5. 合理估值
   PEG < 1.5
   或增长能支撑高估值
```

**成长投资的挑战**：

```
挑战一：估值风险
成长股通常高估值
如果增长不及预期，估值会大幅下降
"戴维斯双杀"：盈利下降 × 估值下降 = 股价大跌

挑战二：增长不可持续
所有增长都会放缓
判断增长何时放缓非常困难

挑战三：竞争
高增长行业会吸引竞争
竞争可能侵蚀利润

挑战四：情绪波动
成长股对市场情绪敏感
波动大，需要承受
```

**代表人物**：

```
菲利普·费雪：
- "怎样选择成长股"作者
- 强调定性分析
- 关注管理层、研发、销售

彼得·林奇：
- "战胜华尔街"作者
- PEG指标
- "买你所了解的"

威廉·欧奈尔：
- CANSLIM模型
- 强调盈利增长、新产品、领导地位
- 技术分析结合基本面
```

### 2.3 指数投资

**核心哲学**：

```
指数投资的核心：

1. 市场有效
   大多数信息已经被定价
   战胜市场非常困难

2. 低成本
   管理费低
   交易成本低
   税务效率高

3. 分散投资
   持有整个市场
   消除个股风险

4. 长期持有
   不择时
   不选股
```

**第一性原理推导**：

```
指数投资的逻辑：

前提一：市场平均收益是所有投资者的平均
所有投资者的平均收益 = 市场收益

前提二：主动投资的成本高于被动投资
主动成本 = 管理费 + 研究成本 + 交易成本
被动成本 = 低管理费

前提三：扣除成本后，主动投资者平均收益低于市场
主动平均收益 = 市场收益 - 成本差

结论：长期来看，被动投资收益 > 主动投资平均收益

指数投资的数学：
被动投资者收益 = 市场收益 - 低成本
主动投资者平均收益 = 市场收益 - 高成本
被动投资者收益 > 主动投资者平均收益

这是数学上的必然，不是观点
```

**指数投资的实践**：

```
选择指数：

宽基指数：
- 沪深300：A股大盘股
- 中证500：A股中盘股
- 标普500：美股大盘股
- MSCI World：全球股票

行业指数：
- 科技指数
- 医药指数
- 消费指数

债券指数：
- 国债指数
- 企业债指数

选择基金：

标准：
- 跟踪误差小
- 管理费低
- 规模适中
- 流动性好

推荐：
- 管理费 < 0.5%
- 跟踪误差 < 0.5%
- 规模 > 10亿
```

**指数投资的优势**：

```
优势一：低成本
指数基金管理费通常0.1-0.5%
主动基金管理费通常1-2%
差距看似小，但复利效应巨大

例子：
年化收益8%
指数基金费0.2% → 实际收益7.8%
主动基金费1.5% → 实际收益6.5%
30年后：
指数基金：100万 × (1+7.8%)^30 = 905万
主动基金：100万 × (1+6.5%)^30 = 696万
差距：209万

优势二：分散风险
持有几百只股票
单一股票风险被分散

优势三：节省时间
不需要研究个股
不需要择时
只需要定期投资和再平衡

优势四：税务效率
交易少，资本利得税少
```

**指数投资的局限**：

```
局限一：获得市场平均收益
不能战胜市场
只能获得市场收益

局限二：市场下跌时无法避免
指数投资不择时
市场下跌时跟着下跌

局限三：跟踪误差
指数基金不能完美跟踪指数
存在微小误差

局限四：市场泡沫
指数投资包含所有股票
包括被高估的股票
市场泡沫时会买在高点
```

**代表人物**：

```
约翰·博格：
- 先锋集团创始人
- 指数基金之父
- "常识投资法"

尤金·法玛：
- 有效市场假说提出者
- 诺贝尔经济学奖得主
- 理论基础

大卫·斯文森：
- 耶鲁大学捐赠基金主管
- 资产配置理论
- 推荐个人投资者用指数基金
```

### 2.4 因子投资

**核心哲学**：

```
因子投资的核心：

1. 收益可以分解为因子
   股票收益 = 市场收益 + 因子收益 + 特质收益

2. 某些因子长期有超额收益
   价值因子、质量因子、动量因子、低波动因子等

3. 通过因子暴露可以获得超额收益
   持有因子暴露高的股票
```

**主要因子**：

```
因子一：价值因子
定义：低估值股票
衡量指标：PE低、PB低、P/CF低
历史收益：长期跑赢市场
原因：风险补偿、行为偏差

因子二：质量因子
定义：高质量公司
衡量指标：高ROE、低负债、稳定盈利
历史收益：长期跑赢市场
原因：质量公司风险低、盈利稳定

因子三：动量因子
定义：过去表现好的股票
衡量指标：过去12个月涨幅
历史收益：中期跑赢市场
原因：行为偏差、信息传播慢

因子四：低波动因子
定义：波动率低的股票
衡量指标：低波动率、低Beta
历史收益：风险调整后跑赢市场
原因：彩票效应、杠杆限制

因子五：规模因子
定义：市值小的公司
衡量指标：市值
历史收益：长期跑赢市场
原因：风险补偿、流动性溢价
```

**因子投资的实践**：

```
方法一：单因子投资
选择一个因子
买入因子暴露高的股票

例子：价值因子
买入PE最低的20%股票
定期调仓

方法二：多因子投资
选择多个因子
买入多因子得分高的股票

例子：价值+质量
计算价值得分和质量得分
买入综合得分最高的股票

方法三：因子ETF
购买因子ETF
低成本、简单

例子：
- 价值ETF
- 质量ETF
- 多因子ETF
```

**因子投资的挑战**：

```
挑战一：因子失效
因子可能长期失效
例如：价值因子在2010-2020年表现不佳

挑战二：拥挤交易
因子投资流行后，因子变得拥挤
超额收益下降

挑战三：交易成本
因子投资需要频繁调仓
交易成本侵蚀收益

挑战四：因子择时
何时超配哪个因子？
非常困难
```

**代表人物**：

```
尤金·法玛、肯尼斯·弗兰奇：
- 三因子模型
- 市场、规模、价值

克利夫·阿斯尼斯：
- AQR资本创始人
- 多因子投资
- 因子研究

罗伯特·诺维-马克克斯：
- 因子投资的局限
- 因子择时
```

## 三、定投策略

### 3.1 定投的本质

**第一性原理**：

```
定投 = 定期定额投资

定投的本质：
1. 分散时点风险
   不知道何时是最好的买入时机
   分散在不同时点买入
   平均买入成本

2. 自动化投资
   不需要择时
   不需要情绪决策

3. 纪律投资
   强制定期投资
   克服拖延
```

**定投的数学原理**：

```
假设：
每月投资1000元
价格从10元跌到5元再涨回10元

时点1：价格10元，买入100股
时点2：价格8元，买入125股
时点3：价格6元，买入167股
时点4：价格5元，买入200股
时点5：价格6元，买入167股
时点6：价格8元，买入125股
时点7：价格10元，买入100股

总投入：7000元
总股数：984股
平均成本：7000/984 = 7.11元

当前价格：10元
当前价值：984 × 10 = 9840元
收益：9840 - 7000 = 2840元
收益率：40.6%

如果一次性投入：
时点1买入：7000元，700股，价值7000元，收益0%

结论：
在价格波动时，定投可以降低平均成本
```

### 3.2 定投的优势

**优势一：分散时点风险**

```
不知道何时是最好的买入时机
定投分散在不同时点买入
避免一次性买在高点

案例：
2007年10月股市高点一次性买入
vs 2007-2017年定投

一次性买入：10年后可能还没回本
定投：在不同时点买入，平均成本更低
```

**优势二：自动化、纪律化**

```
定投不需要择时
不需要判断何时买入
只需要坚持定期投资

避免：
- 择时错误
- 情绪影响
- 拖延不投
```

**优势三：适合工薪阶层**

```
工薪阶层每月有固定收入
定投匹配收入节奏
每月投资一部分收入

无需大笔资金
无需频繁决策
适合长期投资
```

### 3.3 定投的局限

**局限一：无法获得最高收益**

```
定投是分散投资
如果在最低点一次性买入，收益更高

但问题是：
谁能预测最低点？
大多数人在最低点时不敢买入

定投的哲学：
不求最优，但求不错
```

**局限二：持续下跌时亏损**

```
如果市场持续下跌
定投也会亏损

但相比一次性买入：
定投亏损更少
因为平均成本更低

而且：
市场终会反弹
定投积累了更多份额
反弹时收益更高
```

**局限三：不适合短期投资**

```
定投需要时间摊平成本
短期定投效果不明显

建议：
定投期限 > 3年
最好 > 5年
```

### 3.4 定投策略改进

**改进一：价值平均**

```
价值平均 = 保持投资组合价值按目标增长

方法：
设定目标价值增长
如果实际价值低于目标，多买
如果实际价值高于目标，少买或不买

例子：
目标：每月增加10000元

月份1：目标10000，实际10000，买入10000
月份2：目标20000，实际15000，买入5000
月份3：目标30000，实际25000，买入5000
月份4：目标40000，实际45000，卖出5000

效果：
自动实现"低买高卖"
跌的时候多买，涨的时候少买或不买

局限：
需要资金灵活调配
可能需要卖出
```

**改进二：智能定投**

```
智能定投 = 根据市场估值调整投资金额

方法：
市场低估时，增加投资金额
市场高估时，减少投资金额

判断标准：
PE低于历史中位数：低估，增加投资
PE高于历史中位数：高估，减少投资

例子：
基础金额：1000元
低估时：投资1500元
正常时：投资1000元
高估时：投资500元或不投资

效果：
在低估时积累更多份额
在高估时减少买入

局限：
需要判断市场估值
可能判断错误
```

**改进三：止盈不止损**

```
方法：
定投不止损，坚持投入
设定止盈目标，达到后卖出

止盈策略：
目标止盈：达到目标收益率卖出
移动止盈：收益率回撤一定比例卖出

例子：
定投5年，年化目标10%
达到目标后卖出
继续定投

效果：
锁定收益
避免回撤

局限：
可能错过继续上涨
需要重新开始定投
```

## 四、再平衡机制

### 4.1 再平衡的本质

**第一性原理**：

```
再平衡 = 定期调整资产配置，恢复目标比例

为什么需要再平衡？

原因一：资产价格变化导致配置偏离
股票上涨 → 股票比例增加 → 风险增加
债券上涨 → 债券比例增加 → 收益降低

原因二：保持风险水平
目标配置是根据风险承受能力设定的
配置偏离会导致风险偏离

原因三：自动高卖低买
卖出上涨资产，买入下跌资产
自动实现逆向投资
```

**再平衡的数学**：

```
初始配置：60%股票，40%债券
投资金额：100万

一年后：
股票涨20%，债券涨5%
股票价值：60 × 1.2 = 72万
债券价值：40 × 1.05 = 42万
总价值：114万

实际配置：
股票：72/114 = 63.2%
债券：42/114 = 36.8%

偏离：
股票偏离：63.2% - 60% = 3.2%
债券偏离：36.8% - 40% = -3.2%

再平衡操作：
卖出股票：114 × 63.2% - 114 × 60% = 3.6万
买入债券：3.6万

再平衡后：
股票：114 × 60% = 68.4万
债券：114 × 40% = 45.6万

效果：
- 卖出了部分上涨的股票
- 买入了部分相对下跌的债券
- 自动"高卖低买"
```

### 4.2 再平衡的方法

**方法一：时间再平衡**

```
规则：每隔固定时间再平衡

频率选择：
每月再平衡：太频繁，交易成本高
每季度再平衡：适中
每年再平衡：足够，交易成本低

优点：
简单易执行
规律性强

缺点：
可能偏离较大时才再平衡
错过及时调整的机会
```

**方法二：阈值再平衡**

```
规则：偏离超过一定比例时再平衡

阈值选择：
偏离5%：较严格
偏离10%：适中
偏离15%：较宽松

例子：
目标配置：60%股票
阈值：5%
当股票比例超过65%或低于55%时，再平衡

优点：
及时调整
偏离大时才交易，交易成本可控

缺点：
需要监控
可能长期不触发
```

**方法三：组合再平衡**

```
规则：结合时间和阈值

方法：
每季度检查
如果偏离超过阈值，再平衡
否则不操作

例子：
每季度检查
偏离超过5%，再平衡
否则等到下季度

优点：
定期检查
有阈值控制
平衡了交易成本和调整及时性

推荐方法
```

### 4.3 再平衡的效果

**效果一：控制风险**

```
再平衡保持目标配置
防止股票比例过高导致风险增加

案例：
初始：60%股票，40%债券
牛市后：80%股票，20%债券
如果市场下跌20%：
80%股票跌20% = -16%
20%债券涨2% = +0.4%
总亏损：-15.6%

如果及时再平衡：
保持60%股票，40%债券
60%股票跌20% = -12%
40%债券涨2% = +0.8%
总亏损：-11.2%

再平衡减少了4.4%的亏损
```

**效果二：提高收益**

```
再平衡自动实现高卖低买

研究数据（美国市场1970-2020）：

| 策略 | 年化收益 |
|------|---------|
| 不再平衡 | 8.5% |
| 年度再平衡 | 8.8% |
| 季度再平衡 | 8.9% |

再平衡比不再平衡收益高0.3-0.4%

原因：
在牛市后期卖出部分股票
在熊市后期买入部分股票
```

**效果三：提供纪律**

```
再平衡是机械的
不受情绪影响

避免：
- 牛市后期追涨
- 熊市后期恐慌卖出

自动：
- 牛市后期部分获利了结
- 熊市后期低位加仓
```

### 4.4 再平衡的注意事项

**注意一：交易成本**

```
再平衡涉及交易
需要考虑交易成本

交易成本包括：
- 申购费
- 赎回费
- 买卖价差
- 资本利得税（如适用）

策略：
- 选择低费用的基金
- 减少再平衡频率
- 使用阈值再平衡
```

**注意二：税务影响**

```
卖出盈利资产可能产生资本利得税

策略：
- 在税收优惠账户（如养老金账户）再平衡
- 用新资金调整配置
- 用分红再投资
```

**注意三：资金流入流出**

```
有新资金时：
买入低配资产
自动实现再平衡

有资金流出时：
卖出超配资产
自动实现再平衡

策略：
- 用新资金流入实现再平衡
- 减少交易
```

## 本日总结

### 核心概念回顾

1. **主动vs被动**：主动投资试图战胜市场，被动投资获得市场平均收益
2. **价值投资**：在价格低于价值时买入，强调安全边际
3. **成长投资**：寻找高增长公司，享受增长带来的价值
4. **指数投资**：低成本、分散投资，获得市场平均收益
5. **因子投资**：通过因子暴露获得超额收益
6. **定投策略**：定期定额投资，分散时点风险
7. **再平衡**：定期调整配置，控制风险、高卖低买

### 投资策略选择框架

```
第一步：确定投资哲学
- 你相信市场有效吗？
- 你有能力战胜市场吗？

第二步：选择策略
- 主动：价值投资、成长投资、因子投资
- 被动：指数投资
- 混合：核心+卫星

第三步：实施策略
- 选股标准
- 买入条件
- 卖出条件
- 仓位管理

第四步：定期回顾
- 策略是否有效？
- 是否需要调整？
```
