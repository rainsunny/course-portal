# Day 6: 估值方法论（下）

> **核心问题**：如何用相对估值法判断一家公司是否便宜？

---

## 一、相对估值法的基本原理

### 1.1 相对估值 vs 绝对估值

**第一性原理对比**：

```
绝对估值（DCF）：
- 从第一性原理出发
- 计算内在价值
- 回答："这家公司值多少钱？"

相对估值（PE/PB等）：
- 从市场比较出发
- 计算相对价值
- 回答："这家公司相对于其他公司，是否便宜？"
```

**相对估值的逻辑**：

```
假设：市场对相似公司的定价是"合理的"

如果：
- 公司A和公司B各方面都很相似
- 公司A的PE是20倍
- 公司B的PE是10倍

那么：公司B相对便宜

但注意：
相对估值不能告诉你公司A和公司B的绝对价值
只能告诉你谁相对便宜
```

### 1.2 相对估值的类型

**按估值指标分类**：

| 指标 | 计算公式 | 含义 |
|------|---------|------|
| PE | 股价/每股收益 | 多少年收回成本 |
| PB | 股价/每股净资产 | 相对于账面价值的溢价 |
| PS | 股价/每股销售额 | 相对于销售额的溢价 |
| PEG | PE/增长率 | PE相对增长率的比值 |
| EV/EBITDA | 企业价值/EBITDA | 相对于经营利润的溢价 |
| EV/销售额 | 企业价值/销售额 | 相对于销售额的溢价 |

**按比较基准分类**：

| 基准 | 含义 | 用途 |
|------|------|------|
| 历史比较 | 与公司历史估值比较 | 判断是否处于历史低位 |
| 同行比较 | 与同行业公司比较 | 判断相对同行的估值 |
| 市场比较 | 与市场整体比较 | 判断相对市场的估值 |

### 1.3 相对估值的前提假设

**第一性原理分析**：

```
相对估值成立的前提：

假设一：可比公司确实可比
- 业务相似
- 增长相似
- 风险相似
- 盈利能力相似

假设二：市场对可比公司的定价是合理的
- 没有整体泡沫
- 没有整体低估

假设三：差异可以解释
- 如果估值不同，原因是什么？
- 这些原因合理吗？
```

**危险**：

```
相对估值的陷阱：

"这家公司PE只有10倍，同行PE是20倍，所以便宜"

可能的问题：
1. 公司增长预期低于同行
2. 公司风险高于同行
3. 公司盈利质量低于同行
4. 整个行业被高估

结论：相对估值必须结合基本面分析
```

## 二、市盈率（PE）深度解析

### 2.1 PE的定义与计算

**基本公式**：

```
PE = 股价 / 每股收益（EPS）

或

PE = 市值 / 净利润

含义：
- 如果公司盈利不变，多少年能收回投资
- 投资者愿意为1元利润支付多少元
```

**PE的变体**：

| 类型 | 计算方式 | 用途 |
|------|---------|------|
| 静态PE | 股价/去年EPS | 已知数据，但滞后 |
| 动态PE | 股价/预测EPS | 前瞻性，但需要预测 |
| TTM PE | 股价/过去12个月EPS | 最新数据，兼顾滞后性和时效性 |

### 2.2 PE的经济学含义

**第一性原理推导**：

```
从DCF角度看PE：

假设公司永续增长，DCF公式简化为：
P = EPS / (r - g)

其中：
P = 股价
EPS = 每股收益
r = 要求回报率
g = 增长率

变形得：
PE = P / EPS = 1 / (r - g)

这意味着：
PE由增长率和要求回报率决定
```

**PE的决定因素**：

```
PE = f(增长率, 风险, 利率)

1. 增长率：增长越快，PE越高
   原因：未来盈利增长，当前PE虽高但实际回报率高

2. 风险：风险越低，PE越高
   原因：风险低，要求回报率低，折现率低，估值高

3. 利率：利率越低，PE越高
   原因：利率低，折现率低，估值高
```

**案例**：

```
两家公司：
公司A：增长率5%，风险低，PE = 20倍
公司B：增长率2%，风险高，PE = 10倍

为什么PE不同？
- 公司A增长快、风险低 → 要求回报率低 → PE高
- 公司B增长慢、风险高 → 要求回报率高 → PE低

如果公司A和公司B PE相同，那公司A更便宜
```

### 2.3 PE的应用

**应用一：历史比较**

```
某公司当前PE = 15倍
历史PE区间 = 10-30倍
历史中位数 = 18倍

判断：
当前PE低于历史中位数
可能处于历史相对低位

但需验证：
- 基本面是否变化？
- 行业是否变化？
- 市场环境是否变化？
```

**应用二：同行比较**

```
某公司PE = 15倍
同行平均PE = 20倍

判断：
相对同行便宜

但需分析：
- 增长率是否低于同行？
- 风险是否高于同行？
- 盈利质量是否低于同行？

如果各方面相似，则确实便宜
如果公司质量更差，则可能合理
```

**应用三：隐含增长率**

```
反向计算：当前PE隐含什么增长预期？

假设：
要求回报率 r = 10%
当前 PE = 30倍

PE = 1 / (r - g)
30 = 1 / (10% - g)
10% - g = 3.33%
g = 6.67%

意味着：市场预期公司长期增长率约6.67%

问自己：
- 这个增长率合理吗？
- 公司能达到这个增长吗？
```

### 2.4 PE的局限性

**局限一：盈利波动**

```
问题：
周期性行业盈利波动大
盈利低时，PE很高
盈利高时，PE很低

例子：
周期股在周期顶部：
盈利最高 → PE最低 → 看似便宜 → 其实应该卖出

周期股在周期底部：
盈利最低 → PE最高 → 看似贵 → 其实应该买入

结论：周期股不能用简单PE
```

**局限二：盈利质量**

```
问题：
PE只看净利润
但净利润可能被操纵

例子：
公司A：净利润10亿，经营现金流15亿，PE = 20倍
公司B：净利润10亿，经营现金流5亿，PE = 20倍

两家公司PE相同，但质量不同：
- 公司A盈利质量高（现金流>利润）
- 公司B盈利质量低（现金流<利润）

如果用PE比较，会忽视质量差异
```

**局限三：亏损公司**

```
问题：
亏损公司EPS为负，PE无意义

解决：
- 用PS（市销率）
- 用EV/EBITDA
- 等待盈利后再估值
```

**局限四：会计差异**

```
问题：
不同公司会计政策不同，影响PE可比性

例子：
折旧方法：直线法 vs 加速折旧
研发费用：资本化 vs 费用化
存货计价：先进先出 vs 后进先出

这些差异会影响净利润，从而影响PE
```

### 2.5 如何正确使用PE

**第一性原理框架**：

```
正确使用PE的步骤：

第一步：理解PE的经济含义
- PE反映增长预期和风险
- PE高低本身不说明便宜或贵

第二步：选择可比公司
- 业务相似
- 增长相似
- 风险相似

第三步：调整差异
- 增长差异：用PEG
- 风险差异：调整折现率
- 质量差异：看现金流

第四步：结合绝对估值
- 用DCF验证
- 判断整体市场估值

第五步：分析驱动因素
- PE变化的原因是什么？
- 基本面变化还是情绪变化？
```

## 三、市净率（PB）深度解析

### 3.1 PB的定义与计算

**基本公式**：

```
PB = 股价 / 每股净资产

或

PB = 市值 / 净资产

含义：
- 投资者愿意为1元净资产支付多少元
- PB > 1：溢价交易
- PB < 1：折价交易
```

### 3.2 PB的经济学含义

**第一性原理推导**：

```
从ROE（净资产收益率）角度理解PB：

ROE = 净利润 / 净资产

PE = 市值 / 净利润
PB = 市值 / 净资产

因此：
PB = PE × ROE

或：
PB / PE = ROE

这意味着：
PB由PE和ROE共同决定
ROE越高，相同PE下PB越高
```

**进一步推导**：

```
从DCF角度：

假设公司永续增长：
P = EPS / (r - g)
P = BPS × ROE / (r - g)

其中 BPS = 每股净资产

PB = P / BPS = ROE / (r - g)

这意味着：
PB由ROE、增长率、要求回报率决定
```

**PB的决定因素**：

```
PB = f(ROE, 增长率, 风险)

1. ROE：ROE越高，PB越高
   原因：净资产创造利润的能力越强

2. 增长率：增长越快，PB越高
   原因：未来净资产会增长

3. 风险：风险越低，PB越高
   原因：要求回报率低，估值高
```

**案例**：

```
三家公司：
公司A：ROE 20%，PB 3倍
公司B：ROE 10%，PB 1倍
公司C：ROE 5%，PB 0.5倍

如果要求回报率都是10%：
公司A：ROE > 要求回报率 → PB > 1 → 合理
公司B：ROE = 要求回报率 → PB ≈ 1 → 合理
公司C：ROE < 要求回报率 → PB < 1 → 合理

结论：
ROE高的公司应该有高PB
ROE低的公司应该有低PB
如果ROE高但PB低，可能是机会
```

### 3.3 PB的适用场景

**最适合：金融行业**

```
为什么PB适合金融行业？

1. 资产主要是金融资产
   - 银行：贷款、债券
   - 保险：投资组合
   - 券商：金融资产

2. 净资产代表清算价值
   - 金融资产流动性好
   - 账面价值接近真实价值

3. 盈利受周期影响
   - PE波动大
   - PB相对稳定

银行业PB参考：
- 优秀银行：PB 1-1.5倍
- 一般银行：PB 0.5-1倍
- 困境银行：PB < 0.5倍
```

**适合：重资产行业**

```
为什么PB适合重资产行业？

1. 固定资产占比高
   - 制造业、公用事业
   - 净资产反映资产价值

2. PB < 1 意味着：
   - 市值低于清算价值
   - 可能是买入机会

但需注意：
- 资产是否贬值？
- 资产是否闲置？
- 清算成本多少？
```

**不适合：轻资产行业**

```
为什么PB不适合轻资产行业？

1. 无形资产多
   - 品牌、专利、客户关系
   - 这些不在账面上

2. 净资产被低估
   - 账面净资产 << 真实价值
   - PB会很高，但实际不贵

例子：
科技公司：PB可能10倍以上
咨询公司：PB可能5倍以上
品牌公司：PB可能3倍以上

这些高PB可能是合理的
```

### 3.4 PB vs PE：如何选择

| 情况 | 适合用PB | 适合用PE |
|------|---------|---------|
| 行业特点 | 金融、重资产 | 消费、科技 |
| 盈利稳定性 | 盈利波动大 | 盈利稳定 |
| ROE | ROE稳定 | ROE变化大 |
| 周期性 | 周期性行业 | 非周期行业 |
| 资产结构 | 有形资产为主 | 无形资产为主 |

**综合使用**：

```
最理想：同时看PE和PB

案例一：低PE + 低PB
- 可能是被低估
- 也可能是基本面恶化
- 需要深入分析

案例二：低PE + 高PB
- 可能是高ROE公司
- 盈利能力强，但增长慢
- 检查ROE和增长

案例三：高PE + 低PB
- 可能是盈利暂时下降
- ROE低，但有改善空间
- 检查盈利恢复可能

案例四：高PE + 高PB
- 可能是成长股
- 高增长预期
- 检查增长是否可持续
```

## 四、市销率（PS）深度解析

### 4.1 PS的定义与计算

**基本公式**：

```
PS = 股价 / 每股销售额

或

PS = 市值 / 销售额

含义：
- 投资者愿意为1元销售额支付多少元
```

### 4.2 PS的经济学含义

**第一性原理推导**：

```
PS = PE × 净利润率

推导：
PE = 市值 / 净利润
净利润率 = 净利润 / 销售额
PS = 市值 / 销售额

因此：
PS = PE × 净利润率

或：
PE = PS / 净利润率

这意味着：
相同PS下，利润率越高，PE越低
```

**PS的优势**：

```
1. 销售额比利润更稳定
   - 利润受成本、费用影响
   - 销售额更少波动

2. 适用于亏损公司
   - PE对亏损公司无意义
   - PS仍可使用

3. 较少被操纵
   - 销售额比利润更难操纵
   - 更反映真实业务规模
```

**PS的局限**：

```
1. 忽视盈利能力
   - 只看销售规模，不看利润
   - 可能高估亏损公司

2. 利润率差异大
   - 不同行业利润率差异大
   - PS难以跨行业比较

例子：
软件公司：利润率30%，PS 10倍
零售公司：利润率3%，PS 0.5倍

如果用PS比较，软件公司"贵"
但实际上利润率差异是合理的
```

### 4.3 PS的适用场景

**最适合：高增长、未盈利公司**

```
例子：SaaS公司

特点：
- 高增长（收入增长50%+）
- 亏损（前期投入大）
- 市场关注收入而非利润

估值逻辑：
PS × 销售额 = 市值
如果未来利润率达到目标水平：
市值 = PS × 销售额 = (目标PE × 目标利润率) × 销售额

SaaS行业PS参考：
- 高增长（>50%）：PS 10-20倍
- 中增长（20-50%）：PS 5-10倍
- 低增长（<20%）：PS 2-5倍
```

**适合：利润率稳定的行业**

```
如果行业利润率稳定，PS可以跨公司比较

例子：零售业
行业平均利润率 = 3%
公司A：PS = 0.6倍 → 隐含PE = 20倍
公司B：PS = 0.3倍 → 隐含PE = 10倍

公司B更便宜（如果利润率相似）
```

**不适合：利润率差异大的行业比较**

```
不同行业PS差异大：

科技行业：PS 5-15倍
零售行业：PS 0.3-1倍
制造业：PS 0.5-2倍

这些差异反映了利润率的差异
不能直接比较PS
```

## 五、PEG：增长调整后的PE

### 5.1 PEG的定义与计算

**基本公式**：

```
PEG = PE / 增长率

其中增长率通常用未来3-5年的预期增长率

含义：
- PEG = 1：估值合理
- PEG < 1：低估
- PEG > 1：高估
```

### 5.2 PEG的经济学含义

**第一性原理推导**：

```
从DCF角度：

假设公司高速增长g1持续n年，之后永续增长g：
P = EPS × [增长阶段现值 + 永续阶段现值]

简化（假设增长阶段n年，永续增长g）：
P ≈ EPS × n × (1 + g1)^(n/2) / (r - g)

PE ≈ n × (1 + g1)^(n/2) / (r - g)

如果增长g1越高，PE越高
PEG试图标准化增长差异

经验规则：
PEG = 1 意味着PE与增长匹配
```

**PEG的直觉**：

```
公司A：PE 20倍，增长率20%，PEG = 1
公司B：PE 40倍，增长率40%，PEG = 1

两家公司PEG相同，哪个更便宜？

如果增长能持续：
公司A：20年回本（假设增长20%持续）
公司B：如果增长40%持续，更快回本

但增长很难持续，所以PEG只是参考
```

### 5.3 PEG的应用与局限

**应用**：

```
PEG = PE / G

规则：
PEG < 1：可能低估
PEG = 1：合理
PEG > 1：可能高估
PEG > 2：明显高估

例子：
公司A：PE 30倍，增长30%，PEG = 1 → 合理
公司B：PE 30倍，增长15%，PEG = 2 → 高估
公司C：PE 15倍，增长30%，PEG = 0.5 → 低估
```

**局限一：增长预测不确定性**

```
PEG依赖增长率预测
但增长率预测非常不准

例子：
预测增长30%，实际增长10%
PEG = 30/30 = 1（看似合理）
实际PEG = 30/10 = 3（实际高估）

结论：PEG对增长预测高度敏感
```

**局限二：增长可持续性**

```
高增长难以持续

数据：
连续5年增长>20%的公司比例：约15%
连续10年增长>20%的公司比例：约3%

PEG假设增长持续
但高增长公司增长会回归

修正方法：
PEG = PE / 可持续增长率
```

**局限三：忽视利润率**

```
例子：
公司A：增长20%，利润率10%，PEG = 1
公司B：增长20%，利润率5%，PEG = 1

两家PEG相同，但：
公司A利润率更高 → 盈利质量更好
应该给予更高估值

改进：
用PEG结合利润率
```

## 六、EV/EBITDA：企业价值倍数

### 6.1 为什么用EV/EBITDA？

**PE的问题**：

```
PE = 市值 / 净利润

问题：
1. 资本结构不同影响PE
   - 高杠杆公司：利息费用高，净利润低，PE高
   - 低杠杆公司：利息费用低，净利润高，PE低
   
2. 税率不同影响PE
   - 不同国家税率不同
   - 税收优惠影响净利润

3. 折旧政策不同影响PE
   - 重资产行业折旧大
   - 轻资产行业折旧小
```

**EV/EBITDA的优势**：

```
EV = 企业价值 = 市值 + 净债务
EBITDA = 息税折旧摊销前利润

优势：
1. 不受资本结构影响
   - EV包含债务
   - EBITDA在利息前

2. 不受税率影响
   - EBITDA在税前

3. 不受折旧政策影响
   - EBITDA加回折旧摊销

适用：
- 资本结构差异大的公司比较
- 跨国公司比较
- 并购估值
```

### 6.2 EV/EBITDA的计算

**企业价值（EV）**：

```
EV = 市值 + 债务 - 现金

或更精确：
EV = 市值 + 净债务
   = 市值 + 有息债务 - 现金及等价物

为什么减现金？
- 收购公司时，现金可以用来还债
- 所以现金减少了企业价值
```

**EBITDA**：

```
EBITDA = 净利润 + 所得税 + 利息 + 折旧 + 摊销

或从营业利润出发：
EBITDA = 营业利润 + 折旧 + 摊销

含义：
- 扣除利息、税、折旧、摊销前的利润
- 反映经营盈利能力
```

**案例**：

```
某公司财务数据（单位：亿元）：
市值：100
有息债务：30
现金：10
净利润：15
所得税：5
利息费用：3
折旧：8
摊销：2

计算：
EV = 100 + 30 - 10 = 120亿
EBITDA = 15 + 5 + 3 + 8 + 2 = 33亿
EV/EBITDA = 120 / 33 = 3.6倍
```

### 6.3 EV/EBITDA的应用

**应用一：杠杆收购估值**

```
杠杆收购（LBO）：
- 收购方借大量债务收购公司
- 用公司现金流还债

关注点：
- 公司能产生多少现金流？
- 多少年能还清债务？

EV/EBITDA = 收购价格 / 年现金流
≈ 还清债务的年数（粗略）

例子：
EV/EBITDA = 5倍
意味着约5年的EBITDA能收购公司
如果EBITDA大部分可用于还债，约5年还清
```

**应用二：行业比较**

```
EV/EBITDA适合比较资本结构差异大的公司

例子：航空业
公司A：高杠杆，债务高，现金少
公司B：低杠杆，债务少，现金多

如果用PE：
公司A利息费用高 → 净利润低 → PE高
公司B利息费用低 → 净利润高 → PE低
PE差异来自资本结构，非经营能力

如果用EV/EBITDA：
EV包含债务
EBITDA在利息前
两家公司的EV/EBITDA可比
```

**应用三：跨行业比较**

```
EV/EBITDA参考（美国市场）：

| 行业 | EV/EBITDA |
|------|-----------|
| 科技 | 15-25倍 |
| 医疗 | 12-18倍 |
| 消费 | 10-15倍 |
| 工业 | 8-12倍 |
| 能源 | 5-8倍 |

差异来源：
- 增长预期
- 资本密集度
- 行业周期
```

## 七、估值陷阱：如何避免被"便宜"骗

### 7.1 低PE陷阱

**陷阱一：周期股低PE**

```
周期股在周期顶部：
- 盈利最高
- PE最低
- 看似便宜

实际：
- 盈利即将下降
- 应该卖出，不是买入

案例：航运公司
2021年：盈利暴涨，PE 3倍
2022年：盈利暴跌，PE变负

正确做法：
用"周期调整PE"
= 当前市值 / 过去7-10年平均盈利
```

**陷阱二：低盈利质量**

```
公司A：PE 10倍
公司B：PE 20倍

公司A更便宜？

检查盈利质量：
公司A：净利润10亿，经营现金流5亿 → 盈利质量差
公司B：净利润10亿，经营现金流15亿 → 盈利质量好

公司A的净利润可能：
- 应收账款增加
- 存货增加
- 非经常性收益

结论：低PE可能是低质量
```

**陷阱三：一次性收益**

```
公司：PE 8倍

检查利润构成：
经营利润：5亿
出售资产收益：10亿
净利润：15亿

扣除一次性收益：
调整后净利润：5亿
调整后PE：24倍

结论：PE低是因为一次性收益
```

### 7.2 低PB陷阱

**陷阱一：资产减值风险**

```
公司：PB 0.5倍，看似便宜

检查资产质量：
应收账款：可能有坏账
存货：可能贬值
固定资产：可能过时

如果资产减值：
账面净资产100亿
实际净资产可能只有50亿
真实PB = 1倍

结论：低PB可能反映资产质量差
```

**陷阱二：ROE持续下降**

```
公司：PB 0.8倍

历史ROE：
5年前：15%
3年前：10%
1年前：5%
今年：2%

分析：
ROE持续下降 → 盈利能力恶化
PB低是合理的

结论：低PB反映盈利能力差
```

**陷阱三：清算价值低**

```
公司：PB 0.7倍

检查：
账面净资产：100亿
但清算成本：20亿
清算价值：80亿
市值：70亿

如果清算：
股东得到80亿 - 20亿 = 60亿
低于当前市值

结论：PB低但清算价值更低
```

### 7.3 如何避免估值陷阱

**第一性原理框架**：

```
步骤一：检查盈利质量
- 净利润 vs 经营现金流
- 非经常性收益
- 会计政策变化

步骤二：检查可持续性
- 盈利是周期性的吗？
- 盈利是暂时的吗？
- 行业趋势如何？

步骤三：检查资产负债表
- 资产质量
- 债务水平
- 或有负债

步骤四：检查增长
- 增长来源
- 增长可持续性
- 增长质量

步骤五：多维度估值
- PE + PB + PS + EV/EBITDA
- 相对估值 + 绝对估值
- 历史比较 + 同行比较
```

**安全边际原则**：

```
估值不是精确科学，而是模糊判断

格雷厄姆的安全边际：
买入价格 < 内在价值 × (1 - 安全边际)

建议安全边际：
- 高确定性公司：20-30%
- 中等确定性公司：30-50%
- 低确定性公司：50%+

例子：
内在价值 = 100元
安全边际 = 30%
买入价格 < 70元
```

## 八、安全边际：为什么必须要有？

### 8.1 安全边际的定义

**第一性原理**：

```
安全边际 = 内在价值 - 买入价格

或：

安全边际率 = (内在价值 - 买入价格) / 内在价值

含义：
- 为估值错误留出空间
- 为不确定性留出空间
- 为风险留出空间
```

### 8.2 为什么需要安全边际？

**原因一：估值不确定性**

```
估值包含大量假设：
- 未来现金流预测
- 折现率选择
- 永续增长率

每个假设都可能出错

例子：
预测增长率10%，实际8%
估值可能差20-30%

安全边际为预测错误留空间
```

**原因二：信息不对称**

```
投资者 vs 管理层：
- 管理层知道更多信息
- 投资者信息不完整
- 可能存在未知风险

安全边际为未知风险留空间
```

**原因三：市场不可预测**

```
即使估值正确：
- 市场可能继续下跌
- 恢复需要时间
- 短期可能亏损

安全边际提供心理缓冲
```

**原因四：执行风险**

```
投资决策可能出错：
- 分析错误
- 判断错误
- 执行错误

安全边际为人为错误留空间
```

### 8.3 安全边际的量化

**格雷厄姆的建议**：

```
安全边际 = 内在价值 × (1 - 2/3)

即：买入价格不超过内在价值的2/3

例子：
内在价值 = 100元
买入价格 ≤ 67元
安全边际 = 33%
```

**现代实践**：

```
安全边际取决于确定性：

| 确定性 | 安全边际 | 例子 |
|--------|---------|------|
| 高 | 20-30% | 公用事业、消费龙头 |
| 中 | 30-50% | 成长股、周期股 |
| 低 | 50%+ | 困境反转、高风险 |

如何判断确定性：
1. 业务稳定性
2. 财务稳健性
3. 竞争优势持久性
4. 管理层质量
5. 行业前景
```

### 8.4 安全边际的应用

**案例**：

```
某公司：
DCF估值：80-120元（敏感性分析）
同行PE估值：90-110元
历史估值比较：估值区间70-130元

综合判断：
内在价值区间：80-110元
中位数：95元

确定安全边际：
公司属于中等确定性
安全边际：40%

买入价格：
< 95 × (1 - 40%) = 57元

当前价格：75元

决策：
当前价格高于买入价，继续等待
如果价格跌到57元以下，开始买入
```

## 本日总结

### 核心概念回顾

1. **相对估值**：与可比公司比较，判断相对便宜或贵
2. **PE**：市盈率，受增长和风险影响，不适合周期股和亏损股
3. **PB**：市净率，受ROE影响，适合金融和重资产行业
4. **PS**：市销率，适合亏损但高增长的公司
5. **PEG**：增长调整后的PE，对增长预测敏感
6. **EV/EBITDA**：不受资本结构影响，适合并购估值
7. **估值陷阱**：低估值可能是低质量或暂时因素
8. **安全边际**：为不确定性和错误留出空间

### 估值方法选择指南

| 情况 | 推荐方法 | 原因 |
|------|---------|------|
| 盈利稳定 | PE、DCF | 盈利可预测 |
| 盈利波动大 | PB、EV/EBITDA | 避开盈利波动 |
| 亏损但高增长 | PS | PE无意义 |
| 周期性行业 | PB、周期调整PE | 避开周期顶部陷阱 |
| 高杠杆 | EV/EBITDA | 不受资本结构影响 |
| 跨行业比较 | EV/EBITDA | 消除资本结构差异 |

### 相对估值检查清单

```
□ 确认可比公司真的可比
□ 理解估值差异的原因
□ 检查盈利质量
□ 检查增长可持续性
□ 做敏感性分析
□ 结合绝对估值验证
□ 留足安全边际
```
