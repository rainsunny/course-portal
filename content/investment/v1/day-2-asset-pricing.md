# Day 2: 资产定价的底层逻辑

> **核心问题**：价格是如何形成的？价值如何决定？你能赚到什么钱？

---

## 一、价格是什么？供需关系的产物

### 1.1 从交易的本质说起

任何交易的本质是什么？

```
交易 = 买方用货币交换卖方的资产

发生条件：
- 买方认为：资产价值 > 价格（愿意买）
- 卖方认为：资产价值 < 价格（愿意卖）
```

**关键洞察**：交易发生时，买卖双方对资产价值的判断必然不同。价格不是共识，而是双方妥协的结果。

### 1.2 价格是如何形成的？

**拍卖模型**（最简单的价格形成机制）：

假设有一个苹果要卖，5个买家出价：
- A：10元
- B：8元
- C：6元
- D：4元
- E：2元

成交价格是多少？

- 如果只有一个苹果，成交价 ≈ 10元（最高出价）
- 如果有3个苹果，成交价 ≈ 6元（第三个最高出价）

**推论**：价格由边际买方决定，而非平均买方。

### 1.3 股票价格的形成

股票市场的价格形成更复杂：

**供给方**：
- 现有股东卖出
- 公司增发新股
- 做空者借股卖出

**需求方**：
- 新投资者买入
- 现有股东增持
- 做空者平仓买入

**价格 = 供需平衡点**

```
当买盘 > 卖盘 → 价格上涨 → 卖盘增加、买盘减少 → 新平衡
当卖盘 > 买盘 → 价格下跌 → 买盘增加、卖盘减少 → 新平衡
```

### 1.4 价格形成的参与者

| 参与者类型 | 信息优势 | 行为特点 |
|-----------|---------|---------|
| 内部人 | 最优 | 交易受限，需披露 |
| 机构投资者 | 较优 | 专业研究，大资金 |
| 散户 | 较劣 | 信息不对称，情绪化 |
| 量化/高频 | 速度优势 | 算法交易，套利 |
| 做市商 | 流动性优势 | 提供流动性，赚取价差 |

**你的位置**：你在信息链的什么位置？这决定了你可能的盈利空间。

## 二、价值是什么？未来现金流的折现

### 2.1 资产价值的本质

**第一性原理**：一个资产的价值，等于它未来能产生的所有现金流的现值之和。

**为什么？**

因为资产存在的唯一目的是创造价值（现金流）。如果你永远不能从资产获得任何现金流，它对你没有任何价值。

**数学表达**：

```
价值 = Σ CFt / (1 + r)^t

其中：
CFt = 第t年的现金流
r = 折现率（你的机会成本）
t = 时间
```

### 2.2 不同资产的现金流特征

| 资产类型 | 现金流来源 | 现金流特征 |
|----------|-----------|-----------|
| 债券 | 利息 + 本金 | 固定、可预测 |
| 股票 | 分红 + 资本回报 | 不确定、需预测 |
| 房产 | 租金 | 相对稳定 |
| 黄金 | 无 | 无现金流，价值来自稀缺性 |
| 加密货币 | 无（大多数） | 无现金流，价值来自共识 |

**关键区别**：

- 有现金流的资产：价值可以计算（虽然预测有误差）
- 无现金流的资产：价值无法计算，完全取决于他人愿意付多少钱

### 2.3 股票的价值计算

**股票的现金流来源**：

1. **分红**：公司直接返还股东的现金
2. **回购**：公司回购股票，间接增加每股价值
3. **清算价值**：公司清算时股东的剩余索取权

**戈登增长模型**（最简单的估值模型）：

```
股票价值 = D1 / (r - g)

其中：
D1 = 下一年分红
r = 要求回报率
g = 分红增长率
```

**案例**：

假设某公司：
- 下一年分红 2元/股
- 分红年增长率 5%
- 你要求回报率 10%

```
股票价值 = 2 / (10% - 5%) = 40元
```

### 2.4 价值的三个层次

**第一层：清算价值**
- 公司今天清算，股东能得到多少？
- 下限：通常低于此价格会被收购

**第二层：持续经营价值**
- 公司继续经营，未来现金流的现值
- 核心：也是最常用的估值方法

**第三层：成长价值**
- 公司未来增长机会的价值
- 上限：通常包含了很多不确定假设

```
总价值 = 清算价值 + 持续经营价值 + 成长价值
```

### 2.5 价值的主观性

**关键认识**：价值不是客观存在，而是主观判断。

为什么？

1. **现金流预测是主观的**：不同人对未来现金流预测不同
2. **折现率是主观的**：每个人的机会成本不同
3. **风险判断是主观的**：对风险的感知不同

**案例**：同一只股票，不同人的估值

| 投资者 | 预期现金流 | 折现率 | 估值 |
|--------|-----------|--------|------|
| 保守派 | 低增长5% | 高要求12% | 28元 |
| 中性派 | 中增长8% | 中要求10% | 50元 |
| 乐观派 | 高增长12% | 低要求8% | 100元 |

**启示**：价值区间比单一数值更有意义。

## 三、价格与价值：长期会收敛，短期会偏离

### 3.1 价格与价值的关系

**第一性原理**：

```
价格 = 价值 + 偏离

偏离的来源：
1. 信息不对称
2. 认知偏差
3. 情绪波动
4. 流动性需求
5. 市场结构因素
```

### 3.2 短期偏离的原因

**原因一：信息不对称**

- 内部人知道更多
- 机构研究能力更强
- 散户信息滞后

**原因二：认知偏差**

- 过度乐观/悲观
- 羊群效应
- 近期偏差（最近的表现影响判断）

**原因三：情绪波动**

- 恐惧和贪婪
- 市场恐慌时，价格远低于价值
- 市场狂热时，价格远高于价值

**原因四：流动性需求**

- 基金赎回被迫卖出
- 杠杆爆仓强平
- 税务筹划需求

**原因五：市场结构因素**

- 指数基金被动买入
- 做空限制
- 交易成本

### 3.3 长期收敛的机制

**为什么价格长期会收敛到价值？**

**套利机制**：

```
当价格 > 价值：
- 理性投资者卖出
- 卖压增加 → 价格下降 → 回归价值

当价格 < 价值：
- 理性投资者买入
- 买盘增加 → 价格上涨 → 回归价值
```

**实证**：研究显示，股价长期（5-10年）与公司基本面高度相关。

### 3.4 凯恩斯的警告

> "市场保持非理性的时间，可能比你保持偿付能力的时间更长。"

**启示**：

- 价格偏离价值可能持续很久
- 做空（押注价格回归）可能在你破产后才对
- 买入低估资产可能长期不涨

### 3.5 价格偏离的度量

**市盈率（PE）偏离历史均值**：

```
当前PE = 30倍
历史平均PE = 20倍
偏离程度 = (30 - 20) / 20 = 50%
```

**含义**：市场对该股票的预期比历史平均水平高50%，这需要未来业绩增长来支撑，否则价格会回落。

**注意事项**：

- 历史≠未来
- 行业变化可能导致合理PE变化
- 需要结合基本面分析

## 四、有效市场假说 vs 行为金融学

### 4.1 有效市场假说（EMH）

**核心观点**：价格已经反映所有可获得的信息。

**三种有效形式**：

| 形式 | 信息集 | 含义 |
|------|--------|------|
| 弱有效 | 历史价格信息 | 技术分析无效 |
| 半强有效 | 所有公开信息 | 基本面分析无效 |
| 强有效 | 所有信息（含内幕） | 无法获得超额收益 |

**逻辑推导**：

1. 市场有很多聪明的投资者
2. 他们不断寻找定价错误
3. 一旦发现，立即交易
4. 交易使价格迅速修正
5. 因此价格总是"正确"的

### 4.2 有效市场假说的证据

**支持证据**：

- 主动基金长期跑不赢指数
- 股价对新信息反应迅速
- 技术分析没有持续超额收益

**经典案例**：

巴菲特与对冲基金的十年赌约：
- 巴菲特：标普500指数基金
- 对冲基金：精选的5只FOF基金
- 结果：标普500年化7.1%，对冲基金年化2.2%

### 4.3 行为金融学的挑战

**核心观点**：投资者是非理性的，市场不是有效的。

**认知偏差清单**：

| 偏差 | 表现 | 对投资的影响 |
|------|------|-------------|
| 过度自信 | 高估自己的能力 | 过度交易，低估风险 |
| 确认偏误 | 只看支持自己观点的信息 | 忽视风险信号 |
| 锚定效应 | 过度依赖初始信息 | 买入价格影响卖出决策 |
| 损失厌恶 | 损失的痛苦>收益的快乐 | 过早卖出盈利，死守亏损 |
| 近期偏差 | 过度重视近期事件 | 追涨杀跌 |
| 羊群效应 | 跟随多数人 | 在极端时反向操作很难 |

### 4.4 行为金融学的证据

**市场异象**：

- **动量效应**：过去涨的股票未来短期继续涨
- **价值效应**：低估值股票长期跑赢高估值股票
- **规模效应**：小盘股长期跑赢大盘股
- **一月效应**：一月收益异常高

这些异象说明市场并非完全有效。

### 4.5 融合观点：市场有效性是一个光谱

**现实观点**：

```
高度有效 ←————————————————————→ 极度无效
   |                                   |
大盘股、高流动性、                    小盘股、低流动性、
多分析师覆盖                          少分析师覆盖
   |                                   |
难以获得超额收益                      更容易发现机会
```

**启示**：

1. 不要在大盘股上试图战胜市场（高度有效）
2. 关注小盘股、冷门股的机会（相对无效）
3. 利用其他投资者的认知偏差
4. 但也要认识到自己的偏差

## 五、你能赚到的是什么钱？

### 5.1 收益来源分解

**股票投资收益的三个来源**：

```
总收益 = 分红收益 + 盈利增长 + 估值变化

其中：
分红收益 = 分红率 / 买入价格
盈利增长 = 公司净利润增长率
估值变化 = PE变化率
```

**案例**：

假设你以20倍PE买入一只股票：
- 分红率 = 2%
- 盈利增长 = 8%/年
- 持有5年后PE变为25倍

```
估值变化 = (25/20)^(1/5) - 1 = 4.6%/年
总收益 = 2% + 8% + 4.6% = 14.6%/年
```

### 5.2 三种收益的可靠性

| 收益来源 | 可靠性 | 你的控制力 |
|----------|--------|-----------|
| 分红收益 | 高 | 低（取决于公司） |
| 盈利增长 | 中 | 低（取决于公司） |
| 估值变化 | 低 | 中（取决于买入时机） |

**启示**：

- 分红最可靠，但收益有限
- 盈利增长需要研究判断
- 估值变化最难预测，不要依赖它赚钱

### 5.3 你能赚到的三类钱

**第一类：企业创造价值的钱**

- 来源：公司盈利增长、分红
- 特点：正和博弈，所有投资者都能赚到
- 方法：买入好公司，长期持有
- 可持续性：高

**第二类：市场定价错误的钱**

- 来源：价格偏离价值时的修复
- 特点：零和博弈，你赚的是别人亏的
- 方法：价值投资、逆向投资
- 可持续性：中（因为套利会使机会消失）

**第三类：其他投资者的钱**

- 来源：利用其他投资者的错误
- 特点：零和博弈
- 方法：技术分析、量化策略、套利
- 可持续性：低（市场会进化，机会会消失）

### 5.4 选择你的策略

**根据你的优势选择**：

| 你的优势 | 适合的策略 | 赚的钱 |
|----------|-----------|--------|
| 时间优势 | 指数投资 | 企业创造的钱 |
| 研究能力 | 价值投资 | 企业创造 + 定价错误 |
| 信息优势 | 量化/套利 | 其他投资者的钱 |
| 情绪控制 | 逆向投资 | 定价错误 |

**大多数人应该赚第一类钱**：

因为：
- 不需要战胜市场
- 正和博弈，可持续
- 门槛低，不需要特殊能力

### 5.5 一个框架：四象限分析

| | 价格正确 | 价格错误 |
|---|---------|---------|
| 好公司 | 合理价格买入好公司<br>赚企业增长的钱 | 低估买入好公司<br>赚企业增长 + 估值修复 |
| 差公司 | 高价买入差公司<br>避免 | 低价买入差公司（烟蒂）<br>赚估值修复，风险高 |

**最优策略**：低估买入好公司（价格错误 + 好公司）

## 本日总结

### 核心概念回顾

1. **价格**：供需平衡点，由边际买方决定，短期可能偏离价值
2. **价值**：未来现金流的现值，主观但有方法论
3. **价格与价值**：短期偏离，长期收敛
4. **市场有效性**：是一个光谱，不是非黑即白
5. **收益来源**：分红 + 增长 + 估值变化

### 第一性原理总结

| 现象 | 第一性原理 |
|------|-----------|
| 价格形成 | 供需均衡 |
| 价值决定 | 未来现金流的现值 |
| 价格偏离 | 信息不对称 + 认知偏差 + 情绪 |
| 长期收敛 | 套利机制 |
| 投资收益 | 分享企业创造的价值 |

### 课后思考题

1. 你过去投资的收益，主要来自哪个来源？分红、增长还是估值变化？

2. 如果市场是有效的，为什么还有价值投资者能长期跑赢市场？

3. 如何判断当前价格是高估还是低估？你需要什么信息？

4. 为什么说"价值是主观的"？这对你做投资决策有什么启示？
