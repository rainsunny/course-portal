# Day 7: 财务报表分析

> **核心问题**：如何从财务报表看出公司的真实经营状况？

---

## 一、财务报表的本质

### 1.1 财务报表是什么？

**第一性原理**：

```
财务报表 = 公司经营活动的记录和总结

财务报表回答三个核心问题：
1. 公司赚了多少钱？（利润表）
2. 公司有什么资产和负债？（资产负债表）
3. 公司现金从哪来、到哪去？（现金流量表）

财务报表的作用：
- 管理层：了解经营状况，做出决策
- 投资者：评估公司价值，做出投资决策
- 债权人：评估偿债能力，做出借贷决策
- 监管机构：监督合规，保护投资者
```

### 1.2 三张表的逻辑关系

**第一性原理推导**：

```
利润表：
记录一段时间内的收入和费用
回答：公司赚了多少钱？

资产负债表：
记录某一时点的资产和负债
回答：公司有什么资源？欠多少钱？

现金流量表：
记录一段时间内的现金流入和流出
回答：公司的钱从哪来、到哪去？

三者的联系：
利润表（净利润）
    ↓
资产负债表（留存收益增加）
    ↓
现金流量表（经营现金流）

本质：
利润表是"流量"：一段时间的变化
资产负债表是"存量"：某一时点的状态
现金流量表是"流量"：一段时间的变化
```

**会计恒等式**：

```
资产 = 负债 + 所有者权益

为什么？
资产 = 公司拥有的资源
负债 + 所有者权益 = 资源的来源
- 负债 = 借来的
- 所有者权益 = 股东投入的 + 赚来的

资产的每一分钱，必然有来源
所以资产必然等于负债+所有者权益
```

### 1.3 财务报表的局限性

**局限一：历史成本**

```
会计准则要求：资产按历史成本记录

问题：
- 历史成本 ≠ 当前价值
- 土地、房产可能升值，但账面仍是历史成本
- 无形资产（品牌、专利）可能不在账上

例子：
公司20年前买的土地，成本1亿
现在市值10亿
但账面仍是1亿
```

**局限二：会计估计**

```
很多会计数据需要估计：

折旧年限：
- 机器设备：5年还是10年？
- 不同估计影响折旧费用，进而影响利润

坏账准备：
- 应收账款有多少收不回来？
- 不同估计影响坏账费用

存货跌价准备：
- 存货价值下降多少？
- 不同估计影响利润

这些估计给管理层操纵利润的空间
```

**局限三：非财务信息缺失**

```
财务报表只记录财务信息

缺失的重要信息：
- 管理层能力
- 员工素质
- 客户满意度
- 品牌价值
- 竞争优势
- 行业趋势
- 技术创新

这些信息对公司价值同样重要
```

**局限四：时间滞后**

```
财务报表是"后视镜"

年报披露时间：
- 年度报告：次年4月30日前
- 季度报告：季度结束后1个月内

当你看到报表时：
- 情况可能已经变化
- 市场可能已经反映

财务分析必须结合当前情况
```

## 二、利润表：盈利质量分析

### 2.1 利润表的结构

**基本结构**：

```
收入
- 营业成本
= 毛利润
- 销售费用
- 管理费用
- 研发费用
= 营业利润
+ 营业外收入
- 营业外支出
= 利润总额
- 所得税
= 净利润
```

**各项目的含义**：

| 项目 | 含义 | 关注点 |
|------|------|--------|
| 收入 | 销售商品/服务的收入 | 增长来源、可持续性 |
| 营业成本 | 生产商品的成本 | 成本控制、毛利率 |
| 毛利润 | 收入-成本 | 盈利能力的第一道防线 |
| 销售费用 | 推广、销售相关费用 | 销售效率 |
| 管理费用 | 管理相关费用 | 管理效率 |
| 研发费用 | 研发投入 | 创新能力 |
| 营业利润 | 主营业务利润 | 核心盈利能力 |
| 营业外收入/支出 | 非经常性项目 | 是否影响核心盈利 |
| 净利润 | 最终利润 | 股东可分配利润 |

### 2.2 收入分析

**第一性原理**：

```
收入 = 销量 × 价格

收入增长来源：
1. 销量增长
   - 市场扩大
   - 份额提升
   
2. 价格提升
   - 品牌溢价
   - 成本转嫁

分析收入要回答：
- 收入为什么增长？
- 增长是否可持续？
- 增长质量如何？
```

**收入增长的质量分析**：

```
高质量增长：
- 同店销售增长（老客户买更多）
- 新客户增加
- 产品结构优化（高毛利产品占比提升）
- 价格提升（品牌溢价）

低质量增长：
- 收购并表（非内生增长）
- 降价促销（牺牲利润换收入）
- 应收账款大幅增加（放宽信用政策）
- 渠道压货（透支未来需求）
```

**案例：收入质量分析**

```
公司A：
收入增长20%
应收账款增长5%
存货增长10%

公司B：
收入增长20%
应收账款增长30%
存货增长50%

分析：
公司A：收入增长，应收和存货增长合理，增长质量高
公司B：应收和存货增长快于收入，可能有以下问题：
- 放宽信用政策刺激销售
- 渠道压货
- 销售不畅，存货积压

结论：公司A的增长质量更高
```

### 2.3 毛利率分析

**毛利率计算**：

```
毛利率 = 毛利润 / 收入 = (收入 - 营业成本) / 收入

毛利率的含义：
- 每1元收入，扣除成本后赚多少钱
- 反映产品竞争力和定价能力
```

**毛利率的决定因素**：

```
毛利率 = f(行业特性, 竞争优势, 成本结构)

行业特性：
- 软件行业：毛利率70-90%（边际成本低）
- 零售行业：毛利率20-30%（竞争激烈）
- 制造业：毛利率20-40%（成本占比高）

竞争优势：
- 品牌溢价 → 高毛利率
- 规模效应 → 高毛利率
- 技术壁垒 → 高毛利率
- 差异化产品 → 高毛利率

成本结构：
- 固定成本高 → 规模效应 → 高毛利率（销量大时）
- 变动成本高 → 毛利率稳定
```

**毛利率的趋势分析**：

```
毛利率上升：
- 提价成功
- 成本下降
- 产品结构优化
- 规模效应

毛利率下降：
- 价格战
- 成本上升
- 产品结构恶化
- 竞争加剧

重要：毛利率变化趋势比绝对值更重要
```

**案例：毛利率分析**

```
某公司毛利率变化：

年份    毛利率
2020    40%
2021    42%
2022    41%
2023    38%

分析：
2020-2021：毛利率上升，可能原因：
- 提价
- 成本下降
- 产品结构优化

2022-2023：毛利率下降，需要调查：
- 原材料涨价？
- 竞争加剧？
- 降价促销？

下一步：查看成本明细、竞争对手毛利率
```

### 2.4 费用率分析

**三项费用**：

```
销售费用率 = 销售费用 / 收入
管理费用率 = 管理费用 / 收入
研发费用率 = 研发费用 / 收入

费用率反映效率：
- 费用率下降 → 效率提升
- 费用率上升 → 效率下降

但需区分：
- 好的费用增长：研发投入、品牌建设
- 坏的费用增长：管理浪费、效率低下
```

**销售费用率分析**：

```
销售费用率的含义：
- 每1元收入，花费多少销售费用
- 反映销售效率

销售费用率下降：
- 品牌效应增强（客户主动来买）
- 渠道成熟（无需大量推广）
- 效率提升

销售费用率上升：
- 市场竞争激烈
- 新市场开拓
- 销售效率下降

关注：
销售费用率与收入增长的关系
如果销售费用率大幅上升，但收入增长不快
说明销售效率下降
```

**研发费用率分析**：

```
研发费用率的含义：
- 公司对创新的投入程度

研发费用率高：
- 优点：技术积累、创新能力
- 缺点：短期利润压力

研发费用率低：
- 优点：短期利润高
- 缺点：长期竞争力下降

行业比较：
科技行业：研发费用率10-20%
制造业：研发费用率3-5%
传统行业：研发费用率1-3%

关注：
研发费用是否资本化？
资本化 → 当期费用减少，利润增加
费用化 → 当期费用增加，利润减少

过度资本化可能是利润操纵
```

### 2.5 盈利质量分析

**第一性原理**：

```
盈利质量 = 净利润的可持续性和真实性

高质量盈利：
- 来自核心业务
- 有现金流支撑
- 可持续

低质量盈利：
- 来自非经常性项目
- 无现金流支撑
- 不可持续
```

**盈利质量指标**：

```
指标一：经营现金流 / 净利润

含义：每1元净利润，有多少现金支撑

判断：
> 1：盈利质量高（现金流>利润）
≈ 1：盈利质量正常
< 1：盈利质量低（现金流<利润）

警惕：
持续 < 0.8：可能利润操纵
```

```
指标二：核心利润占比

核心利润 = 营业利润 - 投资收益 - 营业外收支
核心利润占比 = 核心利润 / 利润总额

含义：利润中有多少来自核心业务

判断：
> 80%：核心业务强劲
50-80%：正常
< 50%：依赖非核心业务

警惕：
核心利润占比持续下降：主营业务衰退
```

```
指标三：非经常性损益占比

非经常性损益 = 投资收益 + 营业外收支 + 其他非经常项目
非经常性损益占比 = 非经常性损益 / 利润总额

含义：利润中有多少是一次性的

判断：
< 20%：正常
20-50%：关注
> 50%：警惕

警惕：
依赖非经常性损益的公司，盈利不可持续
```

**案例：盈利质量分析**

```
公司财务数据（单位：亿元）：

                    2021    2022    2023
收入                100     120     150
净利润              10      12      18
经营现金流          8       6       5
投资收益            0       2       5
营业外收入          0.5     1       3

分析：

经营现金流/净利润：
2021：8/10 = 0.8
2022：6/12 = 0.5
2023：5/18 = 0.28

趋势：持续下降，盈利质量恶化

核心利润占比：
2021：(10-0-0.5)/10 = 95%
2022：(12-2-1)/12 = 75%
2023：(18-5-3)/18 = 56%

趋势：持续下降，依赖非核心业务

结论：
虽然净利润增长，但盈利质量下降
净利润增长来自投资收益和营业外收入
主营业务可能遇到问题
需要深入调查
```

## 三、资产负债表：财务健康度

### 3.1 资产负债表的结构

**基本结构**：

```
资产
├── 流动资产
│   ├── 货币资金
│   ├── 应收账款
│   ├── 存货
│   └── 其他流动资产
└── 非流动资产
    ├── 固定资产
    ├── 无形资产
    └── 其他非流动资产

负债
├── 流动负债
│   ├── 短期借款
│   ├── 应付账款
│   └── 其他流动负债
└── 非流动负债
    ├── 长期借款
    └── 其他非流动负债

所有者权益
├── 实收资本
├── 资本公积
├── 盈余公积
└── 未分配利润
```

**资产负债表解读**：

```
资产 = 公司拥有的资源
- 流动资产：一年内变现或消耗
- 非流动资产：长期使用的资产

负债 = 公司欠的钱
- 流动负债：一年内要还的
- 非流动负债：一年后要还的

所有者权益 = 资产 - 负债
= 股东的净资产
```

### 3.2 资产质量分析

**第一性原理**：

```
资产质量 = 资产的变现能力和盈利能力

高质量资产：
- 容易变现
- 能产生收益
- 不贬值

低质量资产：
- 难以变现
- 不能产生收益
- 可能贬值
```

**货币资金分析**：

```
货币资金 = 现金 + 银行存款 + 其他货币资金

关注：
1. 货币资金是否充足
   - 货币资金 / 流动负债 > 30% 为健康
   - 过低：流动性风险
   - 过高：资金效率低

2. 货币资金是否真实
   - 检查：货币资金 vs 利息收入
   - 如果货币资金很多，但利息收入很低
   - 可能：货币资金不真实

3. 货币资金是否受限
   - 受限资金：保证金、质押资金
   - 受限资金不能自由使用
   - 检查附注：受限资金金额
```

**应收账款分析**：

```
应收账款 = 客户欠公司的钱

应收账款周转天数 = 应收账款 / 收入 × 365
含义：平均多少天收回货款

分析要点：

1. 周转天数变化
   增加：信用政策放宽或收款困难
   减少：信用政策收紧或收款效率提升

2. 应收账款增速 vs 收入增速
   应收账款增速 > 收入增速：可能放宽信用刺激销售
   应收账款增速 < 收入增速：销售质量好

3. 坏账准备
   坏账准备 / 应收账款：坏账比例
   过低：可能低估坏账
   过高：客户质量差

4. 应收账款集中度
   前五大客户占比：集中度风险
   过高：依赖少数客户
```

**存货分析**：

```
存货 = 原材料 + 在产品 + 产成品

存货周转天数 = 存货 / 营业成本 × 365
含义：平均多少天卖出存货

分析要点：

1. 周转天数变化
   增加：销售不畅或备货过多
   减少：销售畅快或库存管理优化

2. 存货增速 vs 收入增速
   存货增速 > 收入增速：可能滞销
   存货增速 < 收入增速：销售好

3. 存货结构
   原材料占比高：备货积极
   产成品占比高：可能滞销

4. 存货跌价准备
   跌价准备 / 存货：存货贬值比例
   过高：存货可能贬值
   过低：可能低估跌价
```

**固定资产分析**：

```
固定资产 = 房屋、机器、设备等长期资产

固定资产周转率 = 收入 / 固定资产
含义：每1元固定资产产生多少收入

分析要点：

1. 固定资产周转率
   高：资产利用率高
   低：资产利用率低

2. 固定资产成新率
   = 固定资产净值 / 固定资产原值
   高：资产较新
   低：资产老化，需要更新

3. 在建工程
   大量在建工程：扩张中
   长期在建工程：可能问题

4. 折旧政策
   折旧年限：影响利润
   折旧方法：直线法 vs 加速折旧
   检查是否合理
```

**无形资产分析**：

```
无形资产 = 专利、商标、软件等

关注：

1. 无形资产来源
   外购：真实价值
   自创：可能高估（研发资本化）

2. 摊销政策
   摊销年限：影响利润
   是否合理

3. 商誉
   并购产生
   商誉减值风险
   商誉 / 净资产 > 30%：警惕
```

### 3.3 负债分析

**第一性原理**：

```
负债 = 公司欠的钱

负债的本质：
- 有息负债：借钱要付利息
- 无息负债：占用他人资金

分析负债要回答：
1. 公司有多少负债？
2. 负债是否可承受？
3. 有没有偿债风险？
```

**负债结构分析**：

```
资产负债率 = 总负债 / 总资产

含义：公司资产中，有多少是借来的

判断：
< 50%：财务稳健
50-70%：需要关注
> 70%：风险较高

但需区分行业：
- 金融业：资产负债率高是正常的
- 公用事业：资产负债率高可接受
- 制造业：资产负债率应该较低
```

```
有息负债率 = 有息负债 / 总资产

有息负债 = 短期借款 + 长期借款 + 应付债券

含义：真正需要付利息的负债

判断：
< 30%：财务稳健
30-50%：需要关注
> 50%：风险较高

关注：
有息负债 vs 货币资金
如果货币资金 < 有息负债，偿债压力大
```

```
流动比率 = 流动资产 / 流动负债

含义：流动资产能否覆盖流动负债

判断：
> 2：流动性充足
1-2：流动性正常
< 1：流动性风险

注意：
流动比率过高也不好，资金效率低
```

```
速动比率 = (流动资产 - 存货) / 流动负债

含义：扣除存货后，流动资产能否覆盖流动负债

判断：
> 1：流动性充足
0.5-1：流动性正常
< 0.5：流动性风险

为什么扣除存货？
存货变现慢，可能贬值
速动比率更保守
```

**偿债能力分析**：

```
利息保障倍数 = EBIT / 利息费用
或
利息保障倍数 = (净利润 + 利息费用 + 所得税) / 利息费用

含义：息税前利润能覆盖利息多少倍

判断：
> 5：偿债能力强
3-5：偿债能力正常
1-3：偿债能力弱
< 1：无法覆盖利息，高风险

案例：
公司EBIT = 100万，利息费用 = 20万
利息保障倍数 = 100/20 = 5倍
含义：息税前利润是利息的5倍，偿债能力强
```

```
经营现金流 / 有息负债

含义：经营现金流能否覆盖有息负债

判断：
> 0.3：健康（3年多能还清）
0.1-0.3：正常
< 0.1：压力大

为什么用现金流？
利润不代表现金
现金流才是真实偿债能力
```

### 3.4 资产负债表的健康信号

**健康信号**：

```
1. 现金充足
   - 货币资金 > 有息负债
   - 经营现金流持续为正

2. 应收账款健康
   - 应收账款周转天数稳定或下降
   - 坏账准备合理

3. 存货健康
   - 存货周转天数稳定或下降
   - 存货结构合理

4. 负债适度
   - 资产负债率 < 60%
   - 有息负债率 < 40%
   - 利息保障倍数 > 5

5. 流动性充足
   - 流动比率 > 1
   - 速动比率 > 0.5
```

**危险信号**：

```
1. 现金紧张
   - 货币资金 < 有息负债
   - 经营现金流持续为负
   - 利息收入远低于货币资金应有的收益

2. 应收账款恶化
   - 应收账款增速 > 收入增速
   - 应收账款周转天数大幅增加
   - 坏账准备过低

3. 存货恶化
   - 存货增速 > 收入增速
   - 存货周转天数大幅增加
   - 存货跌价准备过低

4. 负债过高
   - 资产负债率 > 70%
   - 有息负债率 > 50%
   - 利息保障倍数 < 3

5. 流动性风险
   - 流动比率 < 1
   - 速动比率 < 0.5
   - 经营现金流持续为负

6. "存贷双高"
   - 货币资金和有息负债都很高
   - 可能是财务造假
   - 为什么有钱还要借钱？
```

## 四、现金流量表：现金流 vs 利润

### 4.1 现金流量表的结构

**基本结构**：

```
经营活动现金流
+ 投资活动现金流
+ 筹资活动现金流
= 现金净增加额

经营活动现金流：
销售商品收到的现金
购买商品支付的现金
支付职工薪酬
支付税费
...

投资活动现金流：
购建固定资产
收购子公司
投资支付
...

筹资活动现金流：
借款收到
还款支付
分红支付
...
```

**三种现金流的含义**：

| 现金流类型 | 含义 | 正常情况 |
|-----------|------|---------|
| 经营现金流 | 主营业务产生的现金 | 持续为正 |
| 投资现金流 | 投资、扩张的现金 | 负（扩张）或正（收缩） |
| 筹资现金流 | 融资的现金 | 视情况而定 |

### 4.2 现金流质量分析

**第一性原理**：

```
现金流比利润更真实

为什么？
利润可以被会计政策影响
现金流是真金白银，难以操纵

现金流与利润的关系：
净利润是"会计利润"
经营现金流是"现金利润"

理想情况：
经营现金流 > 净利润
说明利润质量高，有现金支撑
```

**现金流质量指标**：

```
经营现金流 / 净利润

判断：
> 1.5：优秀（利润有充足现金支撑）
1-1.5：良好
0.8-1：正常
0.5-0.8：关注
< 0.5：警惕（利润质量差）

连续多年 < 0.8，可能是：
1. 利润操纵
2. 应收账款大量增加
3. 存货大量积压
```

**现金流组合分析**：

```
经营现金流 + 投资现金流 + 筹资现金流

理想组合：
(+, -, -)
经营现金流为正：主营业务赚钱
投资现金流为负：在扩张投资
筹资现金流为负：在还债或分红

关注组合：
(+, +, -)
经营和投资都为正：可能在收缩
筹资为负：在还债或分红

危险组合：
(-, -, +)
经营现金流为负：主营业务亏损
投资现金流为负：还在扩张
筹资现金流为正：在借钱维持
→ 烧钱模式，需要持续融资

最危险组合：
(-, +, +)
经营现金流为负：主营业务亏损
投资现金流为正：在变卖资产
筹资现金流为正：在借钱
→ 失血状态，可能破产
```

### 4.3 自由现金流分析

**第一性原理**：

```
自由现金流 = 公司可以自由支配的现金

经营现金流 = 主营业务产生的现金
资本支出 = 维持和扩张业务需要的投资
自由现金流 = 经营现金流 - 资本支出

含义：
经营现金流是"赚来的钱"
资本支出是"必须花的钱"
自由现金流是"剩下的钱" → 可以分红、回购、还债
```

**自由现金流的意义**：

```
自由现金流是股东的真实回报

为什么？
利润可以操纵
分红需要现金
回购需要现金
还债需要现金

只有自由现金流为正，公司才能：
- 分红
- 回购股票
- 还债
- 收购

自由现金流持续为负：
需要持续融资
股东可能被稀释
```

**自由现金流 vs 净利润**：

```
情况一：自由现金流 > 净利润
- 利润质量高
- 现金流充足
- 可以分红回购

情况二：自由现金流 ≈ 净利润
- 利润质量正常
- 现金流健康

情况三：自由现金流 < 净利润
- 利润质量差
- 现金流紧张
- 可能有应收账款、存货问题

情况四：自由现金流为负
- 公司在烧钱
- 需要持续融资
- 可能是成长期公司
```

**案例：现金流分析**

```
某公司现金流数据（单位：亿元）：

                    2021    2022    2023
净利润              10      12      15
经营现金流          12      8       6
投资现金流          -8      -10     -12
筹资现金流          -2      4       8

分析：

经营现金流/净利润：
2021：12/10 = 1.2 → 良好
2022：8/12 = 0.67 → 关注
2023：6/15 = 0.4 → 警惕

趋势：利润增长，但经营现金流下降
利润质量恶化

投资现金流持续为负：
公司在持续扩张

筹资现金流变化：
2021：-2（分红或还债）
2022-2023：正（在借钱）

现金流组合：
2021：(+, -, -) → 理想
2022：(+, -, +) → 关注
2023：(+, -, +) → 关注

结论：
虽然利润增长，但：
- 经营现金流/净利润持续下降
- 需要借款维持
- 可能存在应收账款或存货问题
需要深入调查
```

## 五、关键财务比率与预警信号

### 5.1 盈利能力指标

```
毛利率 = 毛利润 / 收入
判断：行业比较，趋势分析

净利率 = 净利润 / 收入
判断：> 10% 优秀，5-10% 良好，< 5% 一般

ROE = 净利润 / 净资产
判断：> 15% 优秀，10-15% 良好，< 10% 一般

ROA = 净利润 / 总资产
判断：> 10% 优秀，5-10% 良好，< 5% 一般

ROE分解（杜邦分析）：
ROE = 净利率 × 资产周转率 × 权益乘数
    = 净利润/收入 × 收入/总资产 × 总资产/净资产

含义：
净利率：盈利能力
资产周转率：运营效率
权益乘数：财务杠杆

通过杜邦分析，了解ROE的驱动因素
```

### 5.2 运营效率指标

```
应收账款周转天数 = 应收账款 / 收入 × 365
判断：与行业比较，趋势分析

存货周转天数 = 存货 / 营业成本 × 365
判断：与行业比较，趋势分析

总资产周转率 = 收入 / 总资产
判断：> 1 高效，< 1 需改善

运营周期 = 应收账款周转天数 + 存货周转天数 - 应付账款周转天数
含义：从采购到收款的周期
判断：越短越好
```

### 5.3 财务风险指标

```
资产负债率 = 总负债 / 总资产
判断：< 50% 稳健，50-70% 关注，> 70% 风险

流动比率 = 流动资产 / 流动负债
判断：> 2 充足，1-2 正常，< 1 风险

速动比率 = (流动资产 - 存货) / 流动负债
判断：> 1 充足，0.5-1 正常，< 0.5 风险

利息保障倍数 = EBIT / 利息费用
判断：> 5 强，3-5 正常，< 3 弱，< 1 危险

经营现金流 / 有息负债
判断：> 0.3 健康，0.1-0.3 正常，< 0.1 压力大
```

### 5.4 成长性指标

```
收入增长率 = (本期收入 - 上期收入) / 上期收入
判断：> 20% 高增长，10-20% 中增长，< 10% 低增长

净利润增长率 = (本期净利润 - 上期净利润) / 上期净利润
判断：与收入增长率比较

收入增长 vs 利润增长：
收入增长 > 利润增长：可能牺牲利润换增长
利润增长 > 收入增长：盈利能力提升

可持续增长率 = ROE × (1 - 分红率)
含义：不融资情况下的最大增长率
```

## 六、实战：识别财务造假

### 6.1 财务造假的动机

**第一性原理**：

```
财务造假的动机：
1. 融资需求：IPO、定增、债券发行
2. 业绩考核：管理层激励
3. 股价维护：大股东质押、减持
4. 避免退市：连续亏损会被ST

造假的方向：
1. 虚增收入：虚构销售
2. 虚减成本：成本资本化
3. 虚增资产：虚增存货、应收账款
4. 隐瞒负债：表外负债
```

### 6.2 财务造假的预警信号

**信号一：利润与现金流背离**

```
净利润持续增长，经营现金流持续下降

案例：
公司连续3年净利润增长20%
但经营现金流连续3年为负

可能原因：
- 虚增收入（收入确认了，但没收到钱）
- 提前确认收入
- 应收账款大量增加

检查：
应收账款周转天数
应收账款坏账准备
收入确认政策
```

**信号二：存贷双高**

```
货币资金和有息负债都很高

案例：
公司货币资金100亿
有息负债80亿

为什么有钱还要借钱？
- 货币资金不真实
- 货币资金受限
- 利率套利（借钱利率低于存款利率？）

检查：
货币资金的利息收入
有息负债的利息支出
货币资金是否受限
```

**信号三：毛利率异常**

```
毛利率远高于同行，且无合理原因

案例：
公司毛利率50%
同行毛利率20%

为什么毛利率高？
- 品牌溢价？
- 成本优势？
- 技术壁垒？

如果找不到合理原因，可能是虚减成本

检查：
成本结构
成本计算方法
同行对比
```

**信号四：应收账款异常**

```
应收账款增速远高于收入增速

案例：
收入增长20%
应收账款增长50%

可能原因：
- 放宽信用政策刺激销售
- 虚构销售
- 提前确认收入

检查：
应收账款周转天数
应收账款账龄结构
坏账准备
客户集中度
```

**信号五：存货异常**

```
存货增速远高于收入增速

案例：
收入增长20%
存货增长80%

可能原因：
- 销售不畅，积压
- 虚增存货
- 存货减值不足

检查：
存货周转天数
存货结构
存货跌价准备
存货盘点
```

**信号六：关联交易**

```
关联方交易占比过高

案例：
公司收入50%来自关联方

可能问题：
- 通过关联交易虚增收入
- 通过关联交易转移利润
- 关联交易价格不公允

检查：
关联交易披露
关联交易定价
关联方往来款
```

**信号七：频繁更换审计机构**

```
公司频繁更换会计师事务所

可能原因：
- 审计意见分歧
- 掩盖问题

检查：
更换原因
审计意见类型
审计费用
```

**信号八：大额资产减值**

```
某一年突然大额计提资产减值

案例：
公司连续盈利
某年突然计提50亿资产减值，巨亏

可能是：
- "洗大澡"，把以前的问题一次性暴露
- 为未来"调节利润"做准备

检查：
资产减值的原因
资产减值的合理性
历史资产质量
```

### 6.3 财务造假识别框架

```
第一步：整体分析
- 利润与现金流是否匹配？
- 资产负债是否健康？
- 关键指标是否异常？

第二步：细节检查
- 应收账款是否异常？
- 存货是否异常？
- 毛利率是否异常？
- 其他应收/应付是否异常？

第三步：交叉验证
- 收入 vs 纳税
- 利润 vs 分红
- 资产 vs 保险
- 经营数据 vs 行业数据

第四步：非财务信息
- 管理层背景
- 公司治理
- 审计意见
- 监管问询

第五步：综合判断
- 有多少预警信号？
- 预警信号的严重程度？
- 是否有合理解释？
```
