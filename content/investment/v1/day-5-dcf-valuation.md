# Day 5: 估值方法论（上）

> **核心问题**：一家公司值多少钱？如何从第一性原理推导估值？

---

## 一、估值的本质

### 1.1 为什么需要估值？

**第一性原理**：

```
投资决策的核心：
买入价格 < 内在价值 → 买入
买入价格 > 内在价值 → 不买或卖出

因此，估值是投资决策的基础

没有估值，就没有判断"便宜"或"贵"的标准
```

**估值的目的**：

| 目的 | 说明 |
|------|------|
| 买入决策 | 判断是否值得买入 |
| 卖出决策 | 判断是否应该卖出 |
| 持有决策 | 判断是否继续持有 |
| 组合权重 | 决定分配多少资金 |

### 1.2 估值的两种方法

**第一性原理分类**：

```
方法一：绝对估值法
- 从第一性原理出发
- 计算未来现金流的现值
- 代表：DCF（现金流折现模型）

方法二：相对估值法
- 与可比公司比较
- 假设市场对相似公司的定价是合理的
- 代表：PE、PB、PS等

两种方法的本质区别：
- 绝对估值：回答"值多少钱"
- 相对估值：回答"相对于其他公司，是否便宜"
```

### 1.3 估值的主观性

**关键认识**：

```
估值不是精确的科学，而是模糊的艺术

为什么？
1. 未来现金流需要预测 → 不确定
2. 折现率需要估计 → 主观
3. 永续增长率需要假设 → 主观

格雷厄姆说：
"估值的目的不是确定精确的内在价值，
而是判断内在价值是否足够确定，
从而判断价格是否过高或过低。"
```

**估值区间的意义**：

```
不是估值 = 100元
而是估值 = 80-120元

当价格 = 60元时，我们有足够的安全边际
当价格 = 110元时，我们无法判断是否便宜
当价格 = 150元时，我们确定它被高估
```

## 二、内在价值的核心概念

### 2.1 内在价值的定义

**第一性原理推导**：

```
内在价值 = 未来所有现金流的现值之和

为什么？
因为资产存在的唯一目的是产生现金流
所以，资产的价值 = 它能产生的所有现金流的现值

数学表达：
V = Σ [CFt / (1+r)^t]

其中：
V = 内在价值
CFt = 第t期的现金流
r = 折现率
t = 时间
```

### 2.2 为什么需要折现？

**第一性原理**：

```
折现的本质：今天的钱比明天的钱更值钱

原因：
1. 时间价值：钱可以投资，产生收益
2. 机会成本：今天不花，放弃了其他机会
3. 不确定性：未来的钱不一定能拿到
4. 通胀：未来的钱购买力下降

因此，未来的现金流需要"打折"，才能与今天的钱比较
```

**折现的数学**：

```
一年后的100元，今天值多少？

如果折现率r = 10%：
现值 = 100 / (1 + 10%) = 90.91元

两年后的100元：
现值 = 100 / (1 + 10%)² = 82.64元

n年后的100元：
现值 = 100 / (1 + 10%)^n

折现率越高，现值越低
时间越长，现值越低
```

### 2.3 折现率的选择

**折现率的本质**：

```
折现率 = 你要求的最低收益率 = 机会成本

为什么需要收益率？
因为你放弃了其他投资机会
这些机会本可以获得的收益，就是你的机会成本
```

**如何确定折现率？**

**方法一：资本资产定价模型（CAPM）**

```
r = rf + β × (rm - rf)

其中：
rf = 无风险利率（通常用国债利率）
β = 股票相对于市场的波动
rm = 市场预期收益率
rm - rf = 股票风险溢价

例子：
无风险利率 = 3%
市场预期收益 = 10%
某股票β = 1.2

折现率 = 3% + 1.2 × (10% - 3%) = 11.4%
```

**方法二：加权平均资本成本（WACC）**

```
WACC = E/V × Re + D/V × Rd × (1 - Tc)

其中：
E = 股权市值
D = 债务市值
V = E + D（总市值）
Re = 股权成本（用CAPM计算）
Rd = 债务成本（公司债券利率）
Tc = 公司税率

例子：
股权市值 = 80亿，股权成本 = 12%
债务市值 = 20亿，债务成本 = 6%
税率 = 25%

WACC = 0.8 × 12% + 0.2 × 6% × (1-25%)
     = 9.6% + 0.9%
     = 10.5%
```

**方法三：机会成本法（最实用）**

```
问自己：如果不投这家公司，我能获得什么收益？

参考标准：
- 银行存款：2-3%
- 债券基金：4-5%
- 指数基金：8-10%
- 优质股票：10-15%
- 风险投资：20%+

根据公司的风险调整：
- 低风险公司（公用事业）：折现率较低
- 高风险公司（科技初创）：折现率较高

一般参考：
稳健型公司：8-10%
成长型公司：10-12%
高风险公司：15%+
```

**折现率的影响**：

```
假设某公司每年产生100万自由现金流，永续增长3%

折现率10%时：
价值 = 100 / (10% - 3%) = 1429万

折现率12%时：
价值 = 100 / (12% - 3%) = 1111万

折现率8%时：
价值 = 100 / (8% - 3%) = 2000万

结论：
折现率从10%变到12%，估值下降22%
折现率从10%变到8%，估值上升40%

折现率是估值中最敏感的参数！
```

## 三、现金流折现模型（DCF）详解

### 3.1 DCF的基本原理

**第一性原理**：

```
公司价值 = 未来所有自由现金流的现值

为什么用自由现金流？
- 净利润可以被会计操纵
- 自由现金流是真金白银
- 公司的价值 = 它能分配给股东的钱

自由现金流 = 公司经营产生的现金 - 维持经营需要的投资
```

**DCF的基本公式**：

```
企业价值 = Σ [FCFt / (1+WACC)^t] + 终值

股权价值 = 企业价值 - 净债务

其中：
FCF = 自由现金流（Free Cash Flow）
WACC = 加权平均资本成本
终值 = 预测期之后的价值
```

### 3.2 自由现金流的计算

**第一性原理推导**：

```
自由现金流 = 经营活动产生的现金 - 资本支出

为什么？
经营现金 = 公司日常业务赚的现金
资本支出 = 维持/扩大业务需要的投资
自由现金流 = 可以分配给股东的钱

计算公式：
FCF = EBIT × (1 - 税率) + 折旧摊销 - 资本支出 - 营运资本变动

或简化：
FCF = 经营现金流 - 资本支出
```

**详细计算步骤**：

```
从净利润出发：

净利润
+ 折旧摊销          （非现金支出，加回来）
- 资本支出          （投资支出，减去）
- 营运资本变动      （营运资金变化）
= 自由现金流

其中：
营运资本 = 应收账款 + 存货 - 应付账款
营运资本增加 → 占用现金 → FCF减少
营运资本减少 → 释放现金 → FCF增加
```

**案例：某公司自由现金流计算**

```
利润表（单位：百万元）
收入          1000
营业成本       600
毛利           400
销售管理费用   150
研发费用        50
营业利润       200
利息费用        20
税前利润       180
所得税（25%）   45
净利润         135

现金流量表补充信息：
折旧            40
摊销            10
资本支出        80
营运资本变动：
  应收账款增加   30
  存货增加       20
  应付账款增加   15
  营运资本变动 = 30 + 20 - 15 = 35（占用现金）

自由现金流计算：
净利润              135
+ 折旧摊销          50
- 资本支出          80
- 营运资本变动      35
= 自由现金流       70

解释：
公司赚了135万净利润
但加回折旧摊销50万（非现金支出）
减去资本支出80万（投资）
减去营运资本变动35万（被占用的钱）
实际可自由支配的现金 = 70万
```

### 3.3 DCF的两种形式

**形式一：FCFF（企业自由现金流）**

```
FCFF = 企业自由现金流 = 属于所有投资者（股东+债权人）的现金流

计算：
FCFF = EBIT × (1-税率) + 折旧摊销 - 资本支出 - 营运资本变动

折现：
用WACC折现FCFF
得到企业价值
股权价值 = 企业价值 - 净债务

适用场景：
公司有债务，或资本结构变化
```

**形式二：FCFE（股权自由现金流）**

```
FCFE = 股权自由现金流 = 属于股东的现金流

计算：
FCFE = 净利润 + 折旧摊销 - 资本支出 - 营运资本变动 
       + 净借款（新借款 - 还款）

折现：
用股权成本（Re）折现FCFE
直接得到股权价值

适用场景：
公司无债务，或资本结构稳定
```

**两种形式的等价性**：

```
当资本结构稳定时：
FCFE = FCFF - 利息支出 × (1-税率) + 净借款

股权价值 = 企业价值 - 净债务

两种方法应该得到相同的结果
```

### 3.4 DCF的预测期与终值

**预测期**：

```
预测期 = 你能合理预测现金流的年数

一般原则：
- 成熟公司：5-10年
- 成长型公司：3-5年
- 周期型公司：一个完整周期

为什么不能预测太长？
- 预测越远，不确定性越大
- 折现后，远期现金流影响变小
- 实际操作中，5-10年已经足够
```

**终值（Terminal Value）**：

```
终值 = 预测期之后的所有现金流的价值

为什么需要终值？
因为公司理论上永续经营
但我们只能预测有限年份
终值代表预测期之后的价值

计算方法：

方法一：永续增长模型（Gordon Growth Model）

终值 = FCFn × (1+g) / (WACC - g)

其中：
FCFn = 预测期最后一年的自由现金流
g = 永续增长率
WACC = 折现率

方法二：退出倍数法

终值 = EBITDAn × 退出倍数

其中：
EBITDAn = 预测期最后一年的EBITDA
退出倍数 = 行业平均EV/EBITDA
```

**永续增长率的选择**：

```
永续增长率 = 公司预测期后的长期增长率

第一性原理：
永续增长率 <= 名义GDP增长率

为什么？
如果公司永远以高于GDP的速度增长
它最终会比整个经济还大
这显然不可能

实际参考：
- 成熟公司：2-3%（名义GDP增长）
- 衰退行业：0-1%
- 不超过：3-4%

重要：
永续增长率对终值影响极大
必须谨慎选择
```

**终值在总价值中的占比**：

```
典型情况：
终值占比 = 60-80%

这意味着：
DCF估值大部分来自终值
而终值又高度依赖假设

因此：
DCF对假设非常敏感
必须进行敏感性分析
```

### 3.5 DCF完整案例

**案例：A公司DCF估值**

```
已知信息：
- 当前自由现金流：100万元
- 预测期：5年
- 预测期增长率：10%
- 永续增长率：3%
- WACC：10%
- 净债务：200万元

步骤一：预测未来5年自由现金流

Year 0: FCF = 100万
Year 1: FCF = 100 × 1.10 = 110万
Year 2: FCF = 110 × 1.10 = 121万
Year 3: FCF = 121 × 1.10 = 133万
Year 4: FCF = 133 × 1.10 = 146万
Year 5: FCF = 146 × 1.10 = 161万

步骤二：计算终值

终值 = FCF5 × (1+g) / (WACC - g)
     = 161 × (1+3%) / (10% - 3%)
     = 166 × 7%
     = 2371万

步骤三：折现到现值

Year 1: 110 / (1.10)^1 = 100万
Year 2: 121 / (1.10)^2 = 100万
Year 3: 133 / (1.10)^3 = 100万
Year 4: 146 / (1.10)^4 = 100万
Year 5: 161 / (1.10)^5 = 100万
终值:   2371 / (1.10)^5 = 1474万

步骤四：计算企业价值

企业价值 = 100 + 100 + 100 + 100 + 100 + 1474 = 1974万

步骤五：计算股权价值

股权价值 = 企业价值 - 净债务
        = 1974 - 200
        = 1774万

如果发行1000万股：
每股价值 = 1774 / 1000 = 1.77元
```

**敏感性分析**：

```
对关键假设进行敏感性分析：

永续增长率：
如果g = 2%：终值 = 161 × 1.02 / 8% = 2053万，股权价值 = 1505万
如果g = 3%：终值 = 2371万，股权价值 = 1774万
如果g = 4%：终值 = 161 × 1.04 / 6% = 2794万，股权价值 = 2134万

WACC：
如果WACC = 9%：终值 = 166 / 6% = 2767万，股权价值 = 2077万
如果WACC = 10%：终值 = 2371万，股权价值 = 1774万
如果WACC = 11%：终值 = 166 / 8% = 2075万，股权价值 = 1497万

结论：
估值区间大致在 1500-2100万 之间
取决于对永续增长率和WACC的假设
```

## 四、DCF的局限性与改进

### 4.1 DCF的主要局限

**局限一：对假设高度敏感**

```
DCF估值 = 假设驱动

关键假设：
1. 未来现金流预测
2. 折现率
3. 永续增长率

每个假设的小变化，会导致估值大变化

例子：
永续增长率从3%变到4%
估值可能增加20-30%
```

**局限二：预测困难**

```
预测未来现金流是困难的

问题：
1. 经济周期难以预测
2. 行业变化难以预测
3. 公司战略调整
4. 竞争格局变化
5. 技术颠覆

现实：
大多数公司的现金流预测都严重偏离实际
```

**局限三：终值占比过大**

```
终值通常占总价值的60-80%

这意味着：
大部分估值来自"预测期之后"
而预测期之后，我们一无所知

这是一个悖论：
我们不知道的时期，贡献了大部分价值
```

**局限四：不适用于某些公司**

```
DCF不适用的情况：
1. 初创公司：无现金流，无法预测
2. 周期性公司：现金流波动大
3. 困境公司：现金流为负
4. 资产型公司：价值主要在资产，非现金流

这些公司需要用其他估值方法
```

### 4.2 如何改进DCF

**改进一：多情景分析**

```
不只做一个预测，而是做多个情景：

情景一：乐观
- 增长率高
- 利润率高
- 估值上限

情景二：中性
- 增长率中等
- 利润率正常
- 基准估值

情景三：悲观
- 增长率低
- 利润率下降
- 估值下限

然后：
给出估值区间，而非单一值
```

**改进二：反向DCF**

```
正向DCF：假设 → 估值
反向DCF：价格 → 隐含假设

步骤：
1. 假设当前股价是合理的
2. 反推需要什么样的现金流增长
3. 判断这个增长是否合理

例子：
股价 = 50元
反推得出：未来10年需要15%的现金流增长

问自己：
- 这个增长率合理吗？
- 行业增长率多少？
- 公司能持续这个增长吗？
```

**改进三：分阶段预测**

```
不是用一个增长率预测所有年份，而是分阶段：

阶段一：高速增长期（3-5年）
- 增长率：15-20%
- 公司处于成长期

阶段二：过渡期（3-5年）
- 增长率：5-10%
- 公司进入成熟期

阶段三：稳定增长期（永续）
- 增长率：2-3%
- 公司成熟稳定

这更符合公司生命周期
```

**改进四：结合其他估值方法**

```
不要只依赖DCF，而是多方法交叉验证：

1. DCF估值
2. PE/PB相对估值
3. 历史估值比较
4. 并购估值

如果多种方法得出的估值接近，可信度高
如果差异大，需要深入分析原因
```

### 4.3 DCF的正确使用方式

**第一性原理**：

```
DCF的价值不在于精确计算
而在于：

1. 理解价值驱动因素
   - 什么决定公司价值？
   - 哪些因素最重要？

2. 建立估值框架
   - 如何思考估值问题？
   - 哪些假设关键？

3. 判断安全边际
   - 当前价格是否足够便宜？
   - 即使假设保守，是否仍有空间？

4. 比较不同投资机会
   - 哪家公司更值？
   - 相对价值如何？
```

**实用建议**：

```
1. 始终做敏感性分析
   - 改变关键假设
   - 观察估值变化

2. 关注估值区间，而非单一值
   - 不说"估值 = 100元"
   - 说"估值区间 = 80-120元"

3. 检查隐含假设
   - 当前价格隐含什么增长预期？
   - 这个预期合理吗？

4. 结合定性分析
   - 行业前景
   - 竞争优势
   - 管理质量

5. 保持谦逊
   - 承认预测的不确定性
   - 留足安全边际
```

## 五、实战：用DCF给一家公司估值

### 5.1 选择目标公司

**选择标准**：
- 现金流相对稳定
- 业务相对简单
- 财务数据透明

**案例公司：某白酒公司（简化版）**

```
基本信息：
- 品牌知名，市场份额稳定
- 过去5年自由现金流稳定增长
- 资产负债表健康
- 当前市值：2000亿
```

### 5.2 收集数据

**历史数据（单位：亿元）**：

| 年份 | 收入 | 净利润 | 经营现金流 | 资本支出 | 自由现金流 |
|------|------|--------|-----------|---------|-----------|
| 2019 | 800 | 350 | 360 | 30 | 330 |
| 2020 | 880 | 400 | 410 | 35 | 375 |
| 2021 | 970 | 450 | 460 | 40 | 420 |
| 2022 | 1070 | 500 | 510 | 45 | 465 |
| 2023 | 1180 | 550 | 560 | 50 | 510 |

**增长率分析**：
- 收入年增长率：10%
- 净利润年增长率：11-12%
- 自由现金流年增长率：约11%

### 5.3 建立假设

**假设一：未来增长**

```
基于：
- 行业成熟，增长放缓
- 公司市场份额稳定
- 通胀因素

假设：
- 未来5年：收入增长8%，利润率稳定
- 之后进入稳定增长：3%
```

**假设二：利润率和现金流**

```
净利润率：稳定在47%左右
自由现金流/净利润：约93%
（白酒行业资本支出少，现金流质量高）
```

**假设三：折现率**

```
无风险利率：3%
市场风险溢价：7%
公司β：0.8（白酒行业波动较低）

股权成本 = 3% + 0.8 × 7% = 8.6%

公司无债务，WACC = 8.6%
```

### 5.4 进行估值

**预测未来现金流（亿元）**：

| 年份 | 收入 | 净利润 | 自由现金流 |
|------|------|--------|-----------|
| 2024 | 1274 | 599 | 557 |
| 2025 | 1376 | 647 | 602 |
| 2026 | 1486 | 698 | 649 |
| 2027 | 1605 | 754 | 701 |
| 2028 | 1733 | 815 | 758 |

**计算终值**：

```
终值 = FCF2029 / (WACC - g)
     = 758 × 1.03 / (8.6% - 3%)
     = 781 / 5.6%
     = 13946亿
```

**折现到现值**：

| 年份 | FCF | 折现因子 | 现值 |
|------|-----|---------|------|
| 2024 | 557 | 0.921 | 513 |
| 2025 | 602 | 0.848 | 511 |
| 2026 | 649 | 0.781 | 507 |
| 2027 | 701 | 0.719 | 504 |
| 2028 | 758 | 0.662 | 502 |
| 终值 | 13946 | 0.662 | 9232 |

**计算企业价值**：

```
企业价值 = 513 + 511 + 507 + 504 + 502 + 9232 = 11769亿

公司无债务：
股权价值 = 企业价值 = 11769亿
```

### 5.5 敏感性分析

**永续增长率敏感性**：

| 永续增长率 | 股权价值 | 相对当前市值 |
|-----------|---------|-------------|
| 2% | 10068亿 | -0.5倍 |
| 3% | 11769亿 | -0.4倍 |
| 4% | 14023亿 | -0.3倍 |

**折现率敏感性**：

| WACC | 股权价值 | 相对当前市值 |
|------|---------|-------------|
| 7.6% | 14145亿 | -0.3倍 |
| 8.6% | 11769亿 | -0.4倍 |
| 9.6% | 9976亿 | -0.5倍 |

### 5.6 投资决策

**结论**：

```
当前市值：2000亿
DCF估值：约11769亿（中性假设）

敏感度分析：
估值区间：约10000-14000亿

当前市值远低于估值下限，可能的原因：
1. 市场对行业前景悲观
2. 市场对公司增长预期更低
3. 要求更高的回报率

决策：
需要进一步了解市场悲观的原因
如果悲观是暂时的，可能是买入机会
如果悲观有合理原因，需要调整假设
```

**进一步分析**：

```
检查隐含假设：
如果市值 = 2000亿是合理的
需要：
- 未来5年增长0%
- 或永续增长率为负
- 或折现率高达15%+

这明显不合理

可能的解释：
1. 市场极度悲观
2. 存在我们不知道的风险
3. 数据或假设有误

下一步：
深入研究公司基本面
验证假设的合理性
寻找市值与估值差异的原因
```

## 本日总结

### 核心概念回顾

1. **估值本质**：未来现金流的现值
2. **折现**：时间价值、机会成本、不确定性的体现
3. **DCF模型**：预测现金流 → 选择折现率 → 计算终值 → 折现求和
4. **敏感性**：估值对假设高度敏感，必须做敏感性分析
5. **DCF价值**：不在于精确计算，而在于理解价值驱动因素

### DCF估值步骤

```
1. 收集历史数据
2. 预测未来现金流（通常5-10年）
3. 选择折现率（WACC或股权成本）
4. 计算终值
5. 折现求和
6. 敏感性分析
7. 判断安全边际
```

### 关键公式

```
自由现金流 = 净利润 + 折旧摊销 - 资本支出 - 营运资本变动

终值 = FCFn × (1+g) / (WACC - g)

企业价值 = Σ [FCFt / (1+WACC)^t] + 终值 / (1+WACC)^n

股权价值 = 企业价值 - 净债务
```

### 课后思考题

1. 为什么自由现金流比净利润更适合估值？

2. 为什么DCF对假设高度敏感？如何应对这种敏感性？

3. 假设一家公司当前自由现金流为50万元，预计未来5年年增长15%，之后永续增长3%，折现率12%。计算其估值。

4. 对于一家目前亏损的初创公司，DCF是否适用？如果不适用的，应该如何估值？

5. 为什么说"DCF的价值不在于精确计算，而在于建立估值框架"？
