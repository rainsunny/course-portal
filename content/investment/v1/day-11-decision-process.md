# Day 11: 投资决策流程

> **核心问题**：如何系统化地做出投资决策？从选股到卖出的完整流程是什么？

---

## 一、投资决策的系统性框架

### 1.1 为什么需要投资决策流程？

**第一性原理**：

```
投资决策流程的必要性：

原因一：减少情绪影响
情绪是投资的大敌
系统化的流程可以减少情绪干扰

原因二：提高决策质量
系统化决策可以避免遗漏
确保每个重要因素都被考虑

原因三：可复制、可改进
有流程才能复制成功
有流程才能改进优化

原因四：提供纪律约束
流程是纪律的基础
防止冲动决策

投资决策的核心问题：
1. 买什么？（选股）
2. 什么时候买？（买入时机）
3. 买多少？（仓位管理）
4. 什么时候卖？（卖出决策）
```

**投资决策流程的完整性**：

```
完整的投资决策流程：

机会筛选 → 深度研究 → 买入决策 → 持仓管理 → 卖出决策
    ↓          ↓          ↓          ↓          ↓
  建立股票池  分析基本面  确定时机    监控变化    触发条件
```

### 1.2 投资决策的类型

**类型一：自上而下**

```
自上而下 = 宏观 → 行业 → 公司

流程：
1. 宏观分析：判断经济周期、政策环境
2. 行业分析：选择景气行业
3. 公司分析：在行业中选优秀公司

优点：
- 从大局出发
- 适合大资金
- 行业配置清晰

缺点：
- 可能错过小行业的大机会
- 对宏观判断要求高
```

**类型二：自下而上**

```
自下而上 = 公司 → 行业 → 宏观

流程：
1. 公司分析：寻找优秀公司
2. 行业分析：了解公司所处行业
3. 宏观分析：考虑宏观影响

优点：
- 关注公司本身
- 不依赖宏观判断
- 可能发现冷门机会

缺点：
- 可能忽视宏观风险
- 行业风险分散不足
```

**类型三：混合方法**

```
混合 = 自上而下筛选 + 自下而上精选

流程：
1. 宏观分析：排除明显风险的行业
2. 行业筛选：选择景气或稳定的行业
3. 公司精选：在行业中寻找优秀公司

优点：
- 结合两种方法的优点
- 既考虑宏观，又关注公司

推荐方法
```

## 二、机会筛选：如何从4000只股票中找到标的？

### 2.1 筛选的层次

**第一性原理**：

```
筛选 = 从大量标的中选出少数值得研究的标的

筛选的层次：
4000只股票 → 筛选器 → 100只 → 快速研究 → 20只 → 深度研究 → 5只 → 买入

每一层的筛选标准：
- 筛选器：定量标准
- 快速研究：定性标准
- 深度研究：全面分析
```

### 2.2 定量筛选标准

**筛选器一：估值筛选**

```
估值标准：
PE < 行业平均
PE < 历史中位数
PB < 行业平均
PB < 历史中位数
PEG < 1.5

目的：
筛选出低估的股票

注意：
低估值可能是价值陷阱
需要结合其他标准
```

**筛选器二：质量筛选**

```
质量标准：
ROE > 10%（或行业平均）
经营现金流 > 净利润
毛利率 > 行业平均
资产负债率 < 60%

目的：
筛选出财务健康的股票

注意：
高ROE可能是财务杠杆导致
需要看ROA和ROIC
```

**筛选器三：成长筛选**

```
成长标准：
收入增长率 > 10%
利润增长率 > 10%
收入增长稳定（波动小）

目的：
筛选出成长的股票

注意：
增长可能不可持续
需要看增长来源
```

**筛选器四：其他筛选**

```
市值筛选：
市值 > 50亿（避免小市值风险）
市值 < 5000亿（避免大象难跳舞）

流动性筛选：
日均成交额 > 1亿
换手率适中

行业筛选：
排除夕阳行业
选择景气行业或稳定行业

财务健康：
没有ST、退市风险
没有重大财务造假嫌疑
```

### 2.3 筛选案例

```
案例：筛选A股低估质量股

步骤一：估值筛选
PE < 20倍
PB < 3倍
筛选结果：500只

步骤二：质量筛选
ROE > 12%
经营现金流/净利润 > 0.8
资产负债率 < 60%
筛选结果：100只

步骤三：成长筛选
收入增长率 > 5%
利润增长率 > 5%
筛选结果：50只

步骤四：其他筛选
市值 > 100亿
日均成交额 > 2亿
非ST股
筛选结果：30只

步骤五：快速研究
阅读年报
排除明显问题
筛选结果：10只

步骤六：深度研究
详细分析
选择买入标的
筛选结果：3-5只
```

## 三、深度研究框架

### 3.1 商业模式分析

**第一性原理**：

```
商业模式 = 公司如何赚钱

核心问题：
1. 公司卖什么？（产品/服务）
2. 卖给谁？（客户）
3. 怎么卖？（渠道）
4. 怎么赚钱？（盈利模式）
5. 能持续吗？（护城河）
```

**分析框架**：

```
问题一：公司卖什么？
- 产品/服务是什么？
- 有什么独特性？
- 是否有差异化？
- 是否有替代品？

问题二：卖给谁？
- 客户是谁？
- 客户集中度？
- 客户粘性？
- 客户议价能力？

问题三：怎么卖？
- 销售渠道是什么？
- 直销还是经销？
- 线上还是线下？
- 渠道控制力？

问题四：怎么赚钱？
- 收入来源是什么？
- 成本结构是什么？
- 利润率是多少？
- 是否可持续？

问题五：能持续吗？
- 护城河是什么？
- 竞争优势是什么？
- 能持续多久？
```

**案例：白酒行业商业模式**

```
卖什么：白酒产品
- 差异化强：品牌、口味、文化
- 替代性弱：消费者忠诚度高

卖给谁：
- 高端：商务宴请、礼品
- 中端：家庭消费、朋友聚会
- 客户分散，议价能力弱

怎么卖：
- 经销为主，直销为辅
- 渠道控制力强（高端）
- 渠道依赖性高（中低端）

怎么赚钱：
- 高毛利率：70-90%
- 高净利率：30-50%
- 品牌溢价

能持续吗：
- 品牌：历史传承，难以复制
- 文化：社交需求独特
- 护城河深
```

### 3.2 财务分析

**分析框架**：

```
一、盈利能力
- 毛利率：产品定价能力
- 净利率：整体盈利能力
- ROE：股东回报率
- ROIC：投入资本回报率

二、盈利质量
- 经营现金流/净利润
- 核心利润占比
- 非经常性损益占比

三、成长能力
- 收入增长率
- 利润增长率
- 增长来源

四、财务健康
- 资产负债率
- 流动比率
- 利息保障倍数

五、运营效率
- 应收账款周转天数
- 存货周转天数
- 总资产周转率
```

**财务分析案例**：

```
某公司财务分析：

盈利能力：
毛利率：40%（行业平均35%）→ 好
净利率：15%（行业平均12%）→ 好
ROE：18%（行业平均10%）→ 好
ROIC：15%（行业平均10%）→ 好

盈利质量：
经营现金流/净利润：1.2 → 好
核心利润占比：90% → 好
非经常性损益占比：10% → 正常

成长能力：
收入增长率：15% → 好
利润增长率：20% → 好
增长来源：销量增长+提价 → 好

财务健康：
资产负债率：40% → 健康
流动比率：1.5 → 正常
利息保障倍数：8 → 好

运营效率：
应收账款周转天数：30天 → 好
存货周转天数：60天 → 正常
总资产周转率：0.8 → 正常

结论：财务状况健康
```

### 3.3 估值分析

**估值方法选择**：

```
方法一：DCF估值
适用：现金流稳定的公司
优点：理论基础强
缺点：假设敏感

方法二：PE估值
适用：盈利稳定的公司
优点：简单直观
缺点：盈利波动时不准

方法三：PB估值
适用：金融、重资产行业
优点：资产价值
缺点：忽略无形资产

方法四：PS估值
适用：亏损但高增长的公司
优点：不受盈利波动影响
缺点：忽略盈利能力

方法五：PEG估值
适用：成长股
优点：考虑增长
缺点：增长预测不准

选择原则：
- 多种方法交叉验证
- 选择适合公司特点的方法
- 给出估值区间，而非单一值
```

**估值分析案例**：

```
某公司估值分析：

方法一：DCF估值
假设：未来5年增长12%，永续增长3%，折现率10%
估值：80-100元

方法二：PE估值
当前EPS：5元
历史PE区间：10-25倍
同行PE：15-20倍
估值：75-100元

方法三：PB估值
当前BPS：30元
历史PB区间：2-4倍
同行PB：2.5-3倍
估值：75-90元

综合估值：
估值区间：75-100元
中位数：87.5元

当前价格：70元
安全边际：(87.5-70)/87.5 = 20%
```

### 3.4 风险分析

**风险类型**：

```
一、公司风险
- 经营风险：业务恶化
- 财务风险：债务违约
- 治理风险：管理层问题
- 竞争风险：竞争加剧

二、行业风险
- 行业衰退
- 政策变化
- 技术颠覆

三、宏观风险
- 经济衰退
- 利率上升
- 汇率波动

四、市场风险
- 系统性风险
- 流动性风险
- 情绪风险
```

**风险分析框架**：

```
对每个风险评估：
1. 发生概率
2. 影响程度
3. 可预测性
4. 可应对性

风险矩阵：

|          | 低影响 | 中影响 | 高影响 |
|----------|--------|--------|--------|
| 高概率   | 关注   | 重点应对 | 重点应对 |
| 中概率   | 监控   | 关注   | 重点应对 |
| 低概率   | 监控   | 监控   | 关注   |

应对策略：
- 规避：不投资
- 降低：降低仓位
- 转移：对冲
- 接受：承受风险
```

## 四、买入决策

### 4.1 买入时机

**第一性原理**：

```
买入时机 = 价格显著低于价值

什么样的价格算"显著低于"？
取决于安全边际：
- 高确定性公司：安全边际20-30%
- 中等确定性公司：安全边际30-50%
- 低确定性公司：安全边际50%+

买入时机的信号：

基本面信号：
- 估值低于历史中位数
- 估值低于同行
- 估值低于内在价值

市场信号：
- 市场恐慌
- 股价下跌
- 成交量萎缩

事件信号：
- 短期利空
- 行业周期底部
- 公司困境反转
```

**买入时机的错误**：

```
错误一：试图抄底
问题：不知道底在哪里
后果：抄底抄在半山腰

正确做法：分批买入
在估值合理区间开始买入
越跌越买

错误二：等待完美时机
问题：完美时机不存在
后果：错过机会

正确做法：宁可买早，不要错过
估值合理就可以买入

错误三：追高买入
问题：买在高点
后果：被套

正确做法：有纪律，不追高
```

### 4.2 仓位管理

**第一性原理**：

```
仓位管理 = 决定买多少

核心问题：
- 单只股票买多少？
- 总仓位多少？

仓位管理的原则：
1. 确定性高 → 仓位高
2. 确定性低 → 仓位低
3. 风险高 → 仓位低
4. 风险低 → 仓位高
```

**单一股票仓位**：

```
上限原则：
单只股票不超过总资产的10-15%
防止单一股票风险过大

确定性调整：
高确定性：8-10%
中确定性：5-8%
低确定性：3-5%

示例：
总资产100万
高确定性股票A：8-10万
中确定性股票B：5-8万
低确定性股票C：3-5万
```

**总仓位管理**：

```
总仓位 = 股票占总资产的比例

根据市场估值调整：

市场极度低估：总仓位80-100%
市场低估：总仓位60-80%
市场正常：总仓位40-60%
市场高估：总仓位20-40%
市场极度高估：总仓位0-20%

判断市场估值：
- 市场PE相对历史
- 市场情绪
- 新开户数
- 融资余额
```

**凯利公式**：

```
凯利公式：
f* = (p × b - q) / b

其中：
f* = 最优仓位比例
p = 获胜概率
q = 失败概率 = 1 - p
b = 赔率 = 获胜收益/失败损失

例子：
获胜概率p = 60%
失败概率q = 40%
获胜收益 = 50%
失败损失 = 20%
赔率b = 50%/20% = 2.5

f* = (0.6 × 2.5 - 0.4) / 2.5 = 0.44 = 44%

调整：
实际使用通常用半凯利或1/4凯利
即仓位 = 11%-22%

原因：
1. 概率和赔率估计不准确
2. 凯利公式假设可以反复下注
3. 保守一点，避免爆仓
```

### 4.3 分批建仓

**第一性原理**：

```
分批建仓 = 不一次性买入，分多次买入

为什么分批建仓？
1. 不知道底在哪里
2. 降低平均成本
3. 减少买在高点的风险
```

**分批建仓策略**：

```
策略一：等额分批

方法：将总金额分成N等份，分N次买入
例子：总金额10万，分5次，每次2万

优点：简单
缺点：不考虑估值变化

策略二：估值分批

方法：估值越低，买入越多
例子：
PE 20倍：买入20%
PE 18倍：买入30%
PE 16倍：买入50%

优点：低估值时多买
缺点：估值判断可能不准

策略三：价格分批

方法：价格越低，买入越多
例子：
价格100元：买入20%
价格90元：买入30%
价格80元：买入50%

优点：价格直观
缺点：不知道会跌到多少

策略四：时间分批

方法：固定时间间隔买入
例子：每月买入固定金额

优点：简单，分散时点风险
缺点：可能错过最佳买入时机
```

**建仓节奏**：

```
建仓周期：
短期：1-3个月
中期：3-6个月
长期：6-12个月

选择原则：
- 确定性高：建仓周期短
- 确定性低：建仓周期长
- 市场波动大：建仓周期长
- 市场稳定：建仓周期短
```

## 五、持仓管理

### 5.1 监控什么？

**监控内容**：

```
一、公司基本面监控

定期检查：
- 季报、年报
- 业绩是否符合预期
- 财务指标变化
- 重大公告

关注点：
- 收入增长是否放缓
- 利润率是否下降
- 现金流是否恶化
- 负债是否增加

二、行业监控

定期检查：
- 行业景气度
- 竞争格局变化
- 政策变化

关注点：
- 行业是否衰退
- 新竞争者进入
- 政策是否不利

三、估值监控

定期检查：
- 当前估值水平
- 估值是否过高

关注点：
- PE是否显著高于历史
- 估值是否透支未来增长

四、市场监控

定期检查：
- 市场整体估值
- 市场情绪

关注点：
- 市场是否过热
- 是否有系统性风险
```

### 5.2 调仓决策

**调仓的触发条件**：

```
加仓条件：
- 估值下降，安全边际增加
- 基本面超预期
- 仓位过低

减仓条件：
- 估值上升，安全边际减少
- 基本面恶化
- 仓位过高

清仓条件：
- 达到卖出条件（见下一节）
- 发现更好的投资机会
```

**调仓原则**：

```
原则一：有理由才调仓
不要频繁调仓
每次调仓都要有明确理由

原则二：控制交易成本
考虑交易成本和税务影响
避免因小失大

原则三：保持核心仓位
核心仓位不要轻易改变
只调整卫星仓位
```

### 5.3 止损vs不止损

**止损的逻辑**：

```
止损 = 当亏损达到一定程度时卖出

止损的优点：
1. 限制最大亏损
2. 避免深度套牢
3. 保护本金

止损的缺点：
1. 可能止损在最低点
2. 频繁止损侵蚀本金
3. 需要重新买入

止损适用情况：
- 短期交易
- 投机性投资
- 基本面恶化

止损不适用情况：
- 长期价值投资
- 基本面没变化
- 估值更低了
```

**不止损的逻辑**：

```
不止损 = 即使亏损也不卖出

前提：
1. 基本面没有恶化
2. 估值更低了，安全边际更大了
3. 长期投资

做法：
- 继续持有
- 甚至加仓

适用情况：
- 长期价值投资
- 基本面没变化
- 估值更低了

风险：
- 判断错误
- 基本面真的恶化
```

**折中策略**：

```
策略：区分情况

情况一：基本面恶化
→ 止损或减仓
判断标准：财务指标恶化、行业衰退、管理层问题

情况二：基本面没变，估值下降
→ 持有或加仓
判断标准：财务指标健康、行业景气、估值更低

情况三：判断错误
→ 止损
承认错误，及时纠正

止损点设置：
- 短期交易：-8%至-10%
- 中期投资：-15%至-20%
- 长期投资：基本面恶化才止损
```

## 六、卖出决策

### 6.1 卖出的三个理由

**第一性原理**：

```
卖出理由一：达到目标价
价格显著高于价值
实现预期收益

卖出理由二：基本面恶化
买入理由不再成立
公司出现问题

卖出理由三：发现更好的机会
资金有更好的去处
机会成本考量
```

**卖出理由一：达到目标价**

```
达到目标价的标准：
- 估值显著高于价值
- 价格透支未来增长
- 安全边际消失或变为负

如何判断？

方法一：估值判断
PE > 历史高位
PE > 行业平均的2倍
DCF估值显示高估

方法二：目标收益
达到预期收益（如50%、100%）
可以考虑卖出

方法三：风险收益比
上行空间小
下行风险大
风险收益比不划算

案例：
买入价格：50元
内在价值：80元
当前价格：100元
安全边际：(80-100)/80 = -25%
价格高于价值，可以考虑卖出
```

**卖出理由二：基本面恶化**

```
基本面恶化的信号：

财务恶化：
- 收入增长放缓或下降
- 利润率下降
- 现金流恶化
- 负债增加

业务恶化：
- 核心产品竞争力下降
- 市场份额下降
- 客户流失
- 新产品失败

行业恶化：
- 行业衰退
- 政策不利
- 技术颠覆

管理层问题：
- 管理层变动
- 战略错误
- 诚信问题

案例：
买入理由：公司ROE 20%，持续增长
实际情况：ROE下降到10%，增长停滞
买入理由不再成立 → 卖出
```

**卖出理由三：发现更好的机会**

```
机会成本考量：
资金是有限的
如果发现更好的机会，应该换仓

判断标准：
新机会的预期收益 > 现有持仓的预期收益
新机会的风险 < 现有持仓的风险

注意：
不要频繁换仓
确保新机会确实更好
考虑交易成本

案例：
现有持仓：预期收益10%，风险中等
新机会：预期收益20%，风险中等
换仓的预期收益 > 持有成本
可以考虑换仓
```

### 6.2 不应该卖出的理由

```
错误理由一：涨了就要卖
问题：可能还会涨
正确：看估值是否合理

错误理由二：跌了就要卖
问题：可能是加仓机会
正确：看基本面是否恶化

错误理由三：拿了很久了
问题：时间不是卖出理由
正确：看买入理由是否成立

错误理由四：市场下跌了
问题：市场下跌不代表公司问题
正确：看公司基本面

错误理由五：别人在卖
问题：从众不是理由
正确：独立思考

原则：
卖出要有明确理由
不要因为情绪卖出
不要因为没有耐心卖出
```

### 6.3 分批卖出

**第一性原理**：

```
分批卖出 = 不一次性卖出，分多次卖出

为什么分批卖出？
1. 不知道顶在哪里
2. 可能继续上涨
3. 降低卖在低点的风险
```

**分批卖出策略**：

```
策略一：目标价分批

方法：达到不同目标价，卖出不同比例
例子：
价格达到目标价100%：卖出30%
价格达到目标价120%：卖出30%
价格达到目标价150%：卖出40%

优点：锁定部分收益
缺点：可能错过更高价格

策略二：估值分批

方法：估值越高，卖出越多
例子：
PE 30倍：卖出30%
PE 35倍：卖出30%
PE 40倍：卖出40%

优点：估值导向
缺点：估值判断可能不准

策略三：时间分批

方法：固定时间间隔卖出
例子：每季度卖出一定比例

优点：简单
缺点：不考虑价格
```

## 本日总结

### 核心概念回顾

1. **投资决策流程**：机会筛选 → 深度研究 → 买入决策 → 持仓管理 → 卖出决策
2. **机会筛选**：定量筛选+定性筛选，从大量标的中筛选出少数值得研究的标的
3. **深度研究**：商业模式分析、财务分析、估值分析、风险分析
4. **买入决策**：时机、仓位、分批建仓
5. **持仓管理**：监控基本面、行业、估值、市场，根据变化调仓
6. **卖出决策**：达到目标价、基本面恶化、发现更好机会

### 投资决策框架

```
买入前：
- 筛选 → 研究 → 分析 → 决策

买入时：
- 时机 → 仓位 → 分批

持有中：
- 监控 → 评估 → 调整

卖出时：
- 理由 → 时机 → 分批

全程：
- 记录 → 复盘 → 改进
```

### 课后思考题

1. 设计你自己的股票筛选标准：估值、质量、成长各设定什么标准？

2. 选择一家你感兴趣的公司，用本节课的框架进行深度研究：商业模式、财务、估值、风险。

3. 设计你的仓位管理规则：单只股票上限多少？总仓位如何根据市场估值调整？

4. 设计你的卖出规则：什么情况下卖出？分批卖出还是一次性卖出？

5. 建立你的投资日志模板，开始记录每一次投资决策。
